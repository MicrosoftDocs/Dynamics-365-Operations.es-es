[!include[task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="679a9-105">Vaya a Contabilidad general > Consultas e informes > Transacciones de asiento.</span><span class="sxs-lookup"><span data-stu-id="679a9-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="679a9-106">Seleccione el campo para el que desea definir un criterio de filtro.</span><span class="sxs-lookup"><span data-stu-id="679a9-106">Select the field for which you want to define a filter criteria.</span></span>

    * <span data-ttu-id="679a9-108">Podría filtrar en un único valor o en un intervalo.</span><span class="sxs-lookup"><span data-stu-id="679a9-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="679a9-109">Al definir un intervalo, asegúrese de que se use la sintaxis correcta.</span><span class="sxs-lookup"><span data-stu-id="679a9-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="679a9-110">Los valores se deben separar con doble punto (..).</span><span class="sxs-lookup"><span data-stu-id="679a9-110">The values should be separated by a double period (..).</span></span>
