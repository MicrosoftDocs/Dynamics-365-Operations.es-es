# <a name="request-a-leave-of-absence"></a>Solicitar una excedencia

>[!Important]
>La funcionalidad mencionada en este artículo está disponible actualmente para los clientes de Dynamics 365 Human Resources independiente. Algunas o todas las funciones estarán disponibles como parte de una versión futura de la infraestructura de Finance después de la versión 10.0.26 de Finance.


[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Puede enviar una solicitud de excedencia y ver el estado de sus solicitudes de licencia en Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Solicitar una excedencia

1. En el espacio de trabajo **Autoservicio para empleados**, seleccione **Más** (...) en el icono **Saldos de permisos**.

2. Para enviar una solicitud de excedencia, seleccione **Solicitar excedencia**.

3. Introduzca información para **Tipo de baja**, **Fecha inicial** y **Fecha final**.

4. Si necesita enviar documentación complementaria, seleccione **Cargar** debajo de **Archivos adjuntos**.

5. Introduzca información en **Comentario**, si es necesario.

6. Seleccione **Enviar** cuando esté listo para enviar su solicitud. De lo contrario, seleccione **Guardar borrador**.


## <a name="view-leave-of-absence-request-status"></a>Ver el estado de la solicitud de excedencia

1. En el espacio de trabajo **Autoservicio para empleados**, seleccione **Más** (...) en el icono **Saldos de permisos**.

2. Para ver sus solicitudes de excedencia, seleccione **Ver excedencia**.

## <a name="update-a-leave-of-absence-request"></a>Actualizar una solicitud de excedencia

1. En el espacio de trabajo **Autoservicio para empleados**, en el icono **Permiso para ausentarse**, seleccione **Más (...)**.
2. Seleccione la solicitud de excedencia para actualizar y luego seleccione **Permiso para ausentarse actualizado**.
3. En el campo **Fecha de finalización**, actualice el valor según corresponda para ampliar o acortar el permiso de ausencia.
4. Si se confirma la fecha de finalización, establezca la opción **Confirmar fecha de finalización** en **Sí** .
5. Después de que la opción **Confirmar fecha de finalización** se establezca en **Sí**, puede cargar un aviso de vuelta al trabajo. Luego seleccione la casilla para confirmar que se ha cargado un aviso de vuelta al trabajo.
6. Seleccione **Enviar** para actualizar la solicitud de excedencia.

## <a name="cancel-a-leave-of-absence-request"></a>Cancelar una solicitud de excedencia

1. En el espacio de trabajo **Autoservicio para empleados**, en el icono **Permiso para ausentarse**, seleccione **Más (...)**.
2. Seleccione la solicitud de excedencia para cancelar y luego seleccione **Permiso para ausentarse actualizado**.
3. Establezca la opción **Cancelar baja** en **Sí**.
4. Seleccione **Enviar** para cancelar la solicitud de ausencia.

## <a name="importing-leave-requests-from-other-systems-or-older-systems"></a>Importar solicitudes de baja de otros sistemas o sistemas más antiguos

Para importar solicitudes de bajas de otro sistema, debe seguir el flujo de trabajo normal para crear las transacciones de permisos correspondientes. Alternativamente, puede importar las transacciones bancarias de licencia y las solicitudes de licencia en un estado completo. Tenga en cuenta que las transacciones bancarias de licencia no se crean automáticamente si importa solo las solicitudes de licencia.
