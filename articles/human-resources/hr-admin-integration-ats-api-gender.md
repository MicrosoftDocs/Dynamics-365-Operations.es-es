# <a name="gender"></a>Género

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe el conjunto de opciones de sexo para Dynamics 365 Human Resources.

Nombre físico: mshr_hcmpersongender

Esta enumeración proporciona el conjunto de opciones de sexo del candidato. Está disponible en el conjunto de opciones mshr_hcmpersongender.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | None | No se ha especificado ningún sexo. |
| 200000001 | Hombre | Hombre. |
| 200000002 | Mujer | Mujer. |
| 200000003 | NonSpecific | Selección para un género no específico. |
