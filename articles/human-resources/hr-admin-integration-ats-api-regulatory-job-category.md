# <a name="regulatory-job-category"></a>Categoría de trabajo reglamentaria


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artículo describe el conjunto de opciones de categoría de trabajo reglamentaria para Dynamics 365 Human Resources.

Nombre físico: mshr_hcmregulatoryjobcategory

Esta enumeración especifica el conjunto de opciones para los valores del conjunto de opciones de categoría de trabajo reglamentaria. Esto se ofrece en el conjunto de opciones mshr_hcmregulatoryjobcategory existente.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | None | Ninguno. |
| 200000001 | Ejecutivo | Funcionarios y administradores de nivel ejecutivo/superior. |
| 200000002 | Administrador | Funcionarios y administradores de nivel inicial e intermedio. |
| 200000003 | Profesional | Profesionales. |
| 200000004 | Técnico | Técnicos. |
| 200000005 | Vendedor | Vendedores. |
| 200000006 | Administrativo | Trabajadores de soporte administrativo. |
| 200000007 | Artesano | Artesanos. |
| 200000008 | Operativo | Operativos. |
| 200000009 | Obrero | Obreros/Auxiliares. |
| 200000010 | Trabajador de servicios | Trabajadores de servicios. |
