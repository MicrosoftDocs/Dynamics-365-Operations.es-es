# <a name="set-up-tier-codes"></a>Configurar códigos de niveles

[!include [banner](includes/preview-feature.md)]

Dynamics 365 Human Resources usa códigos de niveles con tasas de prestaciones que se pagan por los empleados y el empleador. Puede definir un número ilimitado de códigos de niveles y configurar cálculos de tasas para satisfacer las necesidades de su empresa.

1. En el espacio de trabajo **Administración de prestaciones**, en **Configuración**, seleccione **Códigos de niveles**.

2. Seleccione **Nuevo**.

3. Especifique los valores para los siguientes campos:

   | Campo | Descripción |
   | --- | --- |
   | Código de nivel | Un código o número de nivel de beneficio único para identificar el código de nivel. |
   | Descripción | Una descripción del tipo de nivel de beneficio. |

4. Seleccione **Guardar**.
