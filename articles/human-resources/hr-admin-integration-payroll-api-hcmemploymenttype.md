# <a name="payroll---employment-type-option-set"></a>Nómina: conjunto de opciones de tipo de empleo

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artículo describe el conjunto de opciones de tipo de empleo para Dynamics 365 Human Resources.

## <a name="option-set-employment-type"></a>Conjunto de opciones: tipo de empleo

Esta enumeración proporciona el conjunto de opciones de tipo de empleo del empleado. Está disponible en el conjunto de opciones mshr_hcmemploymenttype.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | Empleado | Empleado |
| 200000001 | Contratista | Contratista |
