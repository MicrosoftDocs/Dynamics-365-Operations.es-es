# <a name="screening-frequency"></a>Frecuencia de cribado

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe el conjunto de opciones de la frecuencia de cribado para Dynamics 365 Human Resources.

Nombre físico: mshr_hcmfrequencyunit

Esta enumeración proporciona el conjunto de opciones de valores de la frecuencia de cribado. 

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 Solo una vez | El cribado solo es necesario una vez. |
| 200000001 Diaria | La frecuencia de cribado se calcula en días. |
| 200000002 Semanal | La frecuencia de cribado se calcula en semanas. |
| 200000003 Mensual | La frecuencia de cribado se calcula en meses. |
| 200000004 Anual | La frecuencia de cribado se calcula en años. |
