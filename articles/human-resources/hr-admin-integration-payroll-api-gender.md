# <a name="payroll---gender-option-set"></a>Nómina - conjunto de opciones Género


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artículo describe el conjunto de opciones de género para Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Conjunto de opciones: opción Género

Esta enumeración proporciona el conjunto de opciones de sexo del empleado. Está disponible en el conjunto de opciones mshr_hcmpersongender.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | None | No se ha especificado ningún sexo. |
| 200000001 | Hombre | Hombre. |
| 200000002 | Mujer | Mujer. |
| 200000003 | NonSpecific | Selección para un género no específico. |
