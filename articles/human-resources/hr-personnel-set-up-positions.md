    * <span data-ttu-id="b8b90-114">La descripción del trabajo, el título y el factor de empleo equivalente a jornada completa se copian automáticamente del trabajo seleccionado al puesto.</span><span class="sxs-lookup"><span data-stu-id="b8b90-114">The Job description, title, and full-time equivalent employment factor are automatically copied from the selected job into the position.</span></span>

6. <span data-ttu-id="b8b90-116">Haga clic en Crear puesto.</span><span class="sxs-lookup"><span data-stu-id="b8b90-116">Click Create position.</span></span>

13. <span data-ttu-id="b8b90-126">Haga clic en Nueva para abrir el cuadro de diálogo desplegable.</span><span class="sxs-lookup"><span data-stu-id="b8b90-126">Click New to open the drop dialog.</span></span>
14. <span data-ttu-id="b8b90-127">En el campo Informa a, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="b8b90-127">In the Reports to field, enter or select a value.</span></span>
15. <span data-ttu-id="b8b90-128">Haga clic en Crear.</span><span class="sxs-lookup"><span data-stu-id="b8b90-128">Click Create.</span></span>
16. <span data-ttu-id="b8b90-129">Expanda la sección Asignación de trabajadores.</span><span class="sxs-lookup"><span data-stu-id="b8b90-129">Expand the Worker assignment section.</span></span>
17. <span data-ttu-id="b8b90-130">Expanda la sección Relaciones.</span><span class="sxs-lookup"><span data-stu-id="b8b90-130">Expand the Relationships section.</span></span>
    * <span data-ttu-id="b8b90-131">Si la organización usa una jerarquía de matriz u otra jerarquía personalizada, puede configurar tipos de jerarquía de puestos y agregar relaciones jerárquicas a los puestos para cada tipo de jerarquía que configure.</span><span class="sxs-lookup"><span data-stu-id="b8b90-131">If your organization uses a matrix hierarchy or another custom hierarchy, you can set up position hierarchy types and then add reporting relationships to positions for each hierarchy type that you set up.</span></span>  
18. <span data-ttu-id="b8b90-132">Haga clic en Agregar.</span><span class="sxs-lookup"><span data-stu-id="b8b90-132">Click Add.</span></span>
19. <span data-ttu-id="b8b90-133">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="b8b90-133">In the list, mark the selected row.</span></span>
20. <span data-ttu-id="b8b90-134">En el campo Nombre de jerarquía, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="b8b90-134">In the Hierarchy name field, enter or select a value.</span></span>
21. <span data-ttu-id="b8b90-135">En el campo Notifica al puesto, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="b8b90-135">In the Reports to position field, enter or select a value.</span></span>
22. <span data-ttu-id="b8b90-136">Expanda la sección Nómina.</span><span class="sxs-lookup"><span data-stu-id="b8b90-136">Expand the Payroll section.</span></span>

24. <span data-ttu-id="b8b90-138">En el campo Pagado por, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="b8b90-138">In the Paid by field, enter or select a value.</span></span>

30. <span data-ttu-id="b8b90-145">En el campo Departamento, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="b8b90-145">In the Department field, enter or select a value.</span></span>
31. <span data-ttu-id="b8b90-146">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="b8b90-146">Click Save.</span></span>
