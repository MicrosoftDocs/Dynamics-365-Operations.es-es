# <a name="skill-level-type"></a>Tipo de nivel de aptitud

Este tema describe el conjunto de opciones de tipo de nivel de aptitud para Dynamics 365 Human Resources.

Nombre físico: mshr_hrmskillleveltype

Esta enumeración proporciona el conjunto de opciones de valores que categorizan el nivel asignado a la aptitud de un candidato.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | Real | Nivel de aptitud real. |
| 200000001 | Objetivo | Valor objetivo para el nivel de aptitud. |
