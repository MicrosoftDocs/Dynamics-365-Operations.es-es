# <a name="benefit-plan-type-cover"></a>Cobertura de tipo de plan de prestaciones


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

En este artículo se describe la opción de cobertura del tipo de plan de prestaciones establecida para Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Conjunto de opciones: tipo de proceso

Nombre físico: mshr_plantypecode

Esta enumeración proporciona el conjunto de opciones para el tipo de proceso en entidades virtuales de Recursos Humanos.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | Cobertura médica | Médica. |
| 200000001 | Dental | Dental. |
| 200000002 | Visión | Visión. |
| 200000003 | Ahorro | Ahorro. |
| 200000004 | Cuenta de gastos flexible | Cuenta de gastos flexible. |
| 200000005 | Vida | Vida. |
| 200000006 | AccidentalDeathDismemberment | AccidentalDeathDismemberment. |
| 200000007 | Otra | Otro.|
| 200000008 | LongTermDisability | LongTermDisability. |
| 200000009 | ShortTermDisability | ShortTermDisability. |
