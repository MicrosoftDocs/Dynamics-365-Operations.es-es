# <a name="contact-type"></a>Tipo de contacto

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe el conjunto de opciones de tipo de contacto para Dynamics 365 Human Resources.

Nombre físico: mshr_logisticselectronicaddressmethodtype

Esta enumeración proporciona el conjunto de opciones de valores de tipos de contacto de candidatos. 

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | None | No se ha seleccionado ningún tipo. |
| 200000001 | Teléfono | Número de teléfono. |
| 200000002 | Dirección de correo electrónico | Dirección de correo electrónico. |
| 200000003 | URL | URL del sitio web. |
| 200000004 | Télex | Número de telex. |
| 200000005 | Fax | Número de fax. |
| 200000006 | Facebook | Cuenta de Facebook. Identificado por id. de usuario. |
| 200000007 | Twitter | Cuenta de Twitter. Identificado por @nombre de usuario. |
| 200000008 | LinkedIn | Cuenta de LinkedIn. Identificado por nombre de usuario. |
