# <a name="award-type"></a>Tipo de prima

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe el conjunto de opciones de tipo de prima para Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Conjunto de opciones: tipo de prima

Nombre físico: mshr_HrmCompVarAwardEmplType

Esta enumeración proporciona el conjunto de opciones para el tipo de prima de compensación variable en entidades virtuales de Recursos Humanos.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | Prima general | Prima general. |
| 200000001 | Pago fijo de suma total | Pago fijo de suma total. |
