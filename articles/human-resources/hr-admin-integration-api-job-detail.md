Este tema proporciona detalles y una consulta de ejemplo para la entidad de detalle del trabajo en Dynamics 365 Human Resources.

| Propiedad<br>**Nombre físico**<br>**_Tipo_** | Utilizar | Descripción |
| --- | --- | --- |
| **Id. de trabajo**<br>mshr_jobid<br>*Cadena* | Solo lectura<br>Obligatorio | Identificación única para un trabajo. |
| **Umbral bajo del precio de mercado**<br>mshr_marketpricelowthreshold<br>*Decimal* | | Valor GUID generado por el sistema para identificar el puesto de forma única.  |
