# <a name="blank-yes-no"></a>En blanco Sí No

Este tema describe el conjunto de opciones En blanco Sí No para Dynamics 365 Human Resources.

Nombre físico: mshr_hrmblankyesno

Esta enumeración proporciona un conjunto de opciones para propiedades sí/no que también permiten el valor en blanco.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | En blanco | No se ha seleccionado ningún valor. |
| 200000001 | Sí | Sí. |
| 200000002 | N.º | Nº |
