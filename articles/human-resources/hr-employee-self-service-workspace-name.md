# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="17d67-103">Cambiar el nombre del espacio de trabajo de autoservicio del empleado</span><span class="sxs-lookup"><span data-stu-id="17d67-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="17d67-104">Si tiene voluntarios u otros no empleados, es posible que desee cambiar el nombre del espacio de trabajo **Autoservicio para empleados**.</span><span class="sxs-lookup"><span data-stu-id="17d67-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="17d67-105">Puede cambiar este espacio de trabajo a **Autoservicio**, en lugar del existente.</span><span class="sxs-lookup"><span data-stu-id="17d67-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="17d67-106">Cambiar el nombre del espacio de trabajo **Autoservicio para empleados** también cambia el elemento del menú que usa internamente Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="17d67-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="17d67-107">Si anteriormente aplicó personalizaciones de seguridad al elemento del menú **HcmEmployeeSelfServiceWorkspace**, recomendamos aplicar los mismos cambios a **HcmSelfServiceWorkspace** para mantener la paridad.</span><span class="sxs-lookup"><span data-stu-id="17d67-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="17d67-108">En Human Resources, seleccione **Gestión de personal**, seleccione **Vínculos** y luego seleccione **Parámetros de recursos humanos**.</span><span class="sxs-lookup"><span data-stu-id="17d67-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="17d67-109">Seleccione la pestaña **Autoservicio para empleados**.</span><span class="sxs-lookup"><span data-stu-id="17d67-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="17d67-110">En **Nombre para mostrar**, seleccione **Autoservicio**.</span><span class="sxs-lookup"><span data-stu-id="17d67-110">Under **Display name**, select **Self service**.</span></span>

   ![Cambiar el nombre del espacio de trabajo de autoservicio a Autoservicio](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="17d67-112">Seleccione **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="17d67-112">Select **Save**.</span></span>
