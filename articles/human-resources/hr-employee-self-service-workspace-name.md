# <a name="change-employee-self-service-workspace-name"></a>Cambiar el nombre del espacio de trabajo de autoservicio del empleado

Si tiene voluntarios u otros no empleados, es posible que desee cambiar el nombre del espacio de trabajo **Autoservicio para empleados**. Puede cambiar este espacio de trabajo a **Autoservicio**, en lugar del existente.

> [!NOTE]
> Cambiar el nombre del espacio de trabajo **Autoservicio para empleados** también cambia el elemento del menú que usa internamente Dynamics 365 Human Resources. Si anteriormente aplicó personalizaciones de seguridad al elemento del menú **HcmEmployeeSelfServiceWorkspace**, recomendamos aplicar los mismos cambios a **HcmSelfServiceWorkspace** para mantener la paridad.

1. En Human Resources, seleccione **Gestión de personal**, seleccione **Vínculos** y luego seleccione **Parámetros de recursos humanos**.

2. Seleccione la pestaña **Autoservicio para empleados**.

3. En **Nombre para mostrar**, seleccione **Autoservicio**.

   ![Cambiar el nombre del espacio de trabajo de autoservicio a Autoservicio](./media/hr-employee-self-service-workspace-name.png)

4. Seleccione **Guardar**.
