# <a name="marital-status"></a>Estado civil

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe el conjunto de opciones de estado marital para Dynamics 365 Human Resources.

Nombre físico: mshr_dirpersonmaritalstatus

Esta enumeración proporciona el conjunto de opciones de estado marital de los registros personales.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | None | No se ha seleccionado ningún valor.
| 200000001 | Soltero/a | Soltero/a. |
| 200000002 | Casado/a | Casado/a. |
| 200000003 | Divorciado/a | Divorciado/a. |
| 200000004 | Viudo/a | Viudedad. |
