# <a name="payroll---no-yes-option-set"></a>Nómina - conjunto de opciones No Sí

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tema describe la opción No Sí para Dynamics 365 Human Resources.

## <a name="option-set-no-yes"></a>Conjunto de opciones: No Sí

Nombre físico: mshr_noyes

Esta enumeración proporciona el conjunto de opciones para las propiedades booleanas típicas en entidades virtuales de Recursos Humanos.

| Valor | Etiqueta | Descripción |
| --- | --- | --- |
| 200000000 | N.º | No. |
| 200000001 | Sí | Sí. |
