# <a name="create-an-operating-unit"></a><span data-ttu-id="5871e-103">Creación de una unidad operativa</span><span class="sxs-lookup"><span data-stu-id="5871e-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="5871e-104">Una unidad operativa es una organización que se usa para dividir el control de los recursos económicos y los procesos operativos de un negocio.</span><span class="sxs-lookup"><span data-stu-id="5871e-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="5871e-105">Las personas de una unidad operativa tienen el deber de optimizar el uso de recursos escasos, mejorar los procesos y responder de su rendimiento.</span><span class="sxs-lookup"><span data-stu-id="5871e-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="5871e-106">Los tipos de unidades operativas incluyen los centros de coste, las unidades de negocio, los flujos de valor y los departamentos.</span><span class="sxs-lookup"><span data-stu-id="5871e-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="5871e-107">Realice el procedimiento siguiente para crear una unidad operativa.</span><span class="sxs-lookup"><span data-stu-id="5871e-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="5871e-108">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="5871e-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="5871e-109">Vaya a **Panel de exploración > Módulos > Administración de la organización > Organizaciones > Unidades operativas**.</span><span class="sxs-lookup"><span data-stu-id="5871e-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="5871e-110">Haga clic en **Nueva** para abrir el cuadro de diálogo desplegable.</span><span class="sxs-lookup"><span data-stu-id="5871e-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="5871e-111">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="5871e-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="5871e-112">Seleccione el tipo de unidad operativa que desea crear.</span><span class="sxs-lookup"><span data-stu-id="5871e-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="5871e-113">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="5871e-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="5871e-114">En el campo **Nombre**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="5871e-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="5871e-115">Expanda la sección **General** si es necesario.</span><span class="sxs-lookup"><span data-stu-id="5871e-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="5871e-116">Proporcione información general acerca de la unidad operativa, como el número de identificación, el número DUNS y el administrador.</span><span class="sxs-lookup"><span data-stu-id="5871e-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="5871e-117">Expanda la sección **Dirección** si es necesario.</span><span class="sxs-lookup"><span data-stu-id="5871e-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="5871e-118">Indique la información de dirección, como la calle y el número, el código postal y la ciudad.</span><span class="sxs-lookup"><span data-stu-id="5871e-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="5871e-119">Haga clic en **Agregar** para especificar un nuevo registro de dirección o haga clic en Editar para modificar uno existente.</span><span class="sxs-lookup"><span data-stu-id="5871e-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="5871e-120">Expanda la sección **Información de contacto** si es necesario.</span><span class="sxs-lookup"><span data-stu-id="5871e-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="5871e-121">Especifique información sobre los métodos de comunicación, como direcciones de correo electrónico, URL y números de teléfono.</span><span class="sxs-lookup"><span data-stu-id="5871e-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="5871e-122">Para especificar un nuevo registro de comunicación, haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="5871e-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="5871e-123">Para modificar un registro de comunicación existente, haga clic en **Más opciones > Avanzado**.</span><span class="sxs-lookup"><span data-stu-id="5871e-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="5871e-124">Haga clic en **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="5871e-124">Click **Save**.</span></span>
