# <a name="hire-an-existing-employee-through-recruiting"></a><span data-ttu-id="950ce-103">Contratar a un empleado existente a través de la contratación</span><span class="sxs-lookup"><span data-stu-id="950ce-103">Hire an existing employee through recruiting</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="950ce-104">A veces las vacantes se pueden ocupar por candidatos que ya son empleados de la organización.</span><span class="sxs-lookup"><span data-stu-id="950ce-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="950ce-105">Este procedimiento le muestra los pasos para contratar a un empleado existente a través del proceso de contratación.</span><span class="sxs-lookup"><span data-stu-id="950ce-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="950ce-106">En este procedimiento, ya hay configurado un proyecto de contratación, y un empleado existente ha enviado ya una solicitud para el proyecto de contratación.</span><span class="sxs-lookup"><span data-stu-id="950ce-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="950ce-107">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="950ce-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="950ce-108">Vaya a Recursos humanos > Contratación > Solicitudes > Solicitudes.</span><span class="sxs-lookup"><span data-stu-id="950ce-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>
2. <span data-ttu-id="950ce-109">En la lista, localice la solicitud del empleado que desea contratar.</span><span class="sxs-lookup"><span data-stu-id="950ce-109">In the list, find application for the employee that you would like to hire.</span></span> <span data-ttu-id="950ce-110">Ejemplo: 00002 John Emory</span><span class="sxs-lookup"><span data-stu-id="950ce-110">Example:  00002  John Emory</span></span>
3. <span data-ttu-id="950ce-111">Haga clic en Estado de la solicitud.</span><span class="sxs-lookup"><span data-stu-id="950ce-111">Click Application status.</span></span>
    * <span data-ttu-id="950ce-112">El estado de la solicitud indica en qué punto del proceso de contratación está la solicitud.</span><span class="sxs-lookup"><span data-stu-id="950ce-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="950ce-113">Cada uno de estos pasos es opcional.</span><span class="sxs-lookup"><span data-stu-id="950ce-113">Each of these steps is optional.</span></span> <span data-ttu-id="950ce-114">Normalmente, una solicitud pasaría por diferentes estados en el siguiente orden: recibido, confirmado y entrevistado.</span><span class="sxs-lookup"><span data-stu-id="950ce-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="950ce-115">Después del proceso de entrevista, se tomará una decisión de contratación.</span><span class="sxs-lookup"><span data-stu-id="950ce-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="950ce-116">Haga clic en Cambiar puesto.</span><span class="sxs-lookup"><span data-stu-id="950ce-116">Click Change position.</span></span>
5. <span data-ttu-id="950ce-117">Seleccione el puesto para el que va a contratar al empleado.</span><span class="sxs-lookup"><span data-stu-id="950ce-117">Select the position that you are hiring the employee into.</span></span>

7. <span data-ttu-id="950ce-119">En el campo Fecha de finalización de asignación, escriba la fecha en la que el empleado dejará de trabajar en su puesto actual.</span><span class="sxs-lookup"><span data-stu-id="950ce-119">In the Assignment end date, enter the date that the employee will stop working in their current position.</span></span>
    * <span data-ttu-id="950ce-120">La fecha de inicio para el nuevo puesto y la fecha de finalización del puesto anterior se pueden superponer.</span><span class="sxs-lookup"><span data-stu-id="950ce-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="950ce-121">Esto puede suceder si una persona está realizando tareas en ambos puestos durante un período de transición.</span><span class="sxs-lookup"><span data-stu-id="950ce-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>  
8. <span data-ttu-id="950ce-122">También puede seleccionar un código de motivo.</span><span class="sxs-lookup"><span data-stu-id="950ce-122">Optionally, you can select a reason code.</span></span> <span data-ttu-id="950ce-123">Por ejemplo, Reorganización.</span><span class="sxs-lookup"><span data-stu-id="950ce-123">Example: Reorganization</span></span>
9. <span data-ttu-id="950ce-124">Haga clic en Cambiar puesto.</span><span class="sxs-lookup"><span data-stu-id="950ce-124">Click Change position.</span></span>
    * <span data-ttu-id="950ce-125">También puede cambiar la compensación en este momento.</span><span class="sxs-lookup"><span data-stu-id="950ce-125">You may also change compensation at this time.</span></span> <span data-ttu-id="950ce-126">Si no asigna una compensación en este momento, puede cambiarlo yendo al formulario del trabajador, seleccionando la ficha Compensación y eligiendo Plan fijo.</span><span class="sxs-lookup"><span data-stu-id="950ce-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="950ce-127">Tras seleccionar Cambiar puesto, el estado de la solicitud se actualizará a Empleado.</span><span class="sxs-lookup"><span data-stu-id="950ce-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
