# <a name="change-the-banner-or-logo"></a><span data-ttu-id="09735-103">Cambiar el encabezado o el logotipo</span><span class="sxs-lookup"><span data-stu-id="09735-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="09735-104">El procedimiento siguiente muestra los pasos que los administradores del sistema pueden usar para actualizar la imagen del banner o del logotipo que se muestra para una entidad jurídica.</span><span class="sxs-lookup"><span data-stu-id="09735-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="09735-105">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="09735-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="09735-106">Vaya a Administración de la organización > Organizaciones > Entidades jurídicas.</span><span class="sxs-lookup"><span data-stu-id="09735-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="09735-107">En la lista de la izquierda, seleccione la entidad jurídica para la que desea actualizar el banner o el logotipo.</span><span class="sxs-lookup"><span data-stu-id="09735-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="09735-108">Si ya está seleccionada, vaya al paso siguiente.</span><span class="sxs-lookup"><span data-stu-id="09735-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="09735-109">Expanda la sección Imágenes.</span><span class="sxs-lookup"><span data-stu-id="09735-109">Expand the Images section.</span></span> <span data-ttu-id="09735-110">Si ya está expandida, vaya al paso siguiente.</span><span class="sxs-lookup"><span data-stu-id="09735-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="09735-111">Haga clic en Editar.</span><span class="sxs-lookup"><span data-stu-id="09735-111">Click Edit.</span></span>
5. <span data-ttu-id="09735-112">En el campo Tipo de imagen de empresa de panel de información, seleccione “Pancarta”.</span><span class="sxs-lookup"><span data-stu-id="09735-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="09735-113">Haga clic en Cambiar.</span><span class="sxs-lookup"><span data-stu-id="09735-113">Click Change.</span></span>
    * <span data-ttu-id="09735-114">Sugerencia: La resolución ideal para una imagen de banner es 1920 x 281 píxeles.</span><span class="sxs-lookup"><span data-stu-id="09735-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="09735-115">El ancho ideal para una imagen de logotipo es 350 píxeles.</span><span class="sxs-lookup"><span data-stu-id="09735-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="09735-116">Cuando se complete la carga, deberá ver la nueva imagen del banner o logotipo en la sección de la Imagen de panel de información.</span><span class="sxs-lookup"><span data-stu-id="09735-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="09735-117">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="09735-117">Click Save.</span></span>
8. <span data-ttu-id="09735-118">Vaya a Común > Común > Panel predeterminado.</span><span class="sxs-lookup"><span data-stu-id="09735-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="09735-119">Debe ver la nueva imagen de banner o logotipo en el panel de información.</span><span class="sxs-lookup"><span data-stu-id="09735-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="09735-120">Es posible que tenga que cambiar su empresa, con el selector de empresa, a aquella para la que cargó el banner.</span><span class="sxs-lookup"><span data-stu-id="09735-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
