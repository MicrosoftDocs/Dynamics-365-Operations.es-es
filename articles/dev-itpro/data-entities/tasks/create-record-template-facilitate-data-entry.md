12. <span data-ttu-id="37054-120">Haga clic en Plantilla de usuario.</span><span class="sxs-lookup"><span data-stu-id="37054-120">Click User template.</span></span>
13. <span data-ttu-id="37054-121">En el campo Nombre, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="37054-121">In the Name field, type a value.</span></span>
