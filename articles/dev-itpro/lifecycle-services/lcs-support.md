# <a name="get-support-for-finance-and-operations-or-lifecycle-services-lcs"></a>Obtener soporte técnico para Finance and Operations o Lifecycle Services (LCS)

[!include [banner](../includes/banner.md)]

Este artículo explica cómo obtener ayuda con Microsoft Dynamics 365 for Finance and Operations, Edición Empresas o Microsoft Dynamics Lifecycle Services (LCS). 

<a name="contact-support"></a>Ponerse en contacto con el soporte técnico
---------------

Use la información de este artículo para ponerse en contacto con Microsoft si tiene consultas técnicas sobre Finances and Operations, o si necesita ayuda para acceder a Microsoft Dynamics Lifecycle Services (LCS).

### <a name="finance-and-operations-support"></a>Soporte técnico de Finance and Operations

Para obtener ayuda para Finance and Operations or LCS, use los recursos en la tabla siguiente. Estos recursos aparece listados en el orden que se recomienda que siga cuando está solucionando problemas.

<table>
<colgroup>
<col width="33%" />
<col width="33%" />
</colgroup>
<thead>
<tr class="header">
<th>Tarea</th>
<th>Más información</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>Pregunte a la comunidad.</td>
<td>Vaya a la página <a href="http://go.microsoft.com/fwlink/?LinkId=221068"><span style="color: #0066cc;">Pregunte a la comunidad</span></a> para encontrar ayuda con sus preguntas de la comunidad de Dynamics.</td>
</tr>
<tr class="even">
<td>Utilice la herramienta de búsqueda del problema.</td>
<td>En <a href="https://lcs.dynamics.com/"><span style="color: #0066cc;">CD</span></a>, utilice la búsqueda de temas para buscar rápidamente los artículos, hotfix, y las soluciones de Microsoft Knowledge Base (KB) para problemas notificados de Finance and Operations. Puede consultar qué problemas están en proceso de corregirse o ya han sido corregidos para un área funcional específica. Para obtener más información, consulte <a href="issue-search-lcs.md"><span style="color: #0066cc;">Búsqueda de problema (Lifecycle Services, LCS)</span></a>.</td>
</tr>
<tr class="odd">
<td>Utilice la herramienta de Soporte técnico con tecnolgía de la nube.</td>
<td>En <a href="https://lcs.dynamics.com/"><span style="color: #0066cc;">LCS</span></a>, <strong>Soporte</strong> tile es una herramienta que ayuda a gestionar incidentes. Para registrar problemas directamente a Microsoft, vaya <strong>Support</strong> tile en el proyecto del LCS. Desde allí, puede enviar problemas de dos maneras:
<ul>
<li>En la <strong>pestaña activa</strong> seleccione su problema y haga clic en <strong>Enviar a Microsoft</strong>.</li>
<li>En la<strong>la pestaña de Enviado a Microsoft</strong> haga clic en <strong>Enviar un nuevo incidente</strong> para enviar el problema.</li>
</ul>
Nota: Microsoft Premier Customers and Partners también puede enviar tickets de ayuda utilizando los pasos de anteriores.</td>
</tr>
<tr class="even">
<td>Abra un ticket de ayuda con el equipo de Finance and Operations.</td>
<td>Si tiene problemas cuando intenta registrar un ticket a Microsoft desde <strong>Support</strong> tile en LCS, vea <a href="https://mbs.microsoft.com/customersource/northamerica/ax/support/support-news/global_support_contacts_eng"><span style="color: #0066cc;">Póngase en contacto con el soporte técico de Microsoft Business Solutions</span></a> para obtener información para ponerse en contacto con un centro de operaciones regional Microsoft Dynamics Regional Operations Center. El equipo de soporte técnico puede ayudarle a resolver los siguientes tipos de problemas técnicos:
<ul>
<li>Errores que aparecen cuando se usa el LCS o problemas que aparecen al intentar obtener acceso a LCS.</li>
<li>Errores que aparecen cuando se usa Finance and Operations.</li>
</ul>
Para las consultas técnicas relacionadas con iniciar sesión y acceder a CustomerSource, e-learning, Búsqueda, Knowledge Base, descarga de archivos, y errores de página o de sistema, vaya a <a href="https://mbs2.microsoft.com/members/VoiceSupport/VoiceSupportInternal.aspx"><span style="color: #0066cc;">Soporte técnico de la página web</span></a>.</td>
</tr>
<tr class="odd">
<td>Obtener ayuda con preguntas sobre la licencia de Finance and Operations.</td>
<td>Póngase en contacto con su socio, distribuidor de grandes cuentas (LAR), o con un representante de ventas de Microsoft.</td>
</tr>
<tr class="even">
<td>Utilice el panel de la supervisión del evento del LCS.</td>
<td>Para ver el estado de la salud actual de la plataforma y servicios, vaya a <strong>Entornos</strong> a la sección de su proyecto de LCS.</td>
</tr>
</tbody>
</table>

Para obtener más información acerca de oferta de soporte de Microsoft Dynamics, consulte [Soporte técnico en línea de Microsoft Dynamics](https://www.microsoft.com/en-us/dynamics/dynamics-online-support.aspx).
