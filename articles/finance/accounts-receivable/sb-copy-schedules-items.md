# <a name="copy-a-billing-schedule"></a>Copiar un programa de facturación

[!include [banner](../includes/banner.md)]

Puedes usar la página **Copiar programa** para crear un nuevo programa de facturación a partir de un programa de facturación existente. El nuevo programa de facturación tendrá la misma configuración que el programa de facturación original. También puede usar la página para copiar elementos de un programa de facturación a otro.

## <a name="copy-a-billing-schedule"></a>Copiar un programa de facturación

Para copiar un programación de facturación, siga estos pasos.

1. En la página de listas **Todos los programas de facturación** o **Programas de facturación activos**, seleccione un programa de facturación existente.
2. En la pestaña **Programa de facturación**, seleccione **Copiar programa**.
3. En el campo **Cuenta de cliente**, seleccione la cuenta de cliente.
4. Actualice las fechas de inicio y finalización de la facturación según sea necesario.
5. Actualice los valores de los siguientes campos opcionales según sea necesario: **Cuenta de usuario final**, **Cuenta de socio**, **Referencia del cliente**, **Solicitud de cliente** y **Fecha de inicio del aplazamiento**.
6. Seleccione **Ver versión preliminar** para ver y seleccionar las líneas de programa de facturación que desee copiar.
7. Seleccione **Copiar**.

Una vez que se completa el proceso de copia, el nuevo programa de facturación aparece en la parte inferior de la página de listas **Todos los programas de facturación** o **Programas de facturación activos** y en la página **Programas de facturación**.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Copie elementos de un programa de facturación a otro

1. En la página de listas **Todos los programas de facturación** o **Programas de facturación activos**, seleccione un programa de facturación existente.
2. En la pestaña **Programa de facturación**, seleccione **Copiar programa**.
3. En el campo **Cuenta de cliente**, seleccione la cuenta de cliente.
4. En el campo **Número de programa de facturación**, seleccione un número de programa de facturación.
4. Actualice las fechas de inicio y finalización de la facturación.
5. Actualice los valores de los siguientes campos según sea necesario: **Cuenta de usuario final**, **Cuenta de socio**, **Referencia del cliente**, **Solicitud de cliente** y **Fecha de inicio del aplazamiento**.
6. Seleccione **Ver versión preliminar** para ver y seleccionar las líneas de programa de facturación que desee copiar en un programa de facturación existente.
7. Seleccione **Copiar**.

Una vez que se complete el proceso de copia, revise el programa de facturación en el que copió los artículos y asegúrese de que los artículos se agregaron como esperaba.
