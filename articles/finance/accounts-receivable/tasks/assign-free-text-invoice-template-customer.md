[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8c927-104">Esta tarea muestra cómo asignar una plantilla de factura de texto libre a un cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-104">This task demonstrates how to assign a free text invoice template to a customer.</span></span> <span data-ttu-id="8c927-105">Esta tarea utiliza la empresa de demostración USMF y se va a utilizar para el usuario responsable de gestionar y de procesar facturas de cuentas por cobrar.</span><span class="sxs-lookup"><span data-stu-id="8c927-105">This task uses the USMF demo company and is intended for the user who is responsible for managing and processing A/R invoices.</span></span>

1. <span data-ttu-id="8c927-106">En el **Panel de exploración**, vaya a **Módulos > Clientes > Clientes > Todos los clientes**.</span><span class="sxs-lookup"><span data-stu-id="8c927-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Customers > All customers**.</span></span>
2. <span data-ttu-id="8c927-107">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="8c927-107">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="8c927-108">En el **panel de acciones**, haga clic en **Factura**.</span><span class="sxs-lookup"><span data-stu-id="8c927-108">On the **Action Pane**, click **Invoice**.</span></span>
4. <span data-ttu-id="8c927-109">Haga clic en **Facturas periódicas**.</span><span class="sxs-lookup"><span data-stu-id="8c927-109">Click **Recurring invoices**.</span></span> <span data-ttu-id="8c927-110">Use esta página para asignar plantillas de factura de texto libre a clientes y especificar la frecuencia con la que se enviarán facturas al cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-110">Use this page to assign free text invoice templates to customers and specify how frequently invoices will be sent to the customer.</span></span>  
5. <span data-ttu-id="8c927-111">Haga clic en **Nuevo** para asignar una plantilla nueva al cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-111">Click **New** to assign a new template to the customer.</span></span>

7. <span data-ttu-id="8c927-113">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="8c927-113">In the list, find and select the desired record.</span></span>
8. <span data-ttu-id="8c927-114">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="8c927-114">In the list, click the link in the selected row.</span></span>
9. <span data-ttu-id="8c927-115">En el campo **Fecha de inicio de la facturación**, especifique la fecha en que se generará la primera factura.</span><span class="sxs-lookup"><span data-stu-id="8c927-115">In the **Billing start date** field, enter the date when the first invoice will be generated.</span></span>
10. <span data-ttu-id="8c927-116">En la sección **Fin de periodicidad**, especifique una fecha final recurrente.</span><span class="sxs-lookup"><span data-stu-id="8c927-116">In the **Recurrence end** section, enter a recurring end date.</span></span>  
    * <span data-ttu-id="8c927-117">Seleccione alguna de las siguientes opciones: No hay fecha final: las facturas se generarán de manera indefinida hasta que se elimine la plantilla de la cuenta del cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-117">Select one of the following: No end date – Invoices will be generated indefinitely until the template is removed from the customer account.</span></span>
    * <span data-ttu-id="8c927-118">Fecha de finalización de la facturación: seleccione esta opción y escriba la última fecha en la que se puede generar la factura.</span><span class="sxs-lookup"><span data-stu-id="8c927-118">Billing end date – Select this option and enter the last date that the invoice can be generated.</span></span>

12. <span data-ttu-id="8c927-122">En la sección **Generar facturas periódicas mediante los valores predeterminados**, seleccione la plantilla de factura de texto libre o la cuenta de cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-122">In the **Generate recurring invoices by using the default values from** section, select either free text invoice template or the customer account.</span></span> <span data-ttu-id="8c927-123">Seleccione si se debe usar la plantilla de factura de servicios o la cuenta de cliente para determinar los valores predeterminados del idioma, el perfil de contabilización, el grupo de impuestos, el grupo de impuestos de artículos, el código de lista, el país o región de la entrega, la divisa, las condiciones de pago, la forma de pago, la especificación del pago, la programación del pago, el descuento por pronto pago, las dimensiones financieras y el resguardo de la transferencia del giro bancario cuando se crean las facturas.</span><span class="sxs-lookup"><span data-stu-id="8c927-123">Select whether to use the free text invoice template or the customer account to determine the default values for the language, posting profile, sales tax group, item sales tax group, list code, country/region for delivery, currency, terms of payment, method of payment, payment specification, payment schedule, cash discount, financial dimensions, and giro money transfer slip when invoices are created.</span></span>  
13. <span data-ttu-id="8c927-124">En el campo **Patrón de periodicidad**, seleccione el patrón de repetición.</span><span class="sxs-lookup"><span data-stu-id="8c927-124">In the **Recurrence pattern** field, select the recurrence pattern.</span></span>
    + <span data-ttu-id="8c927-125">Diariamente: seleccione esta opción y especifique el número de días en el campo Por.</span><span class="sxs-lookup"><span data-stu-id="8c927-125">Daily – Select this option and enter the number of days in the Per field.</span></span> <span data-ttu-id="8c927-126">Por ejemplo, si escribe 15, se generará una factura cada 15 días para este cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-126">For example, if you enter 15, an invoice will be generated every 15 days for this customer.</span></span>
    + <span data-ttu-id="8c927-127">Semanal: seleccione esta opción y especifique el número de semanas en el campo Por.</span><span class="sxs-lookup"><span data-stu-id="8c927-127">Weekly - Select this option and enter the number of weeks in the Per field.</span></span> <span data-ttu-id="8c927-128">Por ejemplo, si escribe 2, se generará una factura cada dos semanas para este cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-128">For example, if you enter 2, an invoice will be generated every two weeks for this customer.</span></span>
    + <span data-ttu-id="8c927-129">Mensual: seleccione esta opción y especifique el número de meses en el campo Por.</span><span class="sxs-lookup"><span data-stu-id="8c927-129">Monthly - Select this option and enter the number of months in the Per field.</span></span> <span data-ttu-id="8c927-130">Por ejemplo, si escribe 6, se generará una factura cada seis meses para este cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-130">For example, if you enter 6, an invoice will be generated every six months for this customer.</span></span>
    + <span data-ttu-id="8c927-131">Anual: seleccione esta opción y especifique el número de años en el campo Por.</span><span class="sxs-lookup"><span data-stu-id="8c927-131">Yearly – Select this option and enter the number of years in the Per field.</span></span> <span data-ttu-id="8c927-132">Por ejemplo, si escribe 2, se generará una factura cada dos años para este cliente.</span><span class="sxs-lookup"><span data-stu-id="8c927-132">For example, if you enter 2, an invoice will be generated every two years for this customer.</span></span>  
14. <span data-ttu-id="8c927-133">En el campo **Por**, especifique un número.</span><span class="sxs-lookup"><span data-stu-id="8c927-133">In the **Per** field, enter a number.</span></span>
