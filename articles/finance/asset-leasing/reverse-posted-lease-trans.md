# <a name="reverse-posted-lease-transactions"></a><span data-ttu-id="28fdb-104">Revertir transacciones de arrendamiento contabilizadas</span><span class="sxs-lookup"><span data-stu-id="28fdb-104">Reverse posted lease transactions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="28fdb-105">Cualquier transacción que se cree mediante Arrendamiento de activos se puede revertir.</span><span class="sxs-lookup"><span data-stu-id="28fdb-105">Any transaction that is created through Asset leasing can be reversed.</span></span> <span data-ttu-id="28fdb-106">Las transacciones que se revierten mediante Arrendamiento de activos actualizan sus datos de arrendamiento.</span><span class="sxs-lookup"><span data-stu-id="28fdb-106">Transactions that are reversed through Asset leasing update your lease data.</span></span> <span data-ttu-id="28fdb-107">Por lo tanto, también actualizan los valores en libros del pasivo por arrendamiento y el activo por derecho de uso (ROU).</span><span class="sxs-lookup"><span data-stu-id="28fdb-107">Therefore, they also update the carrying values of the lease liability and right-of-use (ROU) asset.</span></span>

<span data-ttu-id="28fdb-108">Para crear una transacción de reversión para un arrendamiento, siga estos pasos.</span><span class="sxs-lookup"><span data-stu-id="28fdb-108">To create a reversing transaction for a lease, follow these steps.</span></span>

1. <span data-ttu-id="28fdb-109">En la página **Transacciones de activos** o en la página **Transacciones de pasivo**, seleccione la transacción y luego seleccione **Revertir transacción**.</span><span class="sxs-lookup"><span data-stu-id="28fdb-109">On either the **Asset transactions** page or the **Liability transactions** page, select the transaction, and then select **Reverse transaction**.</span></span>

<span data-ttu-id="28fdb-116">Cuando selecciona **Seguimiento inverso**, aparece un cuadro de diálogo que muestra tanto las transacciones originales como las transacciones revertidas, junto con un número vinculado que se conoce como *número de seguimiento*.</span><span class="sxs-lookup"><span data-stu-id="28fdb-116">When you select **Reverse tracing**, a dialog box appears that shows both the original transactions and the reversed transactions, together with a linked number that is known as a *trace number*.</span></span> <span data-ttu-id="28fdb-117">Para que las reversiones sean más fáciles de entender y mejorar la visibilidad, también puede realizar un seguimiento de las reversiones mediante las programaciones de arrendamiento.</span><span class="sxs-lookup"><span data-stu-id="28fdb-117">To make the reversals easier to understand and to improve visibility, you can also track reversals by using the lease schedules.</span></span>

<span data-ttu-id="28fdb-118">El campo **Último número de diario** de la página **Programación** muestra los números de diario de las transacciones.</span><span class="sxs-lookup"><span data-stu-id="28fdb-118">The **Latest journal number** field on the **Schedule** page shows the journal numbers of transactions.</span></span> <span data-ttu-id="28fdb-119">Cuando se invierte una transacción, este campo se actualiza con el número de diario de una transacción de reversión.</span><span class="sxs-lookup"><span data-stu-id="28fdb-119">When a transaction is reversed, this field is updated with the journal number of a reversing transaction.</span></span> <span data-ttu-id="28fdb-120">Además, la casilla **Invertido** está seleccionada para indicar que la transacción se ha invertido y el campo **Registrado** está seleccionado.</span><span class="sxs-lookup"><span data-stu-id="28fdb-120">Additionally, the **Reversed** check box is selected to indicate that the transaction is reversed, and the **Posted** field is selected.</span></span>

## <a name="revoke-a-reversed-transaction"></a><span data-ttu-id="28fdb-121">Revocar una transacción revertida</span><span class="sxs-lookup"><span data-stu-id="28fdb-121">Revoke a reversed transaction</span></span>
