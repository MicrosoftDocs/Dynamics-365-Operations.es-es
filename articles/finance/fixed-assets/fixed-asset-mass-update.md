# <a name="fixed-asset-mass-update"></a><span data-ttu-id="a7725-103">Actualización masiva de activos fijos</span><span class="sxs-lookup"><span data-stu-id="a7725-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a7725-104">Si utiliza libros, puede modificar las convenciones de depreciación de grupos de activos que forman parte del mismo libro.</span><span class="sxs-lookup"><span data-stu-id="a7725-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="a7725-105">Por ejemplo, si se encuentra en los Estados Unidos y ha puesto más de un 40 por ciento de sus activos en servicio durante el cuarto trimestre del año, deberá utilizar la convención de depreciación de mitad del trimestre.</span><span class="sxs-lookup"><span data-stu-id="a7725-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="a7725-106">Puede utilizar el proceso de actualización masiva para modificar todos los activos a los que se deba aplicar la nueva convención de depreciación</span><span class="sxs-lookup"><span data-stu-id="a7725-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="a7725-107">Cuando se actualiza la convención de depreciación de los activos, se eliminan todas las transacciones de depreciación que existan para dichos activos.</span><span class="sxs-lookup"><span data-stu-id="a7725-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="a7725-108">Eliminará también todas las transacciones para los ajustes de depreciación, las transacciones para la depreciación de bonificación y las transacciones para la depreciación extraordinaria para dichos activos.</span><span class="sxs-lookup"><span data-stu-id="a7725-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="a7725-109">Para actualizar la convención de amortizaciones para los activos que se han sido cancelado, primero debe eliminar las transacciones de cancelación existentes.</span><span class="sxs-lookup"><span data-stu-id="a7725-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="a7725-110">También deben eliminar todas las transacciones que se han generado debido al proceso de cancelación.</span><span class="sxs-lookup"><span data-stu-id="a7725-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="a7725-111">Tras actualizar la convención de depreciación de los activos, puede procesar la depreciación y depreciación extraordinaria de cada activo.</span><span class="sxs-lookup"><span data-stu-id="a7725-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="a7725-112">También puede realizar los ajustes manuales de depreciación, si se requieren.</span><span class="sxs-lookup"><span data-stu-id="a7725-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
