# <a name="number-of-books-per-journal"></a><span data-ttu-id="e63d9-104">Número de libros por diario</span><span class="sxs-lookup"><span data-stu-id="e63d9-104">Number of books per journal</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e63d9-105">Este tema describe la relación entre diarios y libros de activos cuando crea una propuesta de adquisición o depreciación de activos fijos a través de un trabajo por lotes.</span><span class="sxs-lookup"><span data-stu-id="e63d9-105">This topic describes the relationship between journals and asset books when you create a fixed asset acquisition or depreciation proposal through a batch job.</span></span> <span data-ttu-id="e63d9-106">Puede definir el número máximo de libros que se incluyen para cada adquisición y para la depreciación utilizando los campos de la sección **Número de libros por diario**, en la pestaña **General** de la página **Parámetros de activos fijos** (**Activos fijos \> Configurar \> Parámetros de activos fijos**).</span><span class="sxs-lookup"><span data-stu-id="e63d9-106">You can define the maximum number of books that are included for each acquisition and for depreciation by using the fields in the **Number of books per journal** section on the **General** tab of the **Fixed assets parameters** page (**Fixed assets \> Setup \> Fixed assets parameters**).</span></span> <span data-ttu-id="e63d9-107">Estos campos le permiten distribuir el número de libros de activos por diario de adquisiciones y diario de depreciación.</span><span class="sxs-lookup"><span data-stu-id="e63d9-107">These fields let you distribute the number of asset books per acquisition journal and depreciation journal.</span></span>

<span data-ttu-id="e63d9-108">Para una propuesta de adquisición, el valor predeterminado es al menos 10.000 libros.</span><span class="sxs-lookup"><span data-stu-id="e63d9-108">For an acquisition proposal, the default value is at least 10,000 books.</span></span> <span data-ttu-id="e63d9-109">Para una propuesta de depreciación, el valor predeterminado es al menos 2000 libros.</span><span class="sxs-lookup"><span data-stu-id="e63d9-109">For a depreciation proposal, the default value is at least 2,000 books.</span></span>

<span data-ttu-id="e63d9-110">Por ejemplo, si hay 4000 activos fijos y dos libros están asociados con cada activo, un libro se registrará en la capa actual y el otro libro se registrará en la capa de impuestos.</span><span class="sxs-lookup"><span data-stu-id="e63d9-110">For example, if there are 4,000 fixed assets, and two books are associated with each asset, one book will be posted to the current layer, and the other book will be posted to the tax layer.</span></span> <span data-ttu-id="e63d9-111">Si adquiere 4000 activos fijos mediante el procesamiento por lotes, el trabajo por lotes que crea un diario de adquisición de activos fijos contendrá 4000 libros de activos.</span><span class="sxs-lookup"><span data-stu-id="e63d9-111">If you acquire 4,000 fixed assets through batch processing, the batch job that creates one fixed asset acquisition journal will contain 4,000 asset books.</span></span>

> <span data-ttu-id="e63d9-112">El diario que se crea seguirá utilizándose hasta que se complete el trabajo por lotes.</span><span class="sxs-lookup"><span data-stu-id="e63d9-112">The journal that is created will continue to be used until the batch job is completed.</span></span>
>

<span data-ttu-id="e63d9-114">Puede utilizar el procesamiento por lotes para ejecutar la depreciación del mismo conjunto de activos adquiridos.</span><span class="sxs-lookup"><span data-stu-id="e63d9-114">You can use  batch processing to run depreciation for the same set of acquired assets.</span></span> <span data-ttu-id="e63d9-115">Por ejemplo, un trabajo por lotes crea dos diarios de depreciación, cada uno de los cuales consta de 2000 libros de activos.</span><span class="sxs-lookup"><span data-stu-id="e63d9-115">For example, a batch job creates two depreciation journals, each of which consists of 2,000 asset books.</span></span> <span data-ttu-id="e63d9-116">El primer diario contendrá libros asociados con los activos fijos numerados del 1 al 2000.</span><span class="sxs-lookup"><span data-stu-id="e63d9-116">The first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,000.</span></span> <span data-ttu-id="e63d9-117">El segundo diario contendrá entonces los libros asociados con los activos fijos numerados del 2001 al 4000.</span><span class="sxs-lookup"><span data-stu-id="e63d9-117">The second journal will then contain books that are associated with the fixed assets that are numbered 2,001 through 4,000.</span></span>

<span data-ttu-id="e63d9-118">El trabajo de procesamiento por lotes excluye los libros cerrados.</span><span class="sxs-lookup"><span data-stu-id="e63d9-118">The batch processing job excludes closed books.</span></span> <span data-ttu-id="e63d9-119">Por ejemplo, en un trabajo por lotes para depreciación, se cierran 10 de los primeros 2000 libros.</span><span class="sxs-lookup"><span data-stu-id="e63d9-119">For example, in a batch job for depreciation, 10 of the first 2,000 books are closed.</span></span> <span data-ttu-id="e63d9-120">En este caso, el primer diario contendrá libros asociados con los activos fijos numerados del 1 al 2011.</span><span class="sxs-lookup"><span data-stu-id="e63d9-120">In this case, the first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,011.</span></span> <span data-ttu-id="e63d9-121">El segundo diario contendrá entonces los libros asociados con los activos fijos numerados del 2012 al 4000.</span><span class="sxs-lookup"><span data-stu-id="e63d9-121">The second journal will then contain books that are associated with the fixed assets that are numbered 2,012 through 4,000.</span></span>

<span data-ttu-id="e63d9-122">El límite en la cantidad de libros se aplica si no existen id. de activos duplicados en el mismo diario.</span><span class="sxs-lookup"><span data-stu-id="e63d9-122">The limit on the number of books is applied if duplicate asset IDs don't exist in the same journal.</span></span> <span data-ttu-id="e63d9-123">Sin embargo, si el id. del activo es el mismo que el id. del libro, se puede exceder el número de libros por diario para mantener el id. del activo en el mismo diario.</span><span class="sxs-lookup"><span data-stu-id="e63d9-123">However, if the asset ID is the same as the book ID, the number of books per journal can be exceeded to keep the asset ID in the same journal.</span></span>
