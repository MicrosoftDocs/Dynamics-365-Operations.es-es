[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ebc45-104">Este tema explica cómo desechar activo fijo mediante la factura de servicios.</span><span class="sxs-lookup"><span data-stu-id="ebc45-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="ebc45-105">En el Panel de exploración, vaya a **Módulos > Clientes > Facturas > Todas las facturas de servicios**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="ebc45-106">Seleccione **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-106">Select **New**.</span></span>
3. <span data-ttu-id="ebc45-107">En el campo **Cuenta de cliente**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="ebc45-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="ebc45-108">Valide fecha de **Factura** predeterminada y edítela si procede.</span><span class="sxs-lookup"><span data-stu-id="ebc45-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="ebc45-109">Valide los campos de cabecera predeterminados restantes, como **Divisa** y edítelos si procede.</span><span class="sxs-lookup"><span data-stu-id="ebc45-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="ebc45-110">En la sección **Líneas de factura**, valide los campos de **Descripción** y **Cuenta principal**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="ebc45-111">Valide los campos **Grupo de impuestos predeterminado** y el **Grupo de impuestos de artículos**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="ebc45-112">Especifique el **Precio unitario** o el **Importe** de la venta del activo fijo.</span><span class="sxs-lookup"><span data-stu-id="ebc45-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="ebc45-113">Seleccione la sección **Detalles de línea**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="ebc45-114">Escriba o seleccione un valor en el campo **Activo fijo**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="ebc45-115">Seleccione **Registrar**.</span><span class="sxs-lookup"><span data-stu-id="ebc45-115">Select **Post**.</span></span>
