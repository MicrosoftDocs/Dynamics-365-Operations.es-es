# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="bd5aa-103">Cancelar un activo fijo mediante una factura de servicios</span><span class="sxs-lookup"><span data-stu-id="bd5aa-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bd5aa-104">Este tema explica cómo desechar activo fijo mediante la factura de servicios.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="bd5aa-105">En el Panel de exploración, vaya a **Módulos > Clientes > Facturas > Todas las facturas de servicios**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="bd5aa-106">Seleccione **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-106">Select **New**.</span></span>
3. <span data-ttu-id="bd5aa-107">En el campo **Cuenta de cliente**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="bd5aa-108">Valide fecha de **Factura** predeterminada y edítela si procede.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="bd5aa-109">Valide los campos de cabecera predeterminados restantes, como **Divisa** y edítelos si procede.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="bd5aa-110">En la sección **Líneas de factura**, valide los campos de **Descripción** y **Cuenta principal**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="bd5aa-111">Valide los campos **Grupo de impuestos predeterminado** y el **Grupo de impuestos de artículos**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="bd5aa-112">Especifique el **Precio unitario** o el **Importe** de la venta del activo fijo.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="bd5aa-113">Seleccione la sección **Detalles de línea**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="bd5aa-114">Escriba o seleccione un valor en el campo **Activo fijo**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="bd5aa-115">Seleccione **Registrar**.</span><span class="sxs-lookup"><span data-stu-id="bd5aa-115">Select **Post**.</span></span>
