<span data-ttu-id="4ba55-105">La posición de efectivo es la proyección del flujo de efectivo que se prevé para el corto plazo.</span><span class="sxs-lookup"><span data-stu-id="4ba55-105">Cash position is the projection of cash flow that is forecast for the near term.</span></span> <span data-ttu-id="4ba55-106">Se basa en la proyección de los recibos de efectivo de los clientes que pagan facturas y pedidos pendientes, y también en la proyección de los desembolsos de efectivo que se pagan a los proveedores por las facturas y los pedidos de compra.</span><span class="sxs-lookup"><span data-stu-id="4ba55-106">It's based on the projection of cash receipts from customers that pay outstanding invoices and orders, and also on the projection cash disbursements that are paid to vendors for purchase invoices and orders.</span></span>

<span data-ttu-id="4ba55-107">Cuando el sistema predice los pagos de los clientes, utiliza las predicciones de pago de la función de predicción de pagos del cliente.</span><span class="sxs-lookup"><span data-stu-id="4ba55-107">When the system predicts customer payments, it uses the payment predictions from the customer payment prediction feature.</span></span> <span data-ttu-id="4ba55-108">Sin predicciones de pago, el tiempo promedio que se requiere para convertir una factura de cliente en un pago para cada cliente se usa para calcular una fecha de pago.</span><span class="sxs-lookup"><span data-stu-id="4ba55-108">Without payment predictions, the average time that is required to convert a customer invoice to a payment for each customer is used to calculate a payment date.</span></span> <span data-ttu-id="4ba55-109">Para los pedidos abiertos de clientes, el sistema calcula la fecha de la factura utilizando el número promedio de días para que se facturen las líneas de pedido por cliente.</span><span class="sxs-lookup"><span data-stu-id="4ba55-109">For open customer orders, the system calculates the invoice date by using the average number of days for order lines per customer to be invoiced.</span></span> <span data-ttu-id="4ba55-110">A continuación, utiliza la fecha de la factura como entrada para la función de predicción de pagos.</span><span class="sxs-lookup"><span data-stu-id="4ba55-110">It then uses the invoice date as an input for the payment prediction functionality.</span></span> <span data-ttu-id="4ba55-111">La función de predicción de pagos del cliente calcula una fecha de pago para cada línea de pedido.</span><span class="sxs-lookup"><span data-stu-id="4ba55-111">The customer payment prediction functionality calculates a payment date for each order line.</span></span> 

<span data-ttu-id="4ba55-112">La fecha de pago de las facturas pendientes se estima aproximadamente a partir de las predicciones de pago, seleccionando una fecha que corresponda al percentil cincuenta de la función de distribución acumulativa que se obtiene a partir de las probabilidades del caso predicho.</span><span class="sxs-lookup"><span data-stu-id="4ba55-112">The payment date for outstanding invoices is estimated from the payment predictions by picking a date that corresponds to fiftieth percentile of the cumulative distribution function that's obtained from the predicted bucket's probabilities.</span></span>

<span data-ttu-id="4ba55-113">Se utiliza un enfoque similar para predecir los pagos a los proveedores.</span><span class="sxs-lookup"><span data-stu-id="4ba55-113">A similar approach is used to predict payments to vendors.</span></span> <span data-ttu-id="4ba55-114">Para cada proveedor, el sistema calcula el tiempo promedio que se requiere para convertir una factura de proveedor en un pago.</span><span class="sxs-lookup"><span data-stu-id="4ba55-114">For each vendor, the system calculates the average time that is required to convert a vendor invoice to a payment.</span></span> <span data-ttu-id="4ba55-115">Ese número de días se usa después para calcular la fecha de pago.</span><span class="sxs-lookup"><span data-stu-id="4ba55-115">That number of days is then used to calculate the payment date.</span></span> <span data-ttu-id="4ba55-116">Para los pedidos de proveedores abiertos, el sistema calcula la fecha de la factura considerando el número promedio de días que se requieren para convertir las líneas de pedido en una factura para cada proveedor.</span><span class="sxs-lookup"><span data-stu-id="4ba55-116">For open vendor orders, the system calculates the invoice date by considering the average number of days that is required to convert order lines to an invoice for each vendor.</span></span> <span data-ttu-id="4ba55-117">Luego, el sistema calcula la fecha de pago usando el tiempo promedio para convertir una factura de proveedor en un pago para cada proveedor.</span><span class="sxs-lookup"><span data-stu-id="4ba55-117">The system then calculates the payment date by using the average time to convert a vendor invoice to a payment for each vendor.</span></span>
