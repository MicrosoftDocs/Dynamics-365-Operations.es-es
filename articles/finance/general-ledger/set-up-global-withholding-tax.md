# <a name="set-up-global-withholding-tax"></a>Configurar una retención de impuestos global

Este tema muestra los pasos para configurar una retención de impuestos global para ventas y compras. 

1. Configure las autoridades de retención de impuestos en la página **Autoridades de retención de impuestos**.

2. Configure los períodos de liquidación de retenciones en la página **Períodos de liquidación de retención de impuestos**.

3. Configure el grupo de registro de retenciones en la página **Retención de impuestos > grupo de registro**.

   > [!Note] 
   >
   > La cuenta **Retención de impuestos** se asignará con una cuenta principal con un **Tipo de registro** de Retención de impuestos. La cuenta **Compensación de retención de impuestos** también se asignará con una cuenta principal con un **Tipo de registro** de "Compensación de retención de impuestos". Vaya a **Contabilidad general > Plan de cuentas > Cuentas> Cuentas principales**, configure el **Tipo de registro** en el subformulario **Validación del registro** para las cuentas principales.

4. Configure los códigos de retención de impuestos en la página **Códigos de retención de impuestos**.

5. Configure los grupos de retención de impuestos en la página **Grupos de retención de impuestos**.

6. Configure los tipos de ingresos para retención de impuestos en la página **Tipos de ingresos para retención** **de impuestos**.

7. Configure los grupos de retención de impuestos en la página **Grupos de retención de impuestos** para un elemento o servicio.

8. Configure **Importe mínimo de la factura** en la página **Parámetros de contabilidad general > Retención de impuestos**.
