## <a name="map-subsidiary-main-accounts-to-consolidated-main-accounts"></a><span data-ttu-id="7f2d6-109">Asignar cuentas principales de filiales a cuentas principales consolidadas</span><span class="sxs-lookup"><span data-stu-id="7f2d6-109">Map subsidiary main accounts to consolidated main accounts</span></span>

2. <span data-ttu-id="7f2d6-112">Seleccione un plan de cuentas.</span><span class="sxs-lookup"><span data-stu-id="7f2d6-112">Select a chart of accounts.</span></span> <span data-ttu-id="7f2d6-113">En la ficha desplegable **Cuentas principales**, seleccione una cuenta principal y, a continuación, seleccione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="7f2d6-113">On the **Main accounts** FastTab, select a main account, and then select **Edit**.</span></span>

    1. <span data-ttu-id="7f2d6-120">En la *entidad jurídica subsidiaria*, vaya a **Contabilidad general \> Configuración \> Dimensiones financieras \> Dimensiones financieras**, seleccione una dimensión financiera y, a continuación, seleccione **Valores de dimensión financiera**.</span><span class="sxs-lookup"><span data-stu-id="7f2d6-120">In the *subsidiary legal entity*, go to **General ledger \> Setup \> Financial dimensions \> Financial dimensions**, select a financial dimension, and then select **Financial dimension values**.</span></span>

    2. <span data-ttu-id="7f2d6-128">En la ficha **Criterios**, active la casilla **Usar cuenta de consolidación**.</span><span class="sxs-lookup"><span data-stu-id="7f2d6-128">On the **Criteria** tab, select the **Use consolidation account** check box.</span></span>
    3. <span data-ttu-id="7f2d6-129">En la pestaña **Dimensiones financieras**, seleccione las dimensiones financieras adecuadas.</span><span class="sxs-lookup"><span data-stu-id="7f2d6-129">On the **Financial dimensions** tab, select the appropriate financial dimensions.</span></span>
