# <a name="post-periodic-journals"></a><span data-ttu-id="83358-103">Registrar diarios periódicos</span><span class="sxs-lookup"><span data-stu-id="83358-103">Post periodic journals</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="83358-104">Los diarios periódicos a veces que se denominan diarios periódicos dado que el importe, el texto y otra información se repiten cada vez que se recupera el diario periódico.</span><span class="sxs-lookup"><span data-stu-id="83358-104">Periodic journals are sometimes called recurring journals because the amount, text, and other information are repeated each time that the periodic journal is retrieved.</span></span> <span data-ttu-id="83358-105">Al crear el diario periódico, se especifica el intervalo del período para la periodicidad, como días o meses.</span><span class="sxs-lookup"><span data-stu-id="83358-105">When you create the periodic journal, you specify the period interval for the recurrence, such as days or months.</span></span> <span data-ttu-id="83358-106">Esta guía de la tarea creará un diario periódico con una periodicidad mensual.</span><span class="sxs-lookup"><span data-stu-id="83358-106">This task guide will create a periodic journal with a monthly recurrence.</span></span>

1. <span data-ttu-id="83358-107">Vaya al **Panel de exploración > Módulos > Contabilidad general > Tareas periódicas > Diarios periódicos**.</span><span class="sxs-lookup"><span data-stu-id="83358-107">Go to **Navigation pane > Modules > General ledger > Periodic tasks > Periodic journals**.</span></span>
2. <span data-ttu-id="83358-108">Haga clic en **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="83358-108">Click **New**.</span></span>
3. <span data-ttu-id="83358-109">En el campo **Nombre**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-109">In the **Name** field, enter or select a value.</span></span>
4. <span data-ttu-id="83358-110">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="83358-110">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="83358-111">En el campo **Descripción**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-111">In the **Description** field, type a value.</span></span> <span data-ttu-id="83358-112">La descripción será el nombre del diario periódico cuando se recupere más adelante, por lo que debe asegurarse de darle un nombre relevante.</span><span class="sxs-lookup"><span data-stu-id="83358-112">The description will be the name of the Periodic journal when retrieved later so make sure to give it a relevant name.</span></span>
6. <span data-ttu-id="83358-113">En el **panel de acciones**, haga clic en **Líneas**.</span><span class="sxs-lookup"><span data-stu-id="83358-113">On the **Action pane**, click **Lines**.</span></span> <span data-ttu-id="83358-114">Un diario periódico incluirá normalmente varias líneas de diario.</span><span class="sxs-lookup"><span data-stu-id="83358-114">A periodic journal will typically include several journal lines.</span></span> <span data-ttu-id="83358-115">En esta guía de tarea, no obstante, se agregará una sola línea.</span><span class="sxs-lookup"><span data-stu-id="83358-115">this task guide will however only add one line.</span></span>
7. <span data-ttu-id="83358-116">En el campo **Fecha**, escriba una fecha.</span><span class="sxs-lookup"><span data-stu-id="83358-116">In the **Date** field, enter a date.</span></span> <span data-ttu-id="83358-117">El campo **Fecha** contiene la fecha de registro de la siguiente transferencia al diario.</span><span class="sxs-lookup"><span data-stu-id="83358-117">The **Date** field contains the posting date for the next transfer to the daily journal.</span></span> <span data-ttu-id="83358-118">Para los diarios que se recuperarán cada mes, se recomienda usar la fecha del mes en que se registraron, normalmente la primera o última fecha del período.</span><span class="sxs-lookup"><span data-stu-id="83358-118">For journals that will be retrieved each month it is recommended to use the date in the month when it will be posted, typically the first or last date in the period.</span></span> <span data-ttu-id="83358-119">Es posible dejar en blanco el campo de fecha y ofrecer una fecha cuando se recupera el diario, mediante el campo Fecha vacía.</span><span class="sxs-lookup"><span data-stu-id="83358-119">It is possible to leave the Date field blank and to give a date when the journal is retrieved, using the Empty date field.</span></span> <span data-ttu-id="83358-120">El campo se actualiza automáticamente a la siguiente fecha que se repite cuando se recuperan las transacciones.</span><span class="sxs-lookup"><span data-stu-id="83358-120">The field is automatically updated to the next recurring date when transactions are retrieved.</span></span> 
8. <span data-ttu-id="83358-121">En el campo **Cuenta**, especifique los valores deseados.</span><span class="sxs-lookup"><span data-stu-id="83358-121">In the **Account** field, specify the desired values.</span></span>
9. <span data-ttu-id="83358-122">En el campo **Descripción**, seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-122">In the **Description** field, select a value.</span></span>
10. <span data-ttu-id="83358-123">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="83358-123">Close the page.</span></span>
11. <span data-ttu-id="83358-124">En el campo **Débito**, escriba un número.</span><span class="sxs-lookup"><span data-stu-id="83358-124">In the **Debit** field, enter a number.</span></span>
12. <span data-ttu-id="83358-125">En el campo **Cuenta de contrapartida**, especifique los valores deseados.</span><span class="sxs-lookup"><span data-stu-id="83358-125">In the **Offset account** field, specify the desired values.</span></span>
13. <span data-ttu-id="83358-126">En el campo **Unidad**, seleccione "Meses".</span><span class="sxs-lookup"><span data-stu-id="83358-126">In the **Unit** field, select 'Months'.</span></span>
14. <span data-ttu-id="83358-127">En el campo **Número de unidades**, escriba "1".</span><span class="sxs-lookup"><span data-stu-id="83358-127">In the **Number of units** field, enter '1'.</span></span>
15. <span data-ttu-id="83358-128">En el campo **Última fecha**, escriba una fecha.</span><span class="sxs-lookup"><span data-stu-id="83358-128">In the **Last date** field, enter a date.</span></span> <span data-ttu-id="83358-129">Especificar la última fecha del período anterior prevendrá que el diario periódico se cree por error en el período inicial incorrecto.</span><span class="sxs-lookup"><span data-stu-id="83358-129">Entering last date in the preceding period will prevent that the recurring journal by mistake is created in the wrong starting period.</span></span> <span data-ttu-id="83358-130">La última fecha se actualizará después cada vez que se recupere el diario periódico.</span><span class="sxs-lookup"><span data-stu-id="83358-130">Last date will later on be updated each time the periodic journal is retrieved.</span></span> 
16. <span data-ttu-id="83358-131">Haga clic en **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="83358-131">Click **Save**.</span></span>
17. <span data-ttu-id="83358-132">Vaya a **Panel de exploración > Módulos > Contabilidad general > Entradas del diario > Diarios generales**.</span><span class="sxs-lookup"><span data-stu-id="83358-132">Go to **Navigation pane > Modules > General ledger > Journal entries > General journals**.</span></span>
18. <span data-ttu-id="83358-133">Haga clic en **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="83358-133">Click **New**.</span></span>
19. <span data-ttu-id="83358-134">En el campo **Nombre**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-134">In the **Name** field, enter or select a value.</span></span>
20. <span data-ttu-id="83358-135">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="83358-135">In the list, find and select the desired record.</span></span>
21. <span data-ttu-id="83358-136">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="83358-136">In the list, click the link in the selected row.</span></span>
22. <span data-ttu-id="83358-137">En el campo **Descripción**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-137">In the **Description** field, type a value.</span></span>
23. <span data-ttu-id="83358-138">En el **panel de acciones**, haga clic en **Líneas**.</span><span class="sxs-lookup"><span data-stu-id="83358-138">On the **Action pane**, click **Lines**.</span></span>
24. <span data-ttu-id="83358-139">En el **Panel de acciones**, haga clic en **Diario periódico**.</span><span class="sxs-lookup"><span data-stu-id="83358-139">On the **Action pane**, click **Period journal**.</span></span>
25. <span data-ttu-id="83358-140">Haga clic en **Recuperar diario**.</span><span class="sxs-lookup"><span data-stu-id="83358-140">Click **Retrieve journal**.</span></span>
26. <span data-ttu-id="83358-141">En el campo **Fecha final**, especifique una fecha.</span><span class="sxs-lookup"><span data-stu-id="83358-141">In the **To date** field, enter a date.</span></span> <span data-ttu-id="83358-142">La **fecha final** sirve como fecha de corte para las líneas periódicas del asiento.</span><span class="sxs-lookup"><span data-stu-id="83358-142">The **To date** serves as the cutoff date for the periodic voucher lines.</span></span> <span data-ttu-id="83358-143">De acuerdo con la configuración de periodicidad, la última fecha y la fecha final para la misma línea se pueden incluir más de una vez en el diario resultante.</span><span class="sxs-lookup"><span data-stu-id="83358-143">Based on the recurrence setting, the Last date and To date the same line might be included more than once in the resulting journal.</span></span> <span data-ttu-id="83358-144">La fecha final se actualiza automáticamente según la fecha de sesión de la última vez que la línea periódica se transfirió a un diario.</span><span class="sxs-lookup"><span data-stu-id="83358-144">To date is automatically updated based on  session date of the last time the periodic line was transferred to a journal.</span></span> 
27. <span data-ttu-id="83358-145">En el campo **Número de diario periódico**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="83358-145">In the **Periodic journal number** field, enter or select a value.</span></span>
28. <span data-ttu-id="83358-146">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="83358-146">In the list, click the link in the selected row.</span></span>
29. <span data-ttu-id="83358-147">Haga clic en **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="83358-147">Click **OK**.</span></span> <span data-ttu-id="83358-148">El diario del período se puede revisar, aprobar o registrar ahora en función del requisito y la configuración.</span><span class="sxs-lookup"><span data-stu-id="83358-148">The period journal can now be reviewed, approved or posted depending on requirement and setup.</span></span>
