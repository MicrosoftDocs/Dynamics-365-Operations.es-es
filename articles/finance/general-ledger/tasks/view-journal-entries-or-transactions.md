# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="8a358-103">Ver movimientos del diario o transacciones</span><span class="sxs-lookup"><span data-stu-id="8a358-103">View journal entries or transactions</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8a358-104">Este procedimiento muestra cómo usar la consulta de transacciones de asiento para buscar los movimientos del diario o las transacciones.</span><span class="sxs-lookup"><span data-stu-id="8a358-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="8a358-105">Vaya a **Panel de exploración > Módulos > Contabilidad general > Consultas e informes > Transacciones de asiento**.</span><span class="sxs-lookup"><span data-stu-id="8a358-105">Go to **Navigation pane > Modules > General ledger > Inquiries and reports > Voucher transactions**.</span></span>
2. <span data-ttu-id="8a358-106">Seleccione el campo para el que desea definir un criterio de filtro.</span><span class="sxs-lookup"><span data-stu-id="8a358-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="8a358-107">Introduzca sus criterios de filtro para el campo seleccionado.</span><span class="sxs-lookup"><span data-stu-id="8a358-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="8a358-108">Podría filtrar en un único valor o en un intervalo.</span><span class="sxs-lookup"><span data-stu-id="8a358-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="8a358-109">Al definir un intervalo, asegúrese de que se use la sintaxis correcta.</span><span class="sxs-lookup"><span data-stu-id="8a358-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="8a358-110">Los valores se deben separar con doble punto (..).</span><span class="sxs-lookup"><span data-stu-id="8a358-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="8a358-111">Haga clic en la ficha **Combinaciones** para agregar tablas adicionales desde las que filtrar.</span><span class="sxs-lookup"><span data-stu-id="8a358-111">Click the **Joins** tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="8a358-112">En el árbol, seleccione **Tablas/Entrada de diario general**.</span><span class="sxs-lookup"><span data-stu-id="8a358-112">In the tree, select **Tables/General journal entry**.</span></span>
6. <span data-ttu-id="8a358-113">Haga clic en **Agregar combinación de tabla**.</span><span class="sxs-lookup"><span data-stu-id="8a358-113">Click **Add table join**.</span></span>
7. <span data-ttu-id="8a358-114">Haga clic en **Cancelar** si decide no agregar una tabla adicional.</span><span class="sxs-lookup"><span data-stu-id="8a358-114">Click **Cancel** if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="8a358-115">Haga clic en la ficha **Intervalo**.</span><span class="sxs-lookup"><span data-stu-id="8a358-115">Click the **Range** tab.</span></span>
9. <span data-ttu-id="8a358-116">Haga clic en **Aceptar** para ejecutar la consulta.</span><span class="sxs-lookup"><span data-stu-id="8a358-116">Click **OK** to run the query.</span></span>
10. <span data-ttu-id="8a358-117">En el panel de acciones, haga clic en **Origen de transacción**.</span><span class="sxs-lookup"><span data-stu-id="8a358-117">On the Action pane, click **Transaction origin**.</span></span> <span data-ttu-id="8a358-118">Los distintos botones sobre la cuadrícula se pueden usar para buscar información adicional sobre el registro seleccionado del asiento.</span><span class="sxs-lookup"><span data-stu-id="8a358-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="8a358-119">Algunos botones pueden no estar disponibles, según el tipo de transacción y las características de la transacción.</span><span class="sxs-lookup"><span data-stu-id="8a358-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
11. <span data-ttu-id="8a358-120">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="8a358-120">Close the page.</span></span>
12. <span data-ttu-id="8a358-121">En el panel de acciones, haga clic en **Documento original**.</span><span class="sxs-lookup"><span data-stu-id="8a358-121">On the Action pane, Click **Original document**.</span></span>
13. <span data-ttu-id="8a358-122">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="8a358-122">Close the page.</span></span>
