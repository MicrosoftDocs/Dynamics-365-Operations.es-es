[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="98d67-104">Esta guía de tareas describe los pasos de la adición de una cuenta principal a un plan contable existente.</span><span class="sxs-lookup"><span data-stu-id="98d67-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="98d67-105">Esta grabación usa la empresa de demostración USMF.</span><span class="sxs-lookup"><span data-stu-id="98d67-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="98d67-106">Vaya a **Panel de exploración > Módulos > Contabilidad general > Plan contable > Cuentas > Cuentas principales**.</span><span class="sxs-lookup"><span data-stu-id="98d67-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="98d67-107">Haga clic en **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="98d67-107">Click **New**.</span></span>
3. <span data-ttu-id="98d67-108">En el campo **Cuenta principal**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="98d67-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="98d67-109">En el campo **Nombre**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="98d67-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="98d67-110">En el campo **Tipo de cuenta principal**, seleccione el tipo que representa mejor la ubicación y el saldo de cuentas de los resultados financieros.</span><span class="sxs-lookup"><span data-stu-id="98d67-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="98d67-111">En la lista, seleccione la categoría de cuenta a la que pertenece la cuenta principal.</span><span class="sxs-lookup"><span data-stu-id="98d67-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="98d67-112">La categoría de cuenta se usa para los informes financieros predeterminados y el contenido del panel de información de Power BI.</span><span class="sxs-lookup"><span data-stu-id="98d67-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="98d67-113">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="98d67-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="98d67-114">Cambie el saldo de débito o crédito predeterminado.</span><span class="sxs-lookup"><span data-stu-id="98d67-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="98d67-115">En el campo **Divisa predeterminada**, seleccione un valor en la lista de divisas.</span><span class="sxs-lookup"><span data-stu-id="98d67-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="98d67-116">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="98d67-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="98d67-117">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="98d67-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="98d67-118">Expanda la sección **Anulaciones de entidad jurídica**.</span><span class="sxs-lookup"><span data-stu-id="98d67-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="98d67-119">Haga clic en **Agregar** para seleccionar una entidad jurídica.</span><span class="sxs-lookup"><span data-stu-id="98d67-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="98d67-120">En la lista, seleccione la Entidad jurídica.</span><span class="sxs-lookup"><span data-stu-id="98d67-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="98d67-121">Haga clic en **Agregar**.</span><span class="sxs-lookup"><span data-stu-id="98d67-121">Click **Add**.</span></span>
15. <span data-ttu-id="98d67-122">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="98d67-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="98d67-123">Active o desactive la casilla **Suspendido**.</span><span class="sxs-lookup"><span data-stu-id="98d67-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="98d67-124">Expanda la sección **Informes financieros**.</span><span class="sxs-lookup"><span data-stu-id="98d67-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="98d67-125">En el campo **Tipo de cambio**, haga clic en el botón desplegable para abrir la búsqueda.</span><span class="sxs-lookup"><span data-stu-id="98d67-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="98d67-126">En la lista, seleccione el **Tipo de cambio para la cuenta**.</span><span class="sxs-lookup"><span data-stu-id="98d67-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="98d67-127">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="98d67-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="98d67-128">En el campo **Tipo de traducción de divisa**, seleccione el método para calcular tipos de cambio para la cuenta.</span><span class="sxs-lookup"><span data-stu-id="98d67-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="98d67-129">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="98d67-129">Close the page.</span></span>
