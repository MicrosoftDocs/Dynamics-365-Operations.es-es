# <a name="view-an-expense-report"></a><span data-ttu-id="b8003-103">Visualización de un informe de gastos</span><span class="sxs-lookup"><span data-stu-id="b8003-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="b8003-104">Al ejecutar una consulta para ver transacciones en Contabilidad general o Adquisición y abastecimiento, la consulta podría devolver demasiada información.</span><span class="sxs-lookup"><span data-stu-id="b8003-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="b8003-105">Por lo tanto, es posible que desee limitar la búsqueda.</span><span class="sxs-lookup"><span data-stu-id="b8003-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="b8003-106">Para obtener la información que requiere, puede explorar en profundidad una transacción para ver el informe de gastos en el que apareció.</span><span class="sxs-lookup"><span data-stu-id="b8003-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="b8003-107">Para los informes de gastos, puede explorar en profundidad desde una transacción al informe original que se encuentra almacenado en gestión de gastos.</span><span class="sxs-lookup"><span data-stu-id="b8003-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="b8003-108">Además del importe total de gastos, puede ver las transacciones detalladas, las transacciones aptas para la devolución del impuesto sobre el valor añadido (IVA) y los recibos vinculados a los gastos.</span><span class="sxs-lookup"><span data-stu-id="b8003-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="b8003-109">Para desglosar el informe de gastos original que apareció en una transacción, seleccione la transacción y, a continuación, seleccione **Documento original**.</span><span class="sxs-lookup"><span data-stu-id="b8003-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="b8003-110">Al ver las transacciones individuales, también puede ver cualquier dato adicional vinculado a la transacción.</span><span class="sxs-lookup"><span data-stu-id="b8003-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="b8003-111">Por ejemplo, para un gasto de hotel detallado, puede ver una lista detallada de gastos como el acceso a Internet, las llamadas telefónicas y el servicio de habitación.</span><span class="sxs-lookup"><span data-stu-id="b8003-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
