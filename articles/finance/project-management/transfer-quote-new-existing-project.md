# <a name="transfer-a-quotation-to-a-project"></a><span data-ttu-id="f5fa3-103">Transferir un presupuesto a un proyecto</span><span class="sxs-lookup"><span data-stu-id="f5fa3-103">Transfer a quotation to a project</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="f5fa3-104">Cuando un presupuesto de proyecto tiene un estado de **Confirmado**, puede transferir algunas o todas las líneas de presupuesto a un proyecto nuevo o existente utilizando el asistente **Transferencia a proyecto**.</span><span class="sxs-lookup"><span data-stu-id="f5fa3-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="f5fa3-105">Siga estos pasos para usar el asistente.</span><span class="sxs-lookup"><span data-stu-id="f5fa3-105">To use the wizard, follow these steps.</span></span>

1. <span data-ttu-id="f5fa3-106">Vaya a **Gestión de proyectos y contabilidad** > **Presupuestos** > **Presupuestos de proyecto**.</span><span class="sxs-lookup"><span data-stu-id="f5fa3-106">Go to **Project management and accounting** > **Quotations** > **Project quotations**.</span></span>
2. <span data-ttu-id="f5fa3-107">Seleccione el presupuesto que desea transferir y en la pestaña **Seguimiento**, en el grupo **Modificar**, seleccione **Transferencia al proyecto**.</span><span class="sxs-lookup"><span data-stu-id="f5fa3-107">Select the quotation you want to transfer and on the **Follow up** tab, in the **Modify** group, select **Transfer to project**.</span></span>
3. <span data-ttu-id="f5fa3-108">Siga el asistente **Transferir presupuesto a proyecto** para asociar y transferir el presupuesto a un proyecto.</span><span class="sxs-lookup"><span data-stu-id="f5fa3-108">Follow the **Transfer quotation to project** wizard to associate and transfer the quotation with a project.</span></span>
