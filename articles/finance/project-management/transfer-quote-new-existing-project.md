# <a name="transfer-a-quotation-to-a-project"></a>Transferir un presupuesto a un proyecto

[!include [banner](../includes/banner.md)]

Cuando un presupuesto de proyecto tiene un estado de **Confirmado**, puede transferir algunas o todas las líneas de presupuesto a un proyecto nuevo o existente utilizando el asistente **Transferencia a proyecto**. 

Siga estos pasos para usar el asistente.

1. Vaya a **Gestión de proyectos y contabilidad** > **Presupuestos** > **Presupuestos de proyecto**.
2. Seleccione el presupuesto que desea transferir y en la pestaña **Seguimiento**, en el grupo **Modificar**, seleccione **Transferencia al proyecto**.
3. Siga el asistente **Transferir presupuesto a proyecto** para asociar y transferir el presupuesto a un proyecto.
