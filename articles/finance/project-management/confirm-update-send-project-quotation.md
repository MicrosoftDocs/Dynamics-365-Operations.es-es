[!include [banner](../includes/banner.md)]

<span data-ttu-id="b8622-104">Después de crear y enviar un presupuesto de proyecto a un cliente, debe obtener la confirmación del cliente antes de actualizar el estado del presupuesto a **Enviado**.</span><span class="sxs-lookup"><span data-stu-id="b8622-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="b8622-105">Cualquier modificación solicitada por el cliente puede actualizarse en el presupuesto.</span><span class="sxs-lookup"><span data-stu-id="b8622-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="b8622-106">Después de actualizar el estado del presupuesto **Enviado**, no puede modificarlo.</span><span class="sxs-lookup"><span data-stu-id="b8622-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="b8622-107">El siguiente procedimiento describe las opciones para enviar un presupuesto para confirmación, realizar actualizaciones basadas en comentarios y luego enviar el presupuesto.</span><span class="sxs-lookup"><span data-stu-id="b8622-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

<span data-ttu-id="b8622-109">Puede enviar un presupuesto de proyecto existente para confirmación por correo electrónico o como una copia impresa.</span><span class="sxs-lookup"><span data-stu-id="b8622-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span>

6. <span data-ttu-id="b8622-116">En la página **Configuración de destino de impresión** , seleccione **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="b8622-116">On the **Print destination settings** page, select **OK**.</span></span>
