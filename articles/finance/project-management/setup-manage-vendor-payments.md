# <a name="set-up-and-use-pay-when-paid-vendor-payments"></a><span data-ttu-id="5c26c-103">Configurar y usar pagos de proveedor de pago al cobro</span><span class="sxs-lookup"><span data-stu-id="5c26c-103">Set up and use pay-when-paid vendor payments</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="5c26c-104">Al aprobar a un proveedor para trabajar como subcontratista en un proyecto, puede ser conveniente retener el pago al proveedor hasta que el cliente haya pagado el proyecto.</span><span class="sxs-lookup"><span data-stu-id="5c26c-104">When you approve a vendor to work as a subcontractor, you might want to withhold payment to the vendor until your customer pays you for the project.</span></span> <span data-ttu-id="5c26c-105">Para admitir este escenario, puede configurar las condiciones de pago al cobro (AaC) cuando configure el pedido de compra (PO) con el proveedor.</span><span class="sxs-lookup"><span data-stu-id="5c26c-105">To support this scenario, you can set up pay-when-paid (PWP) terms when you set up the purchase order (PO) with the vendor.</span></span>

<span data-ttu-id="5c26c-106">Por ejemplo, cuando un cliente paga un importe de una factura de proyecto, se puede liberar una parte o todo el importe de la factura del proveedor.</span><span class="sxs-lookup"><span data-stu-id="5c26c-106">For example, when a customer pays an amount on a project invoice, you can release some or all of the vendor invoice amount.</span></span> <span data-ttu-id="5c26c-107">Basta con configurar condiciones de pago al cobro que especifiquen que se pagará a un proveedor después de recibir un porcentaje específico del pago relacionado del cliente.</span><span class="sxs-lookup"><span data-stu-id="5c26c-107">Just set up PWP terms that specify that the vendor will be paid after you receive a percentage of the related payment from the customer.</span></span> <span data-ttu-id="5c26c-108">Si recibe un pago parcial de un cliente, puede liberar manualmente algunas de las facturas de proveedor relacionadas para el pago.</span><span class="sxs-lookup"><span data-stu-id="5c26c-108">If you receive partial payment from a customer, you can manually release some of the related vendor invoices for payment.</span></span>

<span data-ttu-id="5c26c-109">El siguiente ejemplo describe el proceso cuando se utilizan los términos PWP.</span><span class="sxs-lookup"><span data-stu-id="5c26c-109">The following example outlines the process when PWP terms are used.</span></span>

## <a name="example"></a><span data-ttu-id="5c26c-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5c26c-110">Example</span></span>

<span data-ttu-id="5c26c-111">Su organización acuerda proporcionar a un cliente 100 computadoras que tengan un software instalado, por un precio de 150,00 dólares estadounidenses (USD) por computadora.</span><span class="sxs-lookup"><span data-stu-id="5c26c-111">Your organization agrees to provide a customer with 100 computers that have software installed, for a price of 150.00 US dollars (USD) per computer.</span></span> <span data-ttu-id="5c26c-112">Luego contrata a un proveedor para que le proporcione las computadoras que tienen el software instalado.</span><span class="sxs-lookup"><span data-stu-id="5c26c-112">You then hire a vendor to provide you with the computers that have software installed.</span></span> <span data-ttu-id="5c26c-113">Según el acuerdo, el cliente desea aprobar la calidad de los equipos antes de pagar a su organización.</span><span class="sxs-lookup"><span data-stu-id="5c26c-113">According to the agreement, the customer must approve the quality of the computers before it pays your organization.</span></span> <span data-ttu-id="5c26c-114">La política de su organización es retener el pago a los proveedores hasta que el cliente le haya pagado.</span><span class="sxs-lookup"><span data-stu-id="5c26c-114">Your organization's policy is to withhold payment to vendors until the customer has paid you.</span></span> <span data-ttu-id="5c26c-115">Por lo tanto, configura el proyecto para que tenga un porcentaje de PWP del 100 por cien.</span><span class="sxs-lookup"><span data-stu-id="5c26c-115">Therefore, you set up the project so that it has a PWP percentage of 100 percent.</span></span>

<span data-ttu-id="5c26c-116">El proveedor le envía las 100 computadoras que tienen el software instalado, junto con una factura por 10 000,00 USD.</span><span class="sxs-lookup"><span data-stu-id="5c26c-116">The vendor sends you the 100 computers that have software installed, together with an invoice for 10,000.00 USD.</span></span> <span data-ttu-id="5c26c-117">Debido a que tiene términos de PWP configurados para su proyecto, no le paga al proveedor al recibir las computadoras.</span><span class="sxs-lookup"><span data-stu-id="5c26c-117">Because PWP terms are set up for your project, you don't pay the vendor upon receipt of the computers.</span></span> <span data-ttu-id="5c26c-118">En su lugar, envía las computadoras al cliente, junto con una factura por 15 000,00.</span><span class="sxs-lookup"><span data-stu-id="5c26c-118">Instead, you send the computers to the customer, together with an invoice for 15,000.00.</span></span> <span data-ttu-id="5c26c-119">El cliente examina los equipos y aprueba el importe total de la factura del proyecto.</span><span class="sxs-lookup"><span data-stu-id="5c26c-119">The customer inspects the computers and approves the full amount of the project invoice.</span></span>

<span data-ttu-id="5c26c-120">Una vez recibido el pago completo del cliente, paga 10 000,00 al proveedor por el importe total de la factura de proveedor.</span><span class="sxs-lookup"><span data-stu-id="5c26c-120">After you receive the full payment from the customer, you pay the vendor 10,000.00, the full amount of the vendor invoice.</span></span>

<span data-ttu-id="5c26c-122">Cuando configura los términos de PWP para un proyecto, debe especificar, como porcentaje, la cantidad mínima que un cliente debe pagarle por el proyecto antes de pagarle al proveedor.</span><span class="sxs-lookup"><span data-stu-id="5c26c-122">When you set up PWP terms for a project, you must specify, as a percentage, the minimum amount that a customer must pay you for the project before you will pay the vendor.</span></span> <span data-ttu-id="5c26c-123">El importe de umbral se calcula automáticamente en las facturas de cliente del proyecto.</span><span class="sxs-lookup"><span data-stu-id="5c26c-123">The threshold amount is automatically calculated on the customer invoices for the project.</span></span> <span data-ttu-id="5c26c-124">Siga estos pasos para configurar el porcentaje del umbral para los términos PWP.</span><span class="sxs-lookup"><span data-stu-id="5c26c-124">Follow these steps to set up the threshold percentage for PWP terms.</span></span>

3. <span data-ttu-id="5c26c-128">En el campo **Código de cuenta**, seleccione una de las opciones siguientes:</span><span class="sxs-lookup"><span data-stu-id="5c26c-128">In the **Account code** field, select one of the following options:</span></span>

    - <span data-ttu-id="5c26c-129">**Tabla**: las condiciones PWP se aplican a un único proveedor.</span><span class="sxs-lookup"><span data-stu-id="5c26c-129">**Table** – The PWP terms apply to a single vendor.</span></span>
    - <span data-ttu-id="5c26c-130">**Grupo**: las condiciones PWP se aplican a todos los proveedores en un grupo de proveedores.</span><span class="sxs-lookup"><span data-stu-id="5c26c-130">**Group** – The PWP terms apply to all vendors in a vendor group.</span></span>
