# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="a6aaf-103">Crear una nota de abono en facturas de proyecto</span><span class="sxs-lookup"><span data-stu-id="a6aaf-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="a6aaf-104">Vaya a Gestión de proyectos y contabilidad > Proyectos > Todos los proyectos.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="a6aaf-105">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="a6aaf-106">Seleccione el identificador del proyecto que se usa para la nota de abono.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="a6aaf-107">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="a6aaf-108">En el panel de acciones, haga clic en Gestionar.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="a6aaf-109">Haga clic en Diarios de facturas.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="a6aaf-110">Seleccione la transacción para la nota de abono.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-110">Select the transaction for credit note.</span></span> <span data-ttu-id="a6aaf-111">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="a6aaf-112">Haga clic en Funciones.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-112">Click Functions.</span></span> 
8. <span data-ttu-id="a6aaf-113">Haga clic en Seleccionar para nota de abono.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="a6aaf-114">Seleccione la transacción para la nota de abono activando la casilla de verificación.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="a6aaf-115">Haga clic en Aceptar</span><span class="sxs-lookup"><span data-stu-id="a6aaf-115">Click OK.</span></span> 

<span data-ttu-id="a6aaf-116">Después de este paso, la nota de abono estará disponible para propuesta de factura.</span><span class="sxs-lookup"><span data-stu-id="a6aaf-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
