# <a name="enhanced-filtering-options-for-finding-configurations-in-the-global-repository"></a>Opciones de filtrado mejoradas para encontrar configuraciones en el repositorio global

[!include [banner](../includes/banner.md)]

Este tema describe capacidades de filtrado mejoradas para el repositorio global Regulatory Configuration Service (RCS), que se han mejorado para incluir los siguientes filtros: 
- **País / región** - basado en los códigos de país ISO  
- **Etiquetas** - para el área funcional / característica; Industria; Tipo de documento comercial 

Puede aplicar filtros, individualmente o en grupos, para encontrar configuraciones específicas o relacionadas. Por ejemplo, para buscar todos los documentos comerciales configurables relacionados con las facturas de proveedores, puede aplicar el filtro **Tipo de documento comercial**. 

Puede refinar aún más una búsqueda seleccionando el código del país y haciendo clic en **Aplicar filtro**.  

[![Sección de filtro para el repositorio global](media/rcs-enhanced-filter-section.JPG)](./media/rcs-enhanced-filter-section.JPG) 

El siguiente ejemplo muestra los resultados al filtrar en **Tipo de documento comercial**. 

[![Filtro aplicado e importación para tipo de documento empresarial](media/rcs-enhanced-filtering-applied.JPG)](./media/rcs-enhanced-filtering-applied.JPG) 

Los resultados filtrados se pueden importar a los usuarios RCS o entorno Dynamics 365 Finance, ya sea individualmente o como un conjunto (seleccionando el grupo de configuraciones) y haciendo clic en **Importar**.
