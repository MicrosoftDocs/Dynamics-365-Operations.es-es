# <a name="view-posted-tds-payments-and-transactions-for-a-tds-settlement-period"></a>Ver pagos de TDS registrados y transacciones por un período de liquidación de TDS

[!include [banner](../includes/banner.md)]

Este artículo explica cómo ver los pagos y transacciones con impuestos deducidos en el origen (TDS) que se registraron para un período de liquidación.

1. Vaya a **Impuestos \> Impuestos indirectos \> Retención de impuestos \> Períodos de liquidación de retención de impuestos**.

    [![Página Períodos de liquidación de retención de impuestos.](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)

2. En la página **Períodos de liquidación de retención de impuestos**, seleccione **Pagos de retención de impuestos** para abrir la página **Pagos de retención de impuestos**, donde puede ver las liquidaciones de TDS que se realizaron para un período de liquidación de TDS específico.

    La pestaña **Información general** muestra la informacion siguiente:

    - **Fecha**: la fecha de liquidación del TDS.
    - **Asiento**: el número de asiento de la transacción de liquidación de TDS.
    - **Tipo de impuesto**: el tipo de impuesto para el que se ejecuta el proceso de liquidación.
    - **Número de cuenta de impuestos (TAN)**: el número de cuenta de impuestos (TAN) de la transacción de TDS liquidada.
    - **Período de liquidación**: el período de liquidación para el que se ejecuta el proceso de liquidación de TDS.
    - **Fecha inicial**: la fecha de inicio del período de liquidación.
    - **Fecha final**: la fecha final del período de liquidación.
    - **Versión del pago de retención de impuestos**: la versión del pago de la retención de impuestos: **Original**, **Últimas correcciones** o **Lista total**.

5. Seleccione **Asiento** para ver las entradas de asiento para la transacción TDS.
6. Seleccione **Transacciones de retención de impuestos** para ver los detalles de las transacciones de TDS que se liquidaron para un período específico durante un período de liquidación. Seleccione **Asiento** para ver las entradas de asiento para la transacción TDS. Seleccione **Componentes de retención de impuestos** para ver los TDS que se calcularon por componente de impuestos de TDS para un código de impuestos de TDS específico.
7. Cierre la página **Pagos de retención de impuestos** para volver a la página **Períodos de liquidación de retención de impuestos**.
8. Seleccione **Transacciones de retención de impuestos** para ver los detalles de las transacciones de TDS que se liquidaron para el período de liquidación.
