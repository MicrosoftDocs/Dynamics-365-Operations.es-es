# <a name="set-up-tds-ledger-accounts"></a>Configurar cuentas contables de TDS

[!include [banner](../includes/banner.md)]

Este tema explica cómo configurar cuentas contables para la características de impuestos deducidos en el origen (TDS). Use la página **Plan de cuentas** para configurar cuentas contables para TDS.

1. Vaya a **Contabilidad general \> Plan de cuentas \> Cuentas \> Cuentas principales**.
2. En la pestaña **Información general**, seleccione **Alt + N** para crear una cuenta contable de TDS e introduzca los detalles necesarios.
3. En la pestaña **Configuración**, en el campo **Tipo de registro**, seleccione **Retención de impuestos**.     

    > [!NOTE]
    > Las cuentas contables que tienen un tipo de registro de **Retención de impuestos** solo se puede definir como cuentas de clientes que se utilizan para registrar el importe de la cuenta de cliente de TDS para un código de impuestos de TDS.

4. Cierre la página.
