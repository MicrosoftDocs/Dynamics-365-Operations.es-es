# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="2d9f8-103">Configurar grupos de componentes de retención de impuestos para el tipo de impuestos de TDS</span><span class="sxs-lookup"><span data-stu-id="2d9f8-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="2d9f8-104">Este tema explica cómo configurar grupos de componentes de retención de impuestos, como **Alquiler** y **Contratista**, para el tipo de impuesto de impuesto deducido en el origen (TDS).</span><span class="sxs-lookup"><span data-stu-id="2d9f8-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="2d9f8-105">Vaya a **Impuesto \> Configuración \> Retención de impuestos \> Grupos de componentes de retención de impuestos**.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="2d9f8-106">[![Página de grupos de componentes de retenciones de impuestos](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="2d9f8-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="2d9f8-107">En el campo **Tipo de impuesto**, seleccione **TDS** para configurar grupos de componentes de retención de impuestos para el tipo de impuesto TDS.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="2d9f8-108">En el panel de acciones, haga clic en **Nuevo** para crear una línea.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="2d9f8-109">En el campo **Grupo de componentes de retención de impuestos**, introduzca un nombre para el grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="2d9f8-110">En el campo **Descripción**, escriba una descripción.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="2d9f8-111">En la ficha desplegable **General**, en el campo **Estado**, seleccione el estado residencial del grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="2d9f8-112">Las opciones son **Residente** y **No residente**.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="2d9f8-113">En el campo **Código de sección**, introduzca el código de sección que se aplica al grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="2d9f8-114">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="2d9f8-114">Close the page.</span></span>
