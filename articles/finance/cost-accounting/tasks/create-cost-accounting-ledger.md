[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c52df-104">Un libro mayor de contabilidad de costes representa la unidad de notificación global.</span><span class="sxs-lookup"><span data-stu-id="c52df-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="c52df-105">Se define por una dimensión de elemento de coste, la dimensión estadística, el calendario fiscal y la divisa.</span><span class="sxs-lookup"><span data-stu-id="c52df-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="c52df-106">Es agnóstico al concepto de entidades jurídicas.</span><span class="sxs-lookup"><span data-stu-id="c52df-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="c52df-107">Una entidad jurídica y sus datos se pueden asociar a varios libros mayores de contabilidad de costes.</span><span class="sxs-lookup"><span data-stu-id="c52df-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="c52df-108">Este registro usa la empresa USP2 con los datos para demostración.</span><span class="sxs-lookup"><span data-stu-id="c52df-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="c52df-109">Vaya a Contabilidad de costes > Configuración de libro mayor > Libros mayores de contabilidad de costes.</span><span class="sxs-lookup"><span data-stu-id="c52df-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="c52df-110">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="c52df-110">Click New.</span></span>
3. <span data-ttu-id="c52df-111">En el campo Nombre, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="c52df-112">En el campo Dimensión de elemento de coste, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="c52df-113">En el campo Calendario fiscal, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="c52df-114">En el campo Divisa de contabilidad, escriba o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="c52df-115">En el campo Tipo de tipo de cambio, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="c52df-116">En el campo Dimensión estadística, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="c52df-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="c52df-117">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="c52df-117">Click Save.</span></span>
