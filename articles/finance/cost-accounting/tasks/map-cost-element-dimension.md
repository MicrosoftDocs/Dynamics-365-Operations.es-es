# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="10365-103">Asignar una dimensión de elemento de coste</span><span class="sxs-lookup"><span data-stu-id="10365-103">Map a cost element dimension</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="10365-104">Un controlador de costes puede utilizar este procedimiento para asignar una dimensión de elemento de coste a una dimensión de elemento de coste en la entidad jurídica de MXMF.</span><span class="sxs-lookup"><span data-stu-id="10365-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="10365-105">Este registro usa la empresa USP2 con los datos para demostración.</span><span class="sxs-lookup"><span data-stu-id="10365-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="10365-106">Vaya a contabilidad de costes > Dimensiones > dimensiones del elemento de coste.</span><span class="sxs-lookup"><span data-stu-id="10365-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="10365-107">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="10365-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="10365-108">Para este ejemplo, seleccione Elemento de coste.</span><span class="sxs-lookup"><span data-stu-id="10365-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="10365-109">Haga clic en Asignaciones de dimensiones.</span><span class="sxs-lookup"><span data-stu-id="10365-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="10365-110">Haga clic en Configurar las asignaciones desde esta dimensión.</span><span class="sxs-lookup"><span data-stu-id="10365-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="10365-111">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="10365-111">Click New.</span></span>
6. <span data-ttu-id="10365-112">En el campo A dimensión, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="10365-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="10365-113">Para este ejemplo, seleccione Elementos de coste MXMF.</span><span class="sxs-lookup"><span data-stu-id="10365-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="10365-114">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="10365-114">Click New.</span></span>
8. <span data-ttu-id="10365-115">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="10365-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="10365-116">En el campo Desde miembro de dimensión, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="10365-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="10365-117">Para este ejemplo, seleccione Gasto de teléfono & fax del miembro de la dimensión 606400.</span><span class="sxs-lookup"><span data-stu-id="10365-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="10365-118">En el campo Hasta miembro de dimensión, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="10365-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="10365-119">Para este ejemplo, seleccione el Teléfono 6001004 del miembro de la dimensión.</span><span class="sxs-lookup"><span data-stu-id="10365-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="10365-120">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="10365-120">Click Save.</span></span>
