<span data-ttu-id="82b92-104">Si necesita realizar cambios en las guías de incorporación en Microsoft Dynamics 365 for Talent: Onboard, puede actualizarlas y insertar los cambios, aunque haya registrado ya las guías.</span><span class="sxs-lookup"><span data-stu-id="82b92-104">If you must make changes to onboarding guides in Microsoft Dynamics 365 for Talent: Onboard, you can update them and push the changes, even if you've already sent the guides.</span></span> <span data-ttu-id="82b92-105">Tiene dos opciones para actualizar una guía de incorporación:</span><span class="sxs-lookup"><span data-stu-id="82b92-105">You have two options for updating an onboarding guide:</span></span>

    ![[<span data-ttu-id="82b92-112">Guardar cambios en una guía de incorporación</span><span class="sxs-lookup"><span data-stu-id="82b92-112">Saving changes to an onboarding guide</span></span>](./media/onboard-save.png)](./media/onboard-save.png)
