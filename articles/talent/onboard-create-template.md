# <a name="create-an-onboarding-template"></a>Crear una plantilla de incorporación

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard proporciona varias plantillas que le ayudan a crear una guía de incorporación lo más rápidamente posible. Puede usar una o más de estas plantillas, o bien crear sus propias plantillas. Onboard proporciona el texto de muestra que puede usar al crear sus propias plantillas. Por lo tanto, el proceso es fácil incluso si empieza a partir de cero.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Crear una plantilla de incorporación a partir de una plantilla existente

1. En el menú izquierdo, seleccione **Plantillas**.
2. En **Plantillas populares de la galería** o **Mis plantillas**, seleccione una plantilla. Para ver más plantillas, seleccione **Más en la galería de plantillas**.
3. Siga uno de estos pasos:

    - Si la plantilla es de la galería, seleccione **Guardar como mi plantilla**, especifique un nuevo nombre para la plantilla, y seleccione **Guardar**.
    - Si la plantilla es de **Mis plantillas**, seleccione **Guardar como plantilla**, especifique un nuevo nombre para la plantilla, y seleccione **Guardar**.

    [![Creación de una plantilla a partir de una plantilla existente](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Crear una plantilla de incorporación nueva

1. En el menú izquierdo, seleccione **Plantillas**.
2. En **Mis plantillas**, seleccione el icono **Agregar** (signo más \[**+**\]).

    [![Crear una nueva plantilla](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. En el cuadro de diálogo **Crear una plantilla de guía**, escriba un nombre para la plantilla y, a continuación, seleccione **Guardar**.
