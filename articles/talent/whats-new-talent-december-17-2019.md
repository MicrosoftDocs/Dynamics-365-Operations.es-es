# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Novedades y cambios en Dynamics 365 Talent (17 de diciembre de 2019)

Este artículo describe las características que son nuevas o que se han cambiado en Dynamics 365 Talent.

## <a name="changes-in-attract"></a>Cambios en Attract

Este lanzamiento incluye correcciones de errores de menor importancia para Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Cambios en Onboard

Este lanzamiento incluye correcciones de errores de menor importancia para Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Cambios en Core HR

Los cambios que se describen en esta sección se aplican a la compilación número 8.1.2674. Los números entre paréntesis en algunos encabezados hacen referencia a los números de soporte en Microsoft Dynamics Lifecycle Services (LCS).

### <a name="print-performance-reviews"></a>Imprimir evaluaciones del rendimiento

Ahora las evaluaciones del rendimiento de impresión están disponibles en todos los entornos. Para habilitar esta funcionalidad, use el espacio de trabajo **Administración de características**. Para obtener más información acerca de la administración de características, consulte [Visión general de la Administración de características](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Para obtener más información sobre las evaluaciones del rendimiento de impresión, vea [Evaluaciones de rendimiento](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Puesto retirado con subordinados directos "flotantes" en el diagrama de organización como vacante (385436)

El lanzamiento de esta semana corrige cómo se muestran los puestos retirados con subordinados directos en la jerarquía de dependencia.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>El sistema no puede generar días de calendario (381195) 

Los cambios de esta semana corrigen un problema que impedía generar días de calendario.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>El informe de saldo actual en licencia de PBI duplica el saldo de empleado (385581)

Este cambio corrige el informe de saldo actual en este escenario.
