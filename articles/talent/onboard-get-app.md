# <a name="get-the-onboard-app"></a>Obtener la aplicación Onboard

[!include [banner](includes/banner.md)]

Puede ver una demostración y probar la aplicación Microsoft Dynamics 365 Talent: Onboard gratis en la [Página de producto de Onboard](https://dynamics.microsoft.com/talent/onboard/)servicios.

> [!NOTE]
> La prueba gratis requiere una cuenta de correo electrónico de negocio.

Puede adquirir una suscripción Onboard como una aplicación independiente o como parte de Dynamics 365 Talent. Para obtener más información sobre cómo comprar Onboard, consulte la [Página de producto de Onboard](https://dynamics.microsoft.com/talent/onboard/).

Durante el proceso de prueba o de compra, configurará su dirección de correo electrónico y contraseña de Microsoft 365. Asegúrese de anotar estos valores.

> [!WARNING]
> No puede migrar datos de la prueba a un entorno de la suscripción pagado. <!--Reviewers: please verify.-->

Para obtener más información sobre características nuevas en Talent, consulte [Novedades o cambios en Dynamics 365 Talent](./whats-new.md) y [Notas de la versión de Dynamics 365 y Power Platform](https://docs.microsoft.com/business-applications-release-notes/index). Si desea obtener una vista preliminar de las nuevas características en Onboard, consulte [Acceder a características de vista previa en Microsoft Dynamics 365 Talent](./access-preview-feature.md).

Si es profesional de TI y desea obtener más información sobre cómo se aprovisionan las dos versiones de la aplicación Onboard, consulte [Aprovisionamiento para la aplicación Dynamics 365 Talent - Onboard](./modular-app-tech-faq.md).

## <a name="get-started-with-onboard"></a>Introducción a Onboard

Cuando abre Onboard por primera vez, se le invita a iniciar un recorrido por el centro de administración de Microsoft 365. El centro de administración es donde se configura su organización, se gestionan los usuarios, y se gestionan las suscripciones. (Una de las suscripciones es su suscripción a Onboard). Para obtener más información acerca del centro de administración de Microsoft 365, consulte [Acerca del centro de administración de Microsoft 365](https://docs.microsoft.com/office365/admin/admin-overview/about-the-admin-center?view=o365-worldwide).

Para obtener la aplicación Onboard, siga estos pasos.

1. Abra la [página de inicio de sesión de Microsoft](https://portal.office.com/).
2. Cuando se le solicite, escriba su dirección de correo electrónico y contraseña de Microsoft 365.
3. Seleccione al iniciador de la aplicación en la parte superior izquierda y, a continuación, seleccione **Dynamics 365**.

    [![Acceso a Dynamics 365](./media/onboard-start-dynamics365.png)](./media/onboard-start-dynamics365.png)

4. Seleccione el mosaico **Talent: Onboard**.

    [![Abrir Onboard](./media/onboard-start-onboard.png)](./media/onboard-start-onboard.png)

Su primer inicio puede tardar algunos minutos, ya que el entorno debe ser inicializado.

## <a name="try-the-walkthrough"></a>Intente el tutorial

Cuando se abra Onboard por primera vez, puede seleccionar **Iniciar el tutorial** para comenzar con una plantilla en funcionamiento.

Si se salta el tutorial, puede obtener acceso más tarde a él si selecciona el botón **Ayuda** (**?**) y después seleccionando **Introducción**.

![[Iniciar el tutorial de Onboard](./media/onboard-start-walkthrough.png)](./media/onboard-start-walkthrough.png)

## <a name="change-the-domain-name"></a>Cambio del nombre de dominio

Si aceptó el nombre de dominio predeterminado al iniciar sesión con Onboard, puede cambiarlo a otro dominio posteriormente. (El nombre de dominio predeterminado termina por **onmicrosoft.com**).

1. Abra la [página de inicio de sesión de Microsoft](https://portal.office.com/).
2. Cuando se le solicite, escriba su dirección de correo electrónico y contraseña de Microsoft 365.
3. Si ve una recomendación para agregar su propio dominio en **Recomendado para usted**, seleccione **Ver recomendación**, y siga las indicaciones. Si no ve la recomendación, seleccione **Mostrar todo** en el menú de la izquierda, seleccione **Configuración**, seleccione **Dominios**, y luego seleccione **Agregar dominio** o **Comprar dominio**. A continuación siga las indicaciones.
