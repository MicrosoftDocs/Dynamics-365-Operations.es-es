    <span data-ttu-id="6c28c-121">Las notificaciones solo se muestran para las operaciones que se agregan en la pestaña desplegable **Notificaciones**, y puede agregar operaciones solo si se ha seleccionado la casilla **Habilitar notificaciones** para dichas operaciones en la página **Operaciones de PDV**.</span><span class="sxs-lookup"><span data-stu-id="6c28c-121">Notifications are shown only for operations that are added on the **Notifications** FastTab, and you can add operations there only if the **Enable notifications** check box for those operations has been selected on the **POS operations** page.</span></span> <span data-ttu-id="6c28c-122">Además, las notificaciones para una operación se muestran a los trabajadores solo si la operación se agrega a los permisos de PDV para esos trabajadores.</span><span class="sxs-lookup"><span data-stu-id="6c28c-122">Additionally, notifications for an operation are shown to workers only if the operation is added to the POS permissions for those workers.</span></span>
