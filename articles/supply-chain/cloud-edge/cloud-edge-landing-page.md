# <a name="scale-unit-capability-for-supply-chain-management-is-currently-paused-for-new-customers"></a>La capacidad de unidad de escalado para Supply Chain Management está actualmente en pausa para nuevos clientes

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> La capacidad de la unidad de escalado para Microsoft Dynamics 365 Supply Chain Management está en pausa para nuevos clientes ya que la arquitectura está experimentando mejoras. Actualmente, no aceptamos nuevos clientes para esta solución, pero estamos trabajando activamente en una arquitectura componible, flexible y fácil de usar para nuestra línea de negocios de Cadena de suministro que mejorará aún más nuestra capacidad para proporcionar una solución unitaria de escalado de clase mundial para nuestros clientes.
>
> Estamos trabajando para mejorar Dynamics 365 Supply Chain Management y permitir una mayor capacidad de composición en módulos específicos como Gestión de almacenes. Continuaremos planificando y desarrollando nuevas ofertas de unidades de escalado en la nube y el perímetro para satisfacer las necesidades de nuestros clientes. Para obtener más información, póngase en contacto con [d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com).
>
> La plataforma central de Commerce Scale Unit (CSU) para Microsoft Dynamics 365 Commerce no está relacionado con la capacidad de la unidad de escala para Dynamics 365 Supply Chain Management de cualquier manera. CSU Core no se ve afectado por este anuncio y todavía es totalmente compatible con la aplicación Dynamics 365 Commerce. Para obtener más información acerca de esa funcionalidad, consulte [Introducción a Commerce Scale Unit (CSU) Core](../../commerce/dev-itpro/CSU-core.md).
