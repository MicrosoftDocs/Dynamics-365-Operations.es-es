# <a name="view-the-status-of-service-orders"></a>Ver el estado de los pedidos de servicio 

[!include [banner](../includes/banner.md)]


El formulario **Panel de distribución** le permite ver el estado de las actividades de servicio de los pedidos de servicio.

## <a name="more-information-about-service-orders"></a>Más información acerca de los pedidos de servicio

El formulario también proporciona herramientas para realizar un seguimiento del flujo de pedidos de servicio e incluye detalles adicionales para ayudarle a evaluar la urgencia de un pedido de servicio, como el acuerdo de servicio y el acuerdo de nivel de servicio con los que está asociado.

El gráfico de Gantt admite el uso de colores para ayudarle a evaluar rápidamente la urgencia de un pedido de servicio. Los colores se usan para diferenciar la prioridad de una llamada de servicio. Estos colores son, por defecto, rojo para prioridad alta, amarillo para prioridad normal y verde para prioridad baja.


> [!TIP]
> <P>Puede cambiar los colores en la ficha <STRONG>Distribución</STRONG> del formulario <STRONG>Parámetros de la gestión de servicio</STRONG>.</P>
