# <a name="set-up-subscription-groups"></a><span data-ttu-id="8f1eb-103">Configurar grupos de suscripciones</span><span class="sxs-lookup"><span data-stu-id="8f1eb-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="8f1eb-104">Vaya a **Gestión de servicio** \> **Configuración** \> **Suscripciones de servicio** \> **Grupos de suscripciones**.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="8f1eb-105">Seleccionar **Nuevo** para crear un grupo de suscripciones nuevo.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="8f1eb-106">En el campo **Grupo de suscripciones** , escriba un nombre para el nuevo grupo de suscripciones.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="8f1eb-107">En el campo **Facturación**, seleccione una frecuencia de facturación para la suscripción.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="8f1eb-108">La frecuencia de facturación viene indicada por un código de período configurado en el formulario **Tipos de período**.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="8f1eb-109">Para acumular ingresos para transacciones de suscripción, active la casilla de verificación **Acumular ingresos** y, a continuación, en el campo **Acumulación**, seleccione una frecuencia.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="8f1eb-110">La frecuencia de acumulación se basa en un código de período configurado en el formulario **Períodos**.</span><span class="sxs-lookup"><span data-stu-id="8f1eb-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
