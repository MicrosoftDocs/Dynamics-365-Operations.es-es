# <a name="indexed-sales-prices"></a>Precios de ventas indexados  

[!include [banner](../includes/banner.md)]


El índice de un precio de ventas de suscripción se configura al crear una cuota de suscripción.

En el formulario **Crear cuota de suscripción**, configure el campo **Obtener precio de** en **Precio base indexado** y, a continuación, multiplique el precio base por el porcentaje del campo **Cambio del precio porcentual** para obtener el precio de ventas de la transacción de suscripción.

Por ejemplo, si el precio base es 1.000 EUR y el índice es 110, el precio de ventas será 1.100 EUR.
