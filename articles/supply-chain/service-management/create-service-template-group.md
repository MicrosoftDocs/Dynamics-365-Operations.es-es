# <a name="create-a-service-template-group"></a>Crear un grupo de plantillas de servicio  

[!include [banner](../includes/banner.md)]

Puede usar grupos de plantillas de servicio para ordenar y filtrar plantillas de servicio.

1. Haga clic en **Gestión de servicio** \> **Configurar** \> **Contratos de servicio** \> **Grupos de plantillas de servicio**.
2. Cree una plantilla de servicio nueva.
3. Especifique el identificador del grupo, una descripción y un grupo de plantillas de servicio principal, si lo hay.
4. Guarde el grupo de plantillas de servicio.
5. Establezca el control deslizante **Control de árbol** en **Sí** para abrir la vista de árbol y ver grupos de plantillas de servicio como una jerarquía en una vista de árbol o como una visión general.
