[!include [banner](../includes/banner.md)]


## <a name="credit-subscription-transactions"></a><span data-ttu-id="95548-104">Abonar transacciones de suscripción</span><span class="sxs-lookup"><span data-stu-id="95548-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="95548-105">Haga clic en **Gestión de servicio** \> **Común** \> **Suscripciones de servicio** \> **Todas las suscripciones de servicio**.</span><span class="sxs-lookup"><span data-stu-id="95548-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="95548-106">Seleccione la suscripción vinculada a la transacción de suscripción para la que desea crear una nota de abono.</span><span class="sxs-lookup"><span data-stu-id="95548-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="95548-107">Seleccione la ficha **Analizar** y, a continuación, haga clic en el botón **Transacciones de cuotas** del panel de acciones.</span><span class="sxs-lookup"><span data-stu-id="95548-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="95548-108">En el formulario **Transacciones de cuotas**, seleccione la transacción para la que desea crear una nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="95548-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="95548-109">Haga clic en **Funciones** \> **Seleccionar para nota de abono**.</span><span class="sxs-lookup"><span data-stu-id="95548-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="95548-110">En el formulario **Seleccionar para nota de abono**, seleccione la transacción que desee abonar y, a continuación, haga clic en **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="95548-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="95548-111">Al crear la nota de abono, asegúrese de seleccionar <STRONG>Notas de abono</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="95548-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="95548-112">Esto se encuentra en la lista <STRONG>Método de facturación</STRONG> del cuadro de diálogo <STRONG>Crear factura</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="95548-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="95548-113">Si el campo **Provisiones inversas en asignación de crédito** del formulario **Parámetros de gestión de servicio** se establece en **Manual**, deberá revertir individualmente cada transacción de ingresos acumulados para poder crear una propuesta de nota de abono para la transacción.</span><span class="sxs-lookup"><span data-stu-id="95548-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
