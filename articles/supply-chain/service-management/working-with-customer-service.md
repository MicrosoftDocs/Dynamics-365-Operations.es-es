# <a name="work-with-customer-service-overview"></a>Visión general del trabajo con el servicio al cliente

[!include [banner](../includes/banner.md)]


El Servicio al cliente está orientado a empresas profesionales basadas en servicios y organizaciones de servicios de fabricación.

El servicio al cliente incluye la siguiente funcionalidad:

  - Procesos clave para gestión de servicios, como acuerdos de servicio, pedidos de servicio y gestión de devoluciones

  - Configuración avanzada para gestión de reparaciones, que incluye el seguimiento de las reparaciones y la gestión de versiones de lista de materiales (L. MAT)

  - Servicio de soporte para roles de empleados, como el gerente de servicio, el distribuidor y el técnico o consultor de servicio

  - Suscripciones de servicio, que pueden generar ingresos periódicos con independencia de la frecuencia de dicho servicio

  - Contratos de nivel de servicio que satisfacen las necesidades de servicio de una base de clientes variada

En los temas de esta sección se proporciona información general acerca del proceso empresarial del servicio al cliente y de cada componente de dicho proceso. También se enumeran los formularios asociados a cada componente del proceso empresarial y se describen las tareas relacionadas con cada componente de dicho proceso.
