[!include [banner](../includes/banner.md)]


<span data-ttu-id="4be39-104">Puede crear suscripciones de servicio desde un proyecto en **Gestión de proyectos y contabilidad**.</span><span class="sxs-lookup"><span data-stu-id="4be39-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="4be39-105">Haga clic en **Gestión de proyectos y contabilidad** \> **Común** \> **Proyectos** \> **Todos los proyectos**.</span><span class="sxs-lookup"><span data-stu-id="4be39-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="4be39-106">Abra el proyecto para el que desea crear una suscripción.</span><span class="sxs-lookup"><span data-stu-id="4be39-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="4be39-107">En el formulario **Proyectos**, en el **Panel de acciones**, en la ficha **Administrar**, haga clic en **Servicio** \> **Suscripción de servicio**.</span><span class="sxs-lookup"><span data-stu-id="4be39-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="4be39-110">Especifique la información restante en el formulario.</span><span class="sxs-lookup"><span data-stu-id="4be39-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="4be39-111">Ahora puede crear transacciones de suscripción para la suscripción.</span><span class="sxs-lookup"><span data-stu-id="4be39-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="4be39-112">Si ha configurado una secuencia numérica para el grupo de suscripciones, se generará automáticamente un identificador de suscripción para la suscripción cuando especifica el grupo de suscripciones.</span><span class="sxs-lookup"><span data-stu-id="4be39-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
