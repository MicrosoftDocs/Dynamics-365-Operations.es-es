# <a name="customer-returns-overview"></a>Visión general de las devoluciones del cliente

[!include [banner](../includes/banner.md)]


Cree y utilice códigos de devolución y códigos de disposición para dar soporte al proceso de devolución de productos.

Use un código de motivo de devolución para describir el motivo por que el cliente desea devolver un artículo. Puede asignar un código de motivo en el formulario **Crear pedidos de devolución**.

Asigne un código de disposición cuando se recibe un artículo o durante la inspección física del artículo devuelto. Puede usar códigos de disposición para describir la condición del artículo. También puede usar códigos de disposición para indicar si es necesario realizar una acción adicional para la transacción. Por ejemplo, cree códigos de disposición de las siguientes acciones:

  - Desechar del artículo devuelto y ofrecer un artículo de sustitución al cliente.

  - Devolver el artículo al inventario y abonar al cliente el coste del artículo.

  - Reparar el artículo y devolverlo al cliente.
