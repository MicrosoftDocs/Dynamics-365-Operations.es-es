# <a name="set-up-service-intervals"></a><span data-ttu-id="20dcb-103">Configurar intervalos de servicio</span><span class="sxs-lookup"><span data-stu-id="20dcb-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="20dcb-104">El intervalo de servicio indica la frecuencia con la que se crean líneas de pedido de servicio para líneas de contrato de servicio cuando crea pedidos de servicio.</span><span class="sxs-lookup"><span data-stu-id="20dcb-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="20dcb-105">Haga clic en **Gestión de servicio** \> **Configurar** \> **Contratos de servicio** \> **Intervalos de servicio**.</span><span class="sxs-lookup"><span data-stu-id="20dcb-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="20dcb-106">Cree un nuevo intervalo de servicio.</span><span class="sxs-lookup"><span data-stu-id="20dcb-106">Create a new service interval.</span></span>
3. <span data-ttu-id="20dcb-107">Especifique el identificador y la descripción del intervalo de servicio.</span><span class="sxs-lookup"><span data-stu-id="20dcb-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="20dcb-108">Seleccione el intervalo en el campo **Intervalo**.</span><span class="sxs-lookup"><span data-stu-id="20dcb-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="20dcb-109">En el campo **Frecuencia**, especifique la frecuencia.</span><span class="sxs-lookup"><span data-stu-id="20dcb-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="20dcb-110">La frecuencia es el factor por el que debe multiplicar el intervalo para obtener el intervalo del acuerdo de servicio.</span><span class="sxs-lookup"><span data-stu-id="20dcb-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="20dcb-111">Presione **Alt+S** para guardar el intervalo de servicio.</span><span class="sxs-lookup"><span data-stu-id="20dcb-111">Press **Alt+S** to save the service interval.</span></span>
