# <a name="set-up-service-order-stages"></a>Configurar las etapas de pedido de servicio 

[!include [banner](../includes/banner.md)]


1.  Haga clic en **Gestión de servicio** \> **Configuración** \> **Pedidos de servicio** \> **Etapas de servicio**.

2.  Presione CTRL+N para crear un nuevo registro.

3.  En los campos **Etapa de servicio** y **Descripción**, especifique un id. de etapa de servicio y una descripción.

4.  Seleccione el parámetro más adecuado para la etapa.

5.  Seleccione la etapa principal para la etapa actual o deje el campo **Principal** en blanco si la etapa es la etapa inicial en la configuración de etapa.


> [!NOTE]
> <P>No puede modificar el campo <STRONG>Principal</STRONG> después de guardar la etapa. En lugar de ello, deberá eliminar el registro y crearlo de nuevo con una selección diferente en el campo <STRONG>Principal</STRONG>.</P>
> <P>Además, sólo puede crear una etapa con un campo <STRONG>Principal</STRONG> en blanco. Es decir, sólo se permite una sola etapa inicial.</P>
