# <a name="rebate-statuses"></a>Estados de devoluciones

[!include [banner](../includes/banner.md)]

Los estados de devoluciones le ayudan a realizar un seguimiento del estado de cada acuerdo de devoluciones. También puede utilizarlos con fines de informes y consultas. Puede configurar y nombrar tantos valores de estado de devolución como necesite para dar soporte a su proceso comercial. 

Para configurar estados de devoluciones, vaya a **Gestión de devoluciones \> Configuración \> Estado \> Estado**. Entonces use los botones del Panel de acciones para agregar y quitar valores de estados según sea necesario. Para cada estado, establezca los siguientes campos:

- **Estado de devolución**: introduzca un nombre para el estado (como *Negociando*, *Acordado*, o *Terminado*).
- **Descripción**: introduzca una descripción del estado para proporcionar más información sobre cómo debe usarse.

En la página [**Parámetros de gestión de devoluciones**](rebate-management-parameters.md), puede seleccionar un estado predeterminado. Luego, ese estado se asignará a todos los nuevos acuerdos de gestión de devoluciones que se creen.
