# <a name="planned-orders-simplified"></a>Pedidos planificados simplificados

[!include [banner](../../includes/banner.md)]

Este tema describe cómo puede ver, administrar y poner en firme pedidos planificados mediante la página simplificada **Pedidos planificados**.

La página **Órdenes previsionales simplificadas** es una versión simplificada de la página **Pedidos planificados**. Proporciona menos opciones, pero es más rápido trabajar con él cuando todo lo que tiene que hacer es revisar, aprobar y confirmar sus pedidos planificados.

## <a name="turn-on-this-feature-for-your-system"></a>Activar esta función para su sistema

Si su sistema aún no incluye las funciones descritas en este tema, vaya a [Gestión de funciones](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) y active la característica *Pedidos planificados simplificados*.

## <a name="view-manage-and-firm-planned-orders"></a>Ver, administrar y poner en firme pedidos planificados

La página **Órdenes previsionales simplificadas** está diseñada principalmente para reafirmar la funcionalidad. Muestra todos los pedidos planificados que se han generado para el plan seleccionado, al igual que hace la la página estándar **Pedidos planificados**. Por lo tanto, se puede utilizar como una alternativa optimizada para ver los resultados de una ejecución de planificación maestra.

Para abrir la página **Órdenes previsionales simplificadas**, vaya a **Planificacion maestra \> Planificacion maestra \> Órdenes previsionales simplificadas**. Puede usar esta página para completar las tareas siguientes:

- Ver todos los pedidos planificados para un plan específico.
- Crear nuevos pedidos planificados.
- Eliminar órdenes planificadas únicas o múltiples.
- Poner en firme órdenes planificadas únicas o múltiples.
- Cambie el estado del pedido planificado para pedidos planificados únicos o múltiples.
- Abra sin problemas los detalles del pedido planificado para un pedido planificado individual para ver los detalles completos del pedido.
- Cree vistas personalizadas utilizando vistas guardadas.

Escenarios adicionales, como split, son compatibles con la página estándar **Pedidos planificados**.
