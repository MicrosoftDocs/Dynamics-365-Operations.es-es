## <a name="coverage-codes"></a><span data-ttu-id="bf447-127">Códigos de cobertura</span><span class="sxs-lookup"><span data-stu-id="bf447-127">Coverage codes</span></span>

<span data-ttu-id="bf447-128">La planificación maestra se puede configurar para utilizar varios métodos de reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="bf447-128">Master planning can be configured to use different replenishment methods.</span></span> <span data-ttu-id="bf447-129">Los métodos de reabastecimiento o los métodos de medición de lotes son técnicas que utiliza el sistema para determinar el tamaño de lote para artículos comprados o producidos.</span><span class="sxs-lookup"><span data-stu-id="bf447-129">The replenishment methods or lot-sizing methods are the techniques used by the system to determine the batch size for purchased or produced items.</span></span> 

<span data-ttu-id="bf447-130">Cada método de reabastecimiento se asigna a uno de los códigos de cobertura siguientes:</span><span class="sxs-lookup"><span data-stu-id="bf447-130">Each replenishment method is assigned one of the following coverage codes:</span></span>

- <span data-ttu-id="bf447-131">**Manual** - El método de medición de lote donde el sistema no sugiere pedidos comprados, transferidos o de producción para el artículo.</span><span class="sxs-lookup"><span data-stu-id="bf447-131">**Manual** - The lot-sizing method where the system does not suggest purchased, transfer, or production orders for the item.</span></span> <span data-ttu-id="bf447-132">El planificador del artículo será responsable de realizar los pedidos necesarios para el reabastecimiento del artículo.</span><span class="sxs-lookup"><span data-stu-id="bf447-132">The planner for the item will be in charge of creating the required orders for the replenishment of the item.</span></span>
- <span data-ttu-id="bf447-133">**Por requisito** - El método de medición de lote en el que el sistema crea una compra, una transferencia, o un pedido de producción planificados de acuerdo con el requisito del artículo.</span><span class="sxs-lookup"><span data-stu-id="bf447-133">**Per requirement** - The lot-sizing method in which the system creates a planned purchase, transfer, or production order per requirement for the item.</span></span> <span data-ttu-id="bf447-134">Se usa normalmente para los artículos costosos con demanda intermitente.</span><span class="sxs-lookup"><span data-stu-id="bf447-134">This is generally used for expensive items with intermittent demand.</span></span>  
- <span data-ttu-id="bf447-135">**Por período** - el método de medición de lote que combina toda la demanda durante un período en un pedido del artículo.</span><span class="sxs-lookup"><span data-stu-id="bf447-135">**Per period** - The lot-sizing method that combines all the demand for a period into one order for the item.</span></span> <span data-ttu-id="bf447-136">El pedido se planificará se para el primer día del período y la cantidad satisfará los requisitos netos durante el período establecido.</span><span class="sxs-lookup"><span data-stu-id="bf447-136">The order will be planned for the first day of the period and its quantity will fulfill the net requirements during the established period.</span></span> <span data-ttu-id="bf447-137">El período comienza con la primera demanda del artículo y cubre la longitud definida de tiempo.</span><span class="sxs-lookup"><span data-stu-id="bf447-137">The period starts with the first demand of the item and covers the defined length in time.</span></span> <span data-ttu-id="bf447-138">El período siguiente comenzará con los siguientes requisitos del artículo.</span><span class="sxs-lookup"><span data-stu-id="bf447-138">The next period will start with the next requirements of the item.</span></span>
- <span data-ttu-id="bf447-139">**Mín/Máx.** - El método de medición de lote que contiene el reaprovisionamiento de inventario hasta un nivel determinado cuando la entrega prevista está por debajo de un umbral.</span><span class="sxs-lookup"><span data-stu-id="bf447-139">**Min/Max** - The lot-sizing method that contains the replenishment of inventory up to a certain level when the predicted on-hand is below a threshold.</span></span> <span data-ttu-id="bf447-140">La cantidad de reabastecimiento será la diferencia entre el nivel máximo y el nivel disponible previsto.</span><span class="sxs-lookup"><span data-stu-id="bf447-140">The replenishment quantity will be the difference between the maximum level and the predicted on-hand level.</span></span>
