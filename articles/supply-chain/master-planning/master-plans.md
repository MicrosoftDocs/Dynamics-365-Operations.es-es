# <a name="master-plans"></a><span data-ttu-id="66e21-103">Planes maestros</span><span class="sxs-lookup"><span data-stu-id="66e21-103">Master plans</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="66e21-104">Utilice varios planes maestros para dar soporte a las operaciones de trabajo diarias de la empresa, simular distintas estrategias de planificación que desee supervisar e implementar una directiva empresarial, como una directiva sobre rendimiento interno o la satisfacción del cliente.</span><span class="sxs-lookup"><span data-stu-id="66e21-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="66e21-105">Puede configurar planes maestros en la página **Planes maestros**.</span><span class="sxs-lookup"><span data-stu-id="66e21-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="66e21-106">Hay dos tipos de planes:</span><span class="sxs-lookup"><span data-stu-id="66e21-106">There are two types of plans:</span></span>
-   <span data-ttu-id="66e21-107">**Plan estático**: el cálculo de la planificación maestra usa los datos actuales para generar un plan neto de los requisitos.</span><span class="sxs-lookup"><span data-stu-id="66e21-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="66e21-108">Este plan permanece sin cambios hasta la próxima vez que se ejecute la planificación maestra.</span><span class="sxs-lookup"><span data-stu-id="66e21-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="66e21-109">Es un plan operativo que varios miembros del personal de la empresa, como el comprador o el encargado de la planificación de producción, pueden utilizar para basar sus decisiones en él y realizar sus actividades y tareas diarias.</span><span class="sxs-lookup"><span data-stu-id="66e21-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="66e21-110">**Plan dinámico**: este plan se inicia con el mismo plan de requisitos netos que generó la planificación maestra.</span><span class="sxs-lookup"><span data-stu-id="66e21-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="66e21-111">No obstante, puede actualizar el plan dinámico cada vez que cambien los datos maestros.</span><span class="sxs-lookup"><span data-stu-id="66e21-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="66e21-112">Esto podría ser al crear un nuevo pedido de ventas, por ejemplo.</span><span class="sxs-lookup"><span data-stu-id="66e21-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="66e21-113">Esto le permite supervisar la red de pedido de cambio y la disponibilidad de artículos sin alterar el plan estático que otros utilizan para sus procesos de trabajo.</span><span class="sxs-lookup"><span data-stu-id="66e21-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="66e21-114">Una empresa puede elegir trabajar solo con un plan dinámico o puede utilizar ambos planes, el estático y el dinámico.</span><span class="sxs-lookup"><span data-stu-id="66e21-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="66e21-115">Además, puede configurar cualquier plan maestro para reflejar una estrategia concreta o dirigir un asunto.</span><span class="sxs-lookup"><span data-stu-id="66e21-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="66e21-116">Los ejemplos son:</span><span class="sxs-lookup"><span data-stu-id="66e21-116">Examples are as follows:</span></span>
-   <span data-ttu-id="66e21-117">Establecer niveles de inventario superiores para garantizar que no se agoten las existencias.</span><span class="sxs-lookup"><span data-stu-id="66e21-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="66e21-118">Establecer márgenes de seguridad mayores para protección frente a los proveedores poco fiables.</span><span class="sxs-lookup"><span data-stu-id="66e21-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="66e21-119">También puede configurar el inicio del plan dinámico de modo que se actualice con el nuevo plan de requisitos cada vez que se ejecute la planificación maestra.</span><span class="sxs-lookup"><span data-stu-id="66e21-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="66e21-120">Puede especificar estos ajustes en la página **Parámetros de planificación maestra**.</span><span class="sxs-lookup"><span data-stu-id="66e21-120">You can specify these settings on the **Master planning parameters** page.</span></span>
