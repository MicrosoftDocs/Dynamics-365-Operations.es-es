## <a name="item-allocation-key"></a><span data-ttu-id="2e722-106">Clave de asignación de artículos</span><span class="sxs-lookup"><span data-stu-id="2e722-106">Item allocation key</span></span>
<span data-ttu-id="2e722-107">Una previsión de la demanda se calcula para un artículo y sus dimensiones únicamente si el artículo es parte de una clave de asignación de artículos.</span><span class="sxs-lookup"><span data-stu-id="2e722-107">A demand forecast is calculated for an item and its dimensions only if the item is part of an item allocation key.</span></span> <span data-ttu-id="2e722-108">Esta regla se aplica para agrupar un gran número de artículos, para poder crear previsiones de demanda más rápidamente.</span><span class="sxs-lookup"><span data-stu-id="2e722-108">This rule is enforced to group large numbers of items, so that demand forecasts can be created more quickly.</span></span> <span data-ttu-id="2e722-109">Se omite el porcentaje de la clave de asignación de artículos cuando se generan previsiones de demanda.</span><span class="sxs-lookup"><span data-stu-id="2e722-109">The item allocation key percentage is ignored when demand forecasts are generated.</span></span> <span data-ttu-id="2e722-110">Las previsiones se crean únicamente en base a los datos históricos.</span><span class="sxs-lookup"><span data-stu-id="2e722-110">Forecasts are created based on historical data only.</span></span> 

<span data-ttu-id="2e722-111">Un artículo y sus dimensiones deben formar parte de una sola clave de asignación de artículos si la clave de asignación de artículos se utiliza durante la creación de la previsión.</span><span class="sxs-lookup"><span data-stu-id="2e722-111">An item and its dimensions must be part of only one item allocation key if the item allocation key is used during forecast creation.</span></span> 

<span data-ttu-id="2e722-112">Para agregar una referencia de almacén (SKU) a una clave de asignación de artículos, vaya a **Planificación maestra** &gt; **Configuración** &gt; **Previsión de demanda** &gt; **Claves de asignación de artículos**.</span><span class="sxs-lookup"><span data-stu-id="2e722-112">To add a stock keeping unit (SKU) to an item allocation key, go to **Master planning** &gt; **Setup** &gt; **Demand forecasting** &gt; **Item allocation keys**.</span></span> <span data-ttu-id="2e722-113">Use la página **Asignar artículos** para asignar un artículo a una clave de asignación.</span><span class="sxs-lookup"><span data-stu-id="2e722-113">Use the **Assign items** page to assign an item to an allocation key.</span></span>

## <a name="intercompany-planning-groups"></a><span data-ttu-id="2e722-114">Grupos de planificación de empresas vinculadas</span><span class="sxs-lookup"><span data-stu-id="2e722-114">Intercompany planning groups</span></span>
<span data-ttu-id="2e722-115">La previsión de demanda genera previsiones entre empresas.</span><span class="sxs-lookup"><span data-stu-id="2e722-115">Demand forecasting generates cross-company forecasts.</span></span> <span data-ttu-id="2e722-116">En Microsoft Dynamics 365 for Finance and Operations, se agrupa a las empresas que se planifican conjuntamente en un grupo de planificación de empresas vinculadas.</span><span class="sxs-lookup"><span data-stu-id="2e722-116">In Microsoft Dynamics 365 for Finance and Operations, companies that are planned together are grouped into one intercompany planning group.</span></span> <span data-ttu-id="2e722-117">Para especificar, según la empresa, las claves de asignación de artículos que se deben tener en cuenta para la previsión de demanda, asocie una clave de asignación de artículos con el miembro de planificación de empresas vinculadas del grupo yendo a **Planificación maestra** &gt; **Configuración** &gt; **Grupos de planificación de empresas vinculadas**.</span><span class="sxs-lookup"><span data-stu-id="2e722-117">To specify, per company, which item allocation keys should be considered for demand forecasting, associate an item allocation key with the intercompany planning group member by going to **Master planning** &gt; **Setup** &gt; **Intercompany planning groups**.</span></span> 

<span data-ttu-id="2e722-118">De forma predeterminada, si no se ha asignado ninguna clave de asignación de artículos a los miembros del grupo de planificación de empresas vinculadas, se calcula una previsión de la demanda para todos los artículos asignados a todas las claves de asignación de artículos de todas las empresas de Dynamics 365 for Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="2e722-118">By default, if no item allocation keys are assigned to intercompany planning group members, a demand forecast is calculated for all items that are assigned to all item allocation keys from all Finance and Operations companies.</span></span> <span data-ttu-id="2e722-119">Las opciones de filtrado adicionales para las empresas y las claves de asignación de artículos están disponibles en la página **Generar la previsión estadística de línea base**.</span><span class="sxs-lookup"><span data-stu-id="2e722-119">Additional filtering options for companies and item allocation keys are available on the **Generate statistical baseline forecast** page.</span></span> 

<span data-ttu-id="2e722-120">Revisión del número de artículos que están previstos.</span><span class="sxs-lookup"><span data-stu-id="2e722-120">Review the number of items that are forecasted.</span></span> <span data-ttu-id="2e722-121">Los artículos innecesarios podrían incrementar los costes cuando se usa el aprendizaje automático de Microsoft Azure.</span><span class="sxs-lookup"><span data-stu-id="2e722-121">Unnecessary items might cause increased costs when you use Microsoft Azure Machine Learning.</span></span>

## <a name="demand-forecasting-parameters"></a><span data-ttu-id="2e722-122">Parámetros de previsión de la demanda</span><span class="sxs-lookup"><span data-stu-id="2e722-122">Demand forecasting parameters</span></span>
<span data-ttu-id="2e722-123">Para configurar los parámetros de previsión de la demanda, vaya a **Planificación maestra** &gt; **Configuración** &gt; **Parámetros de previsión de la demanda**.</span><span class="sxs-lookup"><span data-stu-id="2e722-123">To set up demand forecasting parameters, go to **Master planning** &gt; **Setup** &gt; **Demand forecasting parameters**.</span></span> <span data-ttu-id="2e722-124">Dado que la previsión de demanda se realiza entre empresas, la configuración es global.</span><span class="sxs-lookup"><span data-stu-id="2e722-124">Because demand forecasting runs cross-company, the setup is global.</span></span> <span data-ttu-id="2e722-125">Es decir, la configuración se aplica a todas las empresas.</span><span class="sxs-lookup"><span data-stu-id="2e722-125">In other words, the setup applies to all companies.</span></span> 

<span data-ttu-id="2e722-126">La previsión de demanda genera la previsión en cantidades.</span><span class="sxs-lookup"><span data-stu-id="2e722-126">Demand forecasting generates the forecast in <span data-ttu-id="2e722-132">La previsión de demanda se puede usar para hacer una previsión de la demanda dependiente y la demanda independiente.</span><span class="sxs-lookup"><span data-stu-id="2e722-132">Demand forecasting can be used to forecast both dependent and independent demand.</span></span> <span data-ttu-id="2e722-133">Por ejemplo, si solo se activa la casilla **Pedido de ventas** y si todos los artículos que se consideran para la previsión de demanda son los artículos vendidos, el sistema calculará demanda independiente.</span><span class="sxs-lookup"><span data-stu-id="2e722-133">For example, if only the **Sales order** check box is selected, and if all the items that are considered for demand forecasting are items that are sold, the system calculates independent demand.</span></span> <span data-ttu-id="2e722-134">Sin embargo, se pueden agregas subcomponentes importantes a las claves de asignación de artículos e incluirlos en la previsión de demanda.</span><span class="sxs-lookup"><span data-stu-id="2e722-134">However, critical subcomponents can be added to item allocation keys and included in demand forecasting.</span></span> <span data-ttu-id="2e722-135">En este caso, si se activa la casilla **Línea de producción**, se calcula una previsión dependiente.</span><span class="sxs-lookup"><span data-stu-id="2e722-135">In this case, if the **Production line** check box is selected, a dependent forecast is calculated.</span></span>

<span data-ttu-id="2e722-140">Si hace clic en **Dimensiones de previsión** en el panel izquierdo de la página **Parámetros de previsión de demanda**, también podrá seleccionar el conjunto de dimensiones de previsión que desea usar cuando se genera la previsión de la demanda.</span><span class="sxs-lookup"><span data-stu-id="2e722-140">By clicking **Forecast dimensions** in the left pane of the **Demand forecasting parameters** page, you can also select the set of forecast dimensions to use when the demand forecast is generated.</span></span> <span data-ttu-id="2e722-141">Una dimensión de la previsión indica el nivel de detalle para el que se define la previsión.</span><span class="sxs-lookup"><span data-stu-id="2e722-141">A forecast dimension indicates the level of detail that the forecast is defined for.</span></span> <span data-ttu-id="2e722-142">Empresa, sitio y la clave de asignación de artículos son dimensiones obligatorias de previsión, pero también puede generar previsiones en el almacén, estado de inventario, grupo de clientes, cuenta de cliente, país o región, comunidad autónoma y artículo además de todos los niveles de la dimensión de artículo.</span><span class="sxs-lookup"><span data-stu-id="2e722-142">Company, site, and item allocation key are mandatory forecast dimensions, but you can also generate forecasts at the warehouse, inventory status, customer group, customer account, country/region, state, and item plus all item dimension levels.</span></span>
