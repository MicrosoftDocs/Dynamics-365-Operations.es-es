# <a name="create-a-value-stream"></a>Crear un flujo de valor

[!include[task guide banner](../../includes/task-guide-banner.md)]

Los flujos de valor son unidades operativas, que agrupan uno o varios flujos de producción. Para definir un flujo de producción, debe definir primero un flujo de valor.


## <a name="open-the-form"></a>Abra el formulario.
1. Vaya a Control de producción > Configuración > Flujo de producción lean > Flujos de valor.

## <a name="create-a-new-value-stream"></a>Crear un nuevo flujo de valor
1. Haga clic en Nueva para abrir el cuadro de diálogo desplegable.
2. En el campo Nombre, escriba un valor.

## <a name="select-a-value-stream-manager"></a>Seleccione un administrador de flujo de valor
1. En el campo Administrador, haga clic en el botón desplegable para abrir la búsqueda.
    * El administrador de flujo de valor se define para documentar la organización.  
2. En la lista, busque y seleccione el registro deseado.
3. En la lista, haga clic en el vínculo de la fila seleccionada.

## <a name="save-and-close"></a>Guardar y cerrar
1. Haga clic en Guardar.
2. Cierre la página.
3. Actualice la página.
