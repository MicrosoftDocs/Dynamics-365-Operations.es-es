## <a name="work-templates"></a><span data-ttu-id="24148-108">Plantillas de trabajo</span><span class="sxs-lookup"><span data-stu-id="24148-108">Work templates</span></span>

<span data-ttu-id="24148-109">La página **Plantillas de trabajo** le permite definir las operaciones de trabajo que se deben realizar en el almacén.</span><span class="sxs-lookup"><span data-stu-id="24148-109">The **Work templates** page lets you define the work operations that must be performed in the warehouse.</span></span> <span data-ttu-id="24148-110">Normalmente, las operaciones de trabajo del almacén constan de un par de acciones: un trabajador de almacén recoge inventario disponible en una ubicación y después pone el inventario seleccionado en otra ubicación.</span><span class="sxs-lookup"><span data-stu-id="24148-110">Typically, warehouse work operations consist of a pair of actions: a warehouse worker picks up on-hand inventory in one location and then puts the picked inventory down in another location.</span></span> 

<span data-ttu-id="24148-111">Las plantillas de trabajo constan de un encabezado y las líneas asociadas.</span><span class="sxs-lookup"><span data-stu-id="24148-111">Work templates consists of a header and associated lines.</span></span> <span data-ttu-id="24148-112">Cada plantilla de trabajo es para un determinado *tipo de pedido de trabajo*.</span><span class="sxs-lookup"><span data-stu-id="24148-112">Each work template is for a specific *work order type*.</span></span> <span data-ttu-id="24148-113">Muchos tipos de pedido de trabajo están asociados con los documentos de origen, como pedidos de ventas o compras.</span><span class="sxs-lookup"><span data-stu-id="24148-113">Many work order types are associated with source documents, such as purchase or sales orders.</span></span> <span data-ttu-id="24148-114">Sin embargo, otros tipos de pedido de trabajo representan procesos independientes de almacén, como recuento cíclico.</span><span class="sxs-lookup"><span data-stu-id="24148-114">However, other work order types represent separate warehouse processes, such as cycle counting.</span></span> <span data-ttu-id="24148-115">El *id. del grupo de trabajo* le permite organizar el trabajo en grupos.</span><span class="sxs-lookup"><span data-stu-id="24148-115">The *work pool ID* lets you organize work into groups.</span></span> 

<span data-ttu-id="24148-116">Use los ajustes en la definición del encabezado del trabajo para determinar cuándo se debe crear un nuevo elemento de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-116">Use the settings in the work header definition to determine when a new piece of work should be created.</span></span> <span data-ttu-id="24148-117">Por ejemplo, puede establecer un número máximo de líneas de selcción y un máximo previsto de tiempo de selección.</span><span class="sxs-lookup"><span data-stu-id="24148-117">For example, you can set a maximum number of pick lines and a maximum expected pick time.</span></span> <span data-ttu-id="24148-118">Luego, si el trabajo para un proceso de selección de un pedido de ventas supera cualquiera de estos valores, el trabajo se divide en dos partes.</span><span class="sxs-lookup"><span data-stu-id="24148-118">Then, if the work for a sales order picking process exceeds either of those values, that work is split into two pieces of work.</span></span>

<span data-ttu-id="24148-123">Las líneas de trabajo representan las tareas físicas que son necesarias para procesar el trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-123">The work lines represent the physical tasks that are required to process the work.</span></span> <span data-ttu-id="24148-124">Por ejemplo, para un proceso de salida de almacén, puede haber una línea de trabajo para seleccionar los artículos dentro del almacén y otra línea para configurar dichos artículos en una zona de espera.</span><span class="sxs-lookup"><span data-stu-id="24148-124">For example, for an outbound warehouse process, there might be one line for picking up the items in the warehouse and another line for putting those items into a staging area.</span></span> <span data-ttu-id="24148-125">Después, puede haber una línea adicional para seleccionar los artículos provisionales de la zona de espera, y otra línea para poner los artículos en un camión como parte del proceso de carga.</span><span class="sxs-lookup"><span data-stu-id="24148-125">There can then be an additional line for picking the items from staging and another line for putting the items into a truck as part of the loading process.</span></span> <span data-ttu-id="24148-126">Puede establecer unn *código directivo* en líneas de la plantilla de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-126">You can set a *directive code* on work template lines.</span></span> <span data-ttu-id="24148-127">Un código directivo está vinculado a una ubicación directiva y, por tanto, ayuda a garantizan que el trabajo del almacén se ha procesado en la ubicación correcta en el almacén.</span><span class="sxs-lookup"><span data-stu-id="24148-127">A directive code is linked to a location directive, and therefore helps ensure that the warehouse work is processed in the correct location in the warehouse.</span></span>

<span data-ttu-id="24148-128">Puede configurar una consulta para controlar si se usa una determinada plantilla de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-128">You can set up a query to control when a particular work template is used.</span></span> <span data-ttu-id="24148-129">Por ejemplo, puede establecer una limitación para poder utilizar una determinada plantilla solo para el trabajo en un almacén específico.</span><span class="sxs-lookup"><span data-stu-id="24148-129">For example, you can set a limitation so that a particular template can be used for work only in a specific warehouse.</span></span> <span data-ttu-id="24148-130">Como alternativa, puede tener varias plantillas que se usan para crear el trabajo para procesar pedidos de venta salientes, en función del origen de venta.</span><span class="sxs-lookup"><span data-stu-id="24148-130">Alternatively, you might have several templates that create work for outbound sales order processing, depending on the sales origin.</span></span> <span data-ttu-id="24148-131">El sistema usa el campo **Número de secuencia** para determinar el orden en que las plantillas de trabajo disponibles son evaluadas.</span><span class="sxs-lookup"><span data-stu-id="24148-131">The system uses the **Sequence number** field to determine the order that the available work templates are assessed in.</span></span> <span data-ttu-id="24148-132">Por lo tanto, si tiene una consulta muy específica para una determinada plantilla de trabajo, debe darle un número de secuencia bajo.</span><span class="sxs-lookup"><span data-stu-id="24148-132">Therefore, if you have a very specific query for a particular work template, you should give it a low sequence number.</span></span> <span data-ttu-id="24148-133">Dicha consulta se evaluará antes que otras consultas más generales.</span><span class="sxs-lookup"><span data-stu-id="24148-133">That query will then be evaluated before the other, more general queries.</span></span>

> [!NOTE]
> <span data-ttu-id="24148-134">Para evitar que el sistema sobrescriba automáticamente los *números de secuencia* de la plantilla de trabajo una vez eliminada una plantilla, active la función *Conservar los números de secuencia de la plantilla de trabajo al eliminar* en [Gestión de funciones](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).</span><span class="sxs-lookup"><span data-stu-id="24148-134">To prevent the system from automatically overwriting work template *sequence numbers* after a template has been deleted, turn on the *Preserve work template sequence numbers on delete* feature in [Feature management](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).</span></span>

<span data-ttu-id="24148-135">Para detener o poner en pausa un proceso de trabajo, puede usar la configuración **Detener trabajo** en la línea de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-135">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="24148-136">En ese caso, no se le solicitará al trabajador que se está realizando el trabajo que realice la siguiente etapa de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-136">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="24148-137">Para pasar al siguiente paso, dicho trabajador u otro trabajador debe seleccionar el trabajo de nuevo.</span><span class="sxs-lookup"><span data-stu-id="24148-137">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="24148-138">También puede separar las tareas dentro de un elemento de trabajo mediante un *id. de clase de trabajo* diferente en las líneas de la plantilla de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-138">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="24148-139">Directivas de ubicación</span><span class="sxs-lookup"><span data-stu-id="24148-139">Location directives</span></span>

<span data-ttu-id="24148-140">Las directivas de ubicación son reglas que ayudan a identifica las ubicaciones de picking y de colocación para el movimiento de inventario.</span><span class="sxs-lookup"><span data-stu-id="24148-140">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="24148-141">Por ejemplo, en una transacción de pedido de ventas, una directiva de ubicación determina dónde se hará el picking de los artículos y dónde se colocarán los artículos seleccionados.</span><span class="sxs-lookup"><span data-stu-id="24148-141">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="24148-142">Los directorios de la ubicación constan de un encabezado y las líneas asociadas, y se crean en la página **Directorios de ubicación**.</span><span class="sxs-lookup"><span data-stu-id="24148-142">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span>

<span data-ttu-id="24148-143">En el encabezado, cada directorio de ubicación se debe asociar a un *tipo de pedido de trabajo* que especifica el tipo de transacción de inventario para el cual el directorio se usará, como pedidos de ventas, reabastecimiento o selección de la materia prima.</span><span class="sxs-lookup"><span data-stu-id="24148-143">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="24148-144">El *tipo de trabajo* especifica si el directorio de la ubicación se usará para seleccionar o colocar el trabajo, o para otro proceso de almacén, como cómputo o cambios de estado del inventario.</span><span class="sxs-lookup"><span data-stu-id="24148-144">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="24148-145">También debe especificar un *sitio* y un *almacén*.</span><span class="sxs-lookup"><span data-stu-id="24148-145">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="24148-146">Un *código directivo* que se especifica en el encabezado se puede usar para vincular el directorio de la ubicación a una o varias plantillas de trabajo.</span><span class="sxs-lookup"><span data-stu-id="24148-146">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span> 

<span data-ttu-id="24148-147">En cuanto a plantillas de trabajo, puede configurar una consulta para determinar cuándo se usa un directorio concreto de la ubicación.</span><span class="sxs-lookup"><span data-stu-id="24148-147">As for work templates, you can set up a query to determine when a particular location directive is used.</span></span> <span data-ttu-id="24148-148">Por ejemplo, puede especificar que cuando el comercio electrónico es el origen de un pedido de ventas, el inventario se debe seleccionar en un área dedicada en el almacén.</span><span class="sxs-lookup"><span data-stu-id="24148-148">For example, you can specify that when e-commerce is the origin of a sales order, the inventory must be picked up from a dedicated area in the warehouse.</span></span> <span data-ttu-id="24148-149">El sistema usa el campo **Número de secuencia** para determinar el orden en que las directivas disponibles de la ubicación son evaluadas.</span><span class="sxs-lookup"><span data-stu-id="24148-149">The system uses the **Sequence number** field to determine the order that the available location directives are assessed in.</span></span>

<span data-ttu-id="24148-150">Las líneas directivas de la ubicación establecen restricciones adicionales en la aplicación de las reglas de búsqueda de la ubicación.</span><span class="sxs-lookup"><span data-stu-id="24148-150">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="24148-151">Puede especificar una cantidad mínima y una cantidad máxima en las que el directorio se debe basar, y puede especificar que el directorio debe ser para una unidad de inventario específica.</span><span class="sxs-lookup"><span data-stu-id="24148-151">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="24148-152">Por ejemplo, si la unidad de medida es pallets, entonces los artículos en pallets se pueden colocar en una ubicación concreta.</span><span class="sxs-lookup"><span data-stu-id="24148-152">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="24148-153">También puede especificar si la cantidad se puede dividir entre varias ubicaciones.</span><span class="sxs-lookup"><span data-stu-id="24148-153">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="24148-154">Como el encabezado de la directiva de ubicación, cada línea directiva de la ubicación tiene un número de secuencia que determina la orden en que las líneas se evalúan.</span><span class="sxs-lookup"><span data-stu-id="24148-154">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>

<span data-ttu-id="24148-160">Para obtener más información sobre cómo crear y configurar directivas de ubicación, consulte [Crear una directiva de ubicación](create-location-directive.md).</span><span class="sxs-lookup"><span data-stu-id="24148-160">For more information about how to create and configure location directives, see [Create a location directive](create-location-directive.md).</span></span>

### <a name="how-location-directives-work"></a><span data-ttu-id="24148-161">Cómo funcionan las directivas de ubicación</span><span class="sxs-lookup"><span data-stu-id="24148-161">How location directives work</span></span>
