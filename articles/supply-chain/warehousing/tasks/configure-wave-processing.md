# <a name="configure-wave-processing"></a><span data-ttu-id="ba96a-103">Configurar procesamiento de oleadas</span><span class="sxs-lookup"><span data-stu-id="ba96a-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="ba96a-104">En esta guía se describe cómo configurar los criterios que determinan qué trabajo se genera para un almacén cuando se procesa una oleada y si las oleadas se procesan manual o automáticamente.</span><span class="sxs-lookup"><span data-stu-id="ba96a-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="ba96a-105">Especifique los criterios configurando consultas y plantillas de oleada que hagan coincidir una oleada con las líneas emitidas en pedidos de ventas, pedidos de producción o pedidos kanban.</span><span class="sxs-lookup"><span data-stu-id="ba96a-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="ba96a-106">El procesamiento de oleadas se usa en almacenes que usan la funcionalidad del módulo Administración de almacenes, y no los que usan la funcionalidad del módulo Gestión del inventario.</span><span class="sxs-lookup"><span data-stu-id="ba96a-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="ba96a-107">Puede ejecutar este procedimiento en la empresa USMF de los datos de prueba.</span><span class="sxs-lookup"><span data-stu-id="ba96a-107">You can run this procedure in demo data company USMF.</span></span>

1. <span data-ttu-id="ba96a-108">Vaya a Gestión de almacenes > Configurar > Oleadas > Plantillas de oleada.</span><span class="sxs-lookup"><span data-stu-id="ba96a-108">Go to Warehouse management > Setup > Waves > Wave templates.</span></span>
2. <span data-ttu-id="ba96a-109">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="ba96a-109">Click New.</span></span>
3. <span data-ttu-id="ba96a-110">En el campo Nombre de la plantilla de oleada, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="ba96a-110">In the Wave template name field, type a value.</span></span>
    * <span data-ttu-id="ba96a-111">Al configurar una plantilla de oleada, especifica la secuencia en la que las plantillas se conciliarán con las líneas liberadas en pedidos de ventas, pedidos de producción o kanbans.</span><span class="sxs-lookup"><span data-stu-id="ba96a-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="ba96a-112">Cuando una línea se libera al almacén o a producción, usa la primera plantilla de oleada para la que cumple los criterios.</span><span class="sxs-lookup"><span data-stu-id="ba96a-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="ba96a-113">Se recomienda que configure las plantillas con los criterios más específicos en la parte superior de la lista.</span><span class="sxs-lookup"><span data-stu-id="ba96a-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="ba96a-114">Cuanto más amplios sean los criterios, será más probable que una línea cumpla los criterios y esto podría originar líneas asignadas en la oleada incorrecta.</span><span class="sxs-lookup"><span data-stu-id="ba96a-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>  
4. <span data-ttu-id="ba96a-115">En el campo Descripción de la plantilla de oleada, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="ba96a-115">In the Wave template description field, type a value.</span></span>
5. <span data-ttu-id="ba96a-116">En el campo Sitio, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="ba96a-116">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="ba96a-117">Si está usando USMF, puede seleccionar el sitio 2.</span><span class="sxs-lookup"><span data-stu-id="ba96a-117">If you’re using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="ba96a-118">En el campo Almacén, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="ba96a-118">In the Warehouse field, enter or select a value.</span></span>
    * <span data-ttu-id="ba96a-119">Si está usando USMF, puede seleccionar el almacén 24.</span><span class="sxs-lookup"><span data-stu-id="ba96a-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="ba96a-120">Establezca el campo Automatizar la creación de oleadas en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-120">Set the Automate wave creation field to Yes.</span></span>
    * <span data-ttu-id="ba96a-121">Active esta opción para crear automáticamente una oleada cuando un pedido de ventas, un pedido de producción o un kanban se libera al almacén.</span><span class="sxs-lookup"><span data-stu-id="ba96a-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>

9. <span data-ttu-id="ba96a-124">Establezca la opción Liberación automática de oleada en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-124">Set the Automate wave release option to Yes.</span></span> 
    * <span data-ttu-id="ba96a-125">Active esta opción para liberar la oleada automáticamente.</span><span class="sxs-lookup"><span data-stu-id="ba96a-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="ba96a-126">El trabajo de picking se crea y se pone a disposición en los dispositivos móviles.</span><span class="sxs-lookup"><span data-stu-id="ba96a-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="ba96a-127">Establezca la opción Asignar a oleadas abiertas en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-127">Set the Assign to open waves option to Yes.</span></span> 
    * <span data-ttu-id="ba96a-128">Las líneas se asignan a oleadas basadas en el filtro de consulta de la plantilla de oleada.</span><span class="sxs-lookup"><span data-stu-id="ba96a-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="ba96a-129">Establezca la opción Procesar la oleada automáticamente al alcanzar el umbral en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-129">Set the Process wave automatically at threshold option to Yes.</span></span> 
    * <span data-ttu-id="ba96a-130">Active esta opción para procesar automáticamente la oleada cuando sus valores lleguen a los umbrales del peso, el envío y las líneas especificadas en el grupo de campos Umbrales de oleada.</span><span class="sxs-lookup"><span data-stu-id="ba96a-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="ba96a-131">Esta opción sólo está disponible si se ha seleccionado Envío en el campo Tipo de plantilla oleada.</span><span class="sxs-lookup"><span data-stu-id="ba96a-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>  
12. <span data-ttu-id="ba96a-132">Establezca la opción Automatizar liberación de trabajo de reabastecimiento en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-132">Set the Automate replenishment work release option to Yes.</span></span>

18. <span data-ttu-id="ba96a-146">En el campo Grupo de lotes de procesamiento de oleadas, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="ba96a-146">In the Wave processing batch group field, enter or select a value.</span></span>
19. <span data-ttu-id="ba96a-147">Establezca la opción Procesar oleadas en lote en Sí.</span><span class="sxs-lookup"><span data-stu-id="ba96a-147">Set the Process waves in batch option to Yes.</span></span>

22. <span data-ttu-id="ba96a-152">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="ba96a-152">Close the page.</span></span>
