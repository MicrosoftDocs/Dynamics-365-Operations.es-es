# <a name="replenishment-overview"></a><span data-ttu-id="56617-103">Visión general de la configuración de reabastecimiento</span><span class="sxs-lookup"><span data-stu-id="56617-103">Replenishment overview</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="56617-104">En este tema se describen las estrategias de reabastecimiento que están disponibles para los almacenes que utilizan la funcionalidad disponible en Gestión de almacenes.</span><span class="sxs-lookup"><span data-stu-id="56617-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="56617-105">La información de este tema no se aplica a la solución de almacenamiento que está disponible en Gestión de inventarios.</span><span class="sxs-lookup"><span data-stu-id="56617-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

<span data-ttu-id="56617-106">Las siguientes estrategias de reabastecimiento están disponibles:</span><span class="sxs-lookup"><span data-stu-id="56617-106">The following replenishment strategies are available:</span></span>

- <span data-ttu-id="56617-107">**Reabastecimiento de demanda de oleada**: esta estrategia crea el trabajo de reabastecimiento para los pedidos o las cargas de salida si el inventario no está disponible cuando la oleada crea el trabajo.</span><span class="sxs-lookup"><span data-stu-id="56617-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="56617-108">Por ejemplo, el trabajo de reabastecimiento se puede crear si la cantidad necesaria para un pedido de ventas no está disponible cuando se procesa una oleada.</span><span class="sxs-lookup"><span data-stu-id="56617-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="56617-109">**Reabastecimiento mínimo/máximo**: esta estrategia usa los límites de existencias mínimos y máximos para determinar cuándo se deben reabastecer las ubicaciones.</span><span class="sxs-lookup"><span data-stu-id="56617-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="56617-110">Los criterios de artículo y ubicación definen el inventario que se evalúa para el reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="56617-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="56617-111">Las plantillas de reabastecimiento mínimo/máximo son el mecanismo principal para mantener niveles óptimos en ubicaciones de picking.</span><span class="sxs-lookup"><span data-stu-id="56617-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="56617-112">Con el fin de garantizar que hay suficiente inventario de frente de recogida disponible para satisfacer la demanda de oleada, puede utilizar el reabastecimiento de demanda como suplemento entre los ciclos de reabastecimiento mínimo o máximo.</span><span class="sxs-lookup"><span data-stu-id="56617-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="56617-113">**Reabastecimiento de demanda de carga**: esta estrategia suma la demanda para varias cargas y crea el trabajo de reabastecimiento que es necesario para almacenar las ubicaciones de picking pertinentes.</span><span class="sxs-lookup"><span data-stu-id="56617-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="56617-114">Esta estrategia ayuda a garantizar que las cargas que se crean se pueden seleccionar en el almacén tras su emisión.</span><span class="sxs-lookup"><span data-stu-id="56617-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="56617-115">**Reabastecimiento inmediato**: esta estrategia reaprovisiona el inventario antes de que ejecutar una oleada si falla la asignación de una línea de directiva de ubicación que tenga una plantilla de reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="56617-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="56617-116">Las cuatro estrategias crean trabajo de reabastecimiento, en función de una plantilla de reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="56617-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="56617-117">Reabastecimiento de la demanda de oleada</span><span class="sxs-lookup"><span data-stu-id="56617-117">Wave demand replenishment</span></span>
<span data-ttu-id="56617-118">El reabastecimiento de demanda de oleada crea el trabajo de reabastecimiento, basado en demanda, si la cantidad necesaria para los pedidos de producción, kanbans, pedidos o cargas de salida no está disponible cuando la oleada crea el trabajo.</span><span class="sxs-lookup"><span data-stu-id="56617-118">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders, or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="56617-119">La plantilla de reabastecimiento contiene información sobre los criterios del artículo, la unidad de medida, el incremento de la demanda y la ubicación.</span><span class="sxs-lookup"><span data-stu-id="56617-119">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="56617-120">Las directivas de ubicación se utilizan para determinar qué ubicación se debe reabastecer.</span><span class="sxs-lookup"><span data-stu-id="56617-120">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="56617-121">Vincula estas directivas de ubicación a la plantilla de reabastecimiento mediante el campo **Código de directiva**.</span><span class="sxs-lookup"><span data-stu-id="56617-121">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="56617-122">Si el campo **Código de directiva** no está establecido, las consultas se utilizan para determinar qué directiva de ubicación se debe utilizar.</span><span class="sxs-lookup"><span data-stu-id="56617-122">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="56617-123">Tenga en cuenta que si no se especifica un código de directiva en la plantilla de reabastecimiento y la directiva de ubicación tiene un código de directiva, se omitirá la directiva de ubicación, incluso si la consulta sobre la directiva de ubicación es correcta.</span><span class="sxs-lookup"><span data-stu-id="56617-123">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="56617-124">Las directivas de ubicación de picking se utilizan para determinar dónde obtener inventario para el reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="56617-124">Pick location directives are used to determine where to get inventory for the replenishment.</span></span>

<span data-ttu-id="56617-133">**Unidad de reabastecimiento** es la unidad mínima para reponer.</span><span class="sxs-lookup"><span data-stu-id="56617-133">**Replenishment unit** is the minimum unit to replenish.</span></span> <span data-ttu-id="56617-134">Este debe ser un número entero que sea múltiplo de la unidad.</span><span class="sxs-lookup"><span data-stu-id="56617-134">This must be a whole number that is a multiple of the unit.</span></span> <span data-ttu-id="56617-135">El sistema se redondeará a la unidad más alta posible al crear trabajo.</span><span class="sxs-lookup"><span data-stu-id="56617-135">The system will round up to the highest unit possible when creating work.</span></span>

<span data-ttu-id="56617-136">El reabastecimiento de la demanda se admite para los pedidos de ventas, pedidos de transferencia, pedidos de producción y kanbans.</span><span class="sxs-lookup"><span data-stu-id="56617-136">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="56617-137">Reabastecimiento mínimo/máximo</span><span class="sxs-lookup"><span data-stu-id="56617-137">Min/Max replenishment</span></span>
<span data-ttu-id="56617-138">En el reabastecimiento mínimo/máximo, las existencias se reabastecen para que se encuentren entre los límites mínimo y máximo que se han establecido.</span><span class="sxs-lookup"><span data-stu-id="56617-138">In Min/Max replenishment, stock is replenished so that it's between the minimum and maximum limits that have been set.</span></span> <span data-ttu-id="56617-139">Normalmente, este proceso se produce una vez al día para ayudar a garantizar que todas las ubicaciones de selección se rellenan hasta el nivel máximo antes de que se inicie la selección.</span><span class="sxs-lookup"><span data-stu-id="56617-139">Typically, this process occurs one time every day, to help guarantee that all picking locations are filled to the maximum level before picking starts.</span></span>
