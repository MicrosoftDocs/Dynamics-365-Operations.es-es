<span data-ttu-id="7a7d8-104">Dado que Android es un proyecto de origen abierto, cualquier proveedor de hardware para escáneres de códigos de barras de almacén puede crear un dispositivo para ejecutar el sistema operativo Android.</span><span class="sxs-lookup"><span data-stu-id="7a7d8-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="7a7d8-105">Un dispositivo sólo es compatible con Android si puede ejecutar aplicaciones escritas para el entorno de ejecución de Android.</span><span class="sxs-lookup"><span data-stu-id="7a7d8-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="7a7d8-106">Sin embargo, un proveedor de hardware puede modificar y crear superposiciones para la versión de Android que se ejecuta en su hardware.</span><span class="sxs-lookup"><span data-stu-id="7a7d8-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="7a7d8-107">Microsoft no puede asumir responsabilidad alguna para garantizar que una aplicación móvil de escaneado de códigos de barras para Android es compatible con el hardware de escaneado de códigos de barras de un proveedor y la versión de Android en la que se ejecuta.</span><span class="sxs-lookup"><span data-stu-id="7a7d8-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="7a7d8-110">Como cliente, se recomienda que pruebe la aplicación móvil de escaneado de almacén en el hardware seleccionado antes de decidir acerca del hardware que desea comprar.</span><span class="sxs-lookup"><span data-stu-id="7a7d8-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
