<span data-ttu-id="a9528-104">La selección de clúster es un proceso de selección de piezas que le permite elegir artículos para múltiples pedidos al mismo tiempo agrupándolos en clústeres de selección.</span><span class="sxs-lookup"><span data-stu-id="a9528-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="a9528-105">Después tendrá que visitar el lugar de selección una sola vez.</span><span class="sxs-lookup"><span data-stu-id="a9528-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="a9528-106">Por lo general, esta funcionalidad se usa para seleccionar pedidos pequeños o cantidades que son menores que la de cajas.</span><span class="sxs-lookup"><span data-stu-id="a9528-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>
