[!include [banner](../includes/banner.md)]

<span data-ttu-id="19e3d-104">La selección de clúster es un proceso de selección de piezas que le permite elegir artículos para múltiples pedidos al mismo tiempo agrupándolos en clústeres de selección.</span><span class="sxs-lookup"><span data-stu-id="19e3d-104">Cluster picking is a piece picking process that lets you pick items for multiple orders at the same time by clustering them into pick clusters.</span></span> <span data-ttu-id="19e3d-105">Después tendrá que visitar el lugar de selección una sola vez.</span><span class="sxs-lookup"><span data-stu-id="19e3d-105">You then have to visit the pick location only one time.</span></span> <span data-ttu-id="19e3d-106">Por lo general, esta funcionalidad se usa para seleccionar pedidos pequeños o cantidades que son menores que la de cajas.</span><span class="sxs-lookup"><span data-stu-id="19e3d-106">Typically, this functionality is used with small order picking or quantities that are less than case quantities.</span></span>
