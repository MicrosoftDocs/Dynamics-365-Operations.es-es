# <a name="iot-intelligence-now-replaced-by-sensor-data-intelligence"></a>Inteligencia de IoT reemplazada ahora por Sensor Data Intelligence

[!include [banner](../../includes/banner.md)]

La característica de Inteligencia de IoT se ha actualizado a la versión 2.0 y, al mismo tiempo, se le ha cambiado el nombre a *Sensor Data Intelligence*. La nueva versión está actualmente disponible como característica en vista previa para Supply Chain Management, versión 10.0.30 o posterior.

Sensor Data Intelligence presenta un nuevo modelo de implementación en el que los componentes necesarios de Azure se implementan en su propia suscripción de Azure en lugar de que Microsoft los administre. Este nuevo modelo mejora su capacidad para personalizar la característica e integrarla con sistemas de terceros. La nueva versión también incluye soporte para dos nuevos escenarios de Internet de las cosas (IoT) para la gestión de activos.

Debido a estos próximos cambios, Microsoft recomienda no iniciar ningún proyecto nuevo basado en la característica Inteligencia de IoT existente. Si está interesado en agregar capacidades de IoT a su sistema, le recomendamos que comience trabajando con la versión preliminar de Sensor Data Intelligence. Para obtener más información sobre Sensor Data Intelligence y cómo trabajar con esta característica, consulte la [página principal de Sensor Data Intelligence](../sensor-data-intelligence/sdi-home-page.md).
