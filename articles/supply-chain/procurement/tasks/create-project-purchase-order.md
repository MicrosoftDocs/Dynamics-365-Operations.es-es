[!include [banner](../../includes/banner.md)]

<span data-ttu-id="aa075-104">Este procedimiento muestra cómo crear un pedido de compra del proyecto.</span><span class="sxs-lookup"><span data-stu-id="aa075-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="aa075-105">Esta tarea usa el conjunto de datos USSI.</span><span class="sxs-lookup"><span data-stu-id="aa075-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="aa075-106">Vaya a Gestión de proyectos y contabilidad > Proyectos > Todos los proyectos.</span><span class="sxs-lookup"><span data-stu-id="aa075-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="aa075-107">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="aa075-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="aa075-108">En el panel de acciones, haga clic en Gestionar.</span><span class="sxs-lookup"><span data-stu-id="aa075-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="aa075-109">Haga clic en Tarea de artículo.</span><span class="sxs-lookup"><span data-stu-id="aa075-109">Click Item task.</span></span>
5. <span data-ttu-id="aa075-110">Haga clic en Pedido de compra.</span><span class="sxs-lookup"><span data-stu-id="aa075-110">Click Purchase order.</span></span>
6. <span data-ttu-id="aa075-111">En el campo Cuenta de proveedor, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="aa075-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="aa075-112">En el campo Sitio, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="aa075-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="aa075-113">Estos pasos no son necesarios, pero simplifican el pedido de compra configurando un sitio predeterminado y un almacén para las líneas de pedido de compra.</span><span class="sxs-lookup"><span data-stu-id="aa075-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="aa075-114">En el campo Almacén, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="aa075-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="aa075-115">Haga clic en Aceptar</span><span class="sxs-lookup"><span data-stu-id="aa075-115">Click OK.</span></span>
10. <span data-ttu-id="aa075-116">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="aa075-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="aa075-117">En el campo Número de artículo, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="aa075-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="aa075-118">Esto puede ser el número de artículo o una categoría de compras.</span><span class="sxs-lookup"><span data-stu-id="aa075-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="aa075-119">Expanda la sección Detalles de línea.</span><span class="sxs-lookup"><span data-stu-id="aa075-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="aa075-120">Haga clic en la pestaña Projecto.</span><span class="sxs-lookup"><span data-stu-id="aa075-120">Click the Project tab.</span></span>
    * <span data-ttu-id="aa075-121">Compruebe que las ventas y los precios de coste están disponibles.</span><span class="sxs-lookup"><span data-stu-id="aa075-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="aa075-122">Si no están disponibles pero son necesarias, especifique la información.</span><span class="sxs-lookup"><span data-stu-id="aa075-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="aa075-123">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="aa075-123">Click Save.</span></span>
