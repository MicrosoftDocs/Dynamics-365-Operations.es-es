## <a name="solver-strategy"></a><span data-ttu-id="b9a08-112">Estrategia de solucionador</span><span class="sxs-lookup"><span data-stu-id="b9a08-112">Solver strategy</span></span> 

<span data-ttu-id="b9a08-113">Se puede formular un modelo de configuración de productos como un [problema de satisfacción de restricción (CSP)](http://aima.cs.berkeley.edu/2nd-ed/newchap05.pdf)</span><span class="sxs-lookup"><span data-stu-id="b9a08-113">A product configuration model can be formulated as a [constraint satisfaction problem (CSP)](http://aima.cs.berkeley.edu/2nd-ed/newchap05.pdf).</span></span> <span data-ttu-id="b9a08-114">Microsoft Solver Foundation (MSF) ofrece dos tipos de estrategias de solucionador para resolver los CSP que se pueden utilizar de los modelos de configuración de productos.</span><span class="sxs-lookup"><span data-stu-id="b9a08-114">Microsoft Solver Foundation (MSF) provides two types of solver strategies to solve the CSPs that can be used from product configuration models.</span></span> <span data-ttu-id="b9a08-115">Estas estrategias de solucionador confían en la [heurística](https://techterms.com/definition/heuristic), que se emplea para determinar el orden en que se consideran las variables de los CSP cuando se está solucionando el problema.</span><span class="sxs-lookup"><span data-stu-id="b9a08-115">These solver strategies rely on [heuristics](https://techterms.com/definition/heuristic), which are used to determine the order that the variables of the CSPs are considered in when the problem is being solved.</span></span> <span data-ttu-id="b9a08-116">La heurística puede afectar considerablemente al rendimiento cuando se está solucionando un problema o una clase de problemas.</span><span class="sxs-lookup"><span data-stu-id="b9a08-116">Heuristics can significantly affect performance when a problem or class of problems is being solved.</span></span>

<span data-ttu-id="b9a08-117">En Finance and Operations, la estrategia de solucionador para modelos de configuración de productos determina qué solucionador se emplea con la heurística.</span><span class="sxs-lookup"><span data-stu-id="b9a08-117">In Finance and Operations, the solver strategy for product configuration models determines which solver is used with heuristics.</span></span> <span data-ttu-id="b9a08-118">Las estrategias **Predeterminado**, **Dominios mínimos primero** y **De arriba a abajo** utilizan dos solucionadores de MSF, mientras que la estrategia **Z3** emplea el solucionador Z3.</span><span class="sxs-lookup"><span data-stu-id="b9a08-118">The **Default**, **Minimal domains first**, and **Top-down** strategies use the two solvers from MSF, whereas the **Z3** strategy uses the Z3 solver.</span></span> 

<span data-ttu-id="b9a08-119">Los estudios reales de implementación de clientes han mostrado que un cambio en la estrategia de solucionador de un modelo de configuración de productos puede reducir el tiempo de respuesta de minutos a milisegundos.</span><span class="sxs-lookup"><span data-stu-id="b9a08-119">Real customer implementation studies have shown that a change in the solver strategy for a product configuration model can reduce the response time from minutes to milliseconds.</span></span> <span data-ttu-id="b9a08-120">Por lo tanto, vale la pena intentar diferentes estrategias de solucionador para encontrar la estrategia más eficiente para su modelo de configuración de productos.</span><span class="sxs-lookup"><span data-stu-id="b9a08-120">Therefore, it's worth the effort to try different solver strategies to find the most efficient strategy for your product configuration model.</span></span>
