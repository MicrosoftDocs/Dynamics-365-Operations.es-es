# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="20d74-103">Crear un código de barras para un producto</span><span class="sxs-lookup"><span data-stu-id="20d74-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="20d74-104">En este tema se muestra cómo crear manualmente un código de barras mediante el número de artículo M0001 como ejemplo.</span><span class="sxs-lookup"><span data-stu-id="20d74-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="20d74-105">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="20d74-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="20d74-106">Seleccione **Mantenimiento de producto emitido** en la página principal.</span><span class="sxs-lookup"><span data-stu-id="20d74-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="20d74-107">Vaya a **Productos > Productos emitidos** en la sección **Vínculos**.</span><span class="sxs-lookup"><span data-stu-id="20d74-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="20d74-108">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="20d74-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="20d74-109">Para este ejemplo, seleccione el código de artículo **M0001**.</span><span class="sxs-lookup"><span data-stu-id="20d74-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="20d74-110">En el panel de acciones, seleccione **Administrar inventario**.</span><span class="sxs-lookup"><span data-stu-id="20d74-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="20d74-111">Seleccione **Códigos de barras**.</span><span class="sxs-lookup"><span data-stu-id="20d74-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="20d74-112">Seleccione **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="20d74-112">Select **New**.</span></span>
7. <span data-ttu-id="20d74-113">Marque la fila seleccionada que se crea en la lista mostrada a continuación.</span><span class="sxs-lookup"><span data-stu-id="20d74-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="20d74-114">En el campo **Configuración de código de barras**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="20d74-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="20d74-115">En el campo **Código de barras**, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="20d74-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="20d74-116">En el campo **Código de barras**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="20d74-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="20d74-117">En el campo **Cantidad**, especifique un número.</span><span class="sxs-lookup"><span data-stu-id="20d74-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="20d74-118">Seleccione **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="20d74-118">Select **Save**.</span></span>
13. <span data-ttu-id="20d74-119">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="20d74-119">Close the page.</span></span>
