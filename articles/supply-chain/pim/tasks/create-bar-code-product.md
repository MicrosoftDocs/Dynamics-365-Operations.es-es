# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="8850c-103">Crear un código de barras para un producto</span><span class="sxs-lookup"><span data-stu-id="8850c-103">Create a bar code for a product</span></span>

<span data-ttu-id="8850c-104">Este procedimiento muestra cómo crear manualmente un código de barras mediante el número de artículo M0001 como ejemplo.</span><span class="sxs-lookup"><span data-stu-id="8850c-104">This procedure shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="8850c-105">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="8850c-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="8850c-106">Haga clic en Mantenimiento de producto emitido.</span><span class="sxs-lookup"><span data-stu-id="8850c-106">Click Released product maintenance.</span></span>
2. <span data-ttu-id="8850c-107">Haga clic en Productos emitidos.</span><span class="sxs-lookup"><span data-stu-id="8850c-107">Click Released products.</span></span>
3. <span data-ttu-id="8850c-108">En la lista, busque y seleccione el registro deseado.</span><span class="sxs-lookup"><span data-stu-id="8850c-108">In the list, find and select the desired record.</span></span>

5. <span data-ttu-id="8850c-110">Haga clic en Códigos de barras.</span><span class="sxs-lookup"><span data-stu-id="8850c-110">Click Bar codes.</span></span>
6. <span data-ttu-id="8850c-111">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="8850c-111">Click New.</span></span>
7. <span data-ttu-id="8850c-112">En la lista, marque la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="8850c-112">In the list, mark the selected row.</span></span>

9. <span data-ttu-id="8850c-114">En el campo Código de barras, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="8850c-114">In the Bar code field, enter or select a value.</span></span>
10. <span data-ttu-id="8850c-115">En el campo Código de barras, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="8850c-115">In the Bar code field, type a value.</span></span>
    * <span data-ttu-id="8850c-116">Presione el tabulador.</span><span class="sxs-lookup"><span data-stu-id="8850c-116">Press the Tab key.</span></span>  
11. <span data-ttu-id="8850c-117">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="8850c-117">Close the page.</span></span>
12. <span data-ttu-id="8850c-118">En el campo Cantidad, especifique un número.</span><span class="sxs-lookup"><span data-stu-id="8850c-118">In the Quantity field, enter a number.</span></span>
13. <span data-ttu-id="8850c-119">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="8850c-119">Click Save.</span></span>
    * <span data-ttu-id="8850c-120">Al hacer clic en Guardar, se ejecuta la comprobación del código de barras y, en este caso, mostrará un error que indica que el dígito de control previsto es 8, pero que se ha encontrado el 3.</span><span class="sxs-lookup"><span data-stu-id="8850c-120">When you click Save, the barcode check is run, and in this case it will display an error stating that the expected check digit is 8, but that 3 was found.</span></span> <span data-ttu-id="8850c-121">Actualice manualmente el número de código de barras de modo que el 8 esté al final.</span><span class="sxs-lookup"><span data-stu-id="8850c-121">Manually update the barcode number so that 8 is at the end.</span></span>  
14. <span data-ttu-id="8850c-122">En el campo Código de barras, especifique o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="8850c-122">In the Bar code field, enter or select a value.</span></span>
15. <span data-ttu-id="8850c-123">En el campo Código de barras, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="8850c-123">In the Bar code field, type a value.</span></span>
    * <span data-ttu-id="8850c-124">Presione el tabulador.</span><span class="sxs-lookup"><span data-stu-id="8850c-124">Press the Tab key.</span></span>  
16. <span data-ttu-id="8850c-125">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="8850c-125">Close the page.</span></span>
17. <span data-ttu-id="8850c-126">Haga clic en Guardar.</span><span class="sxs-lookup"><span data-stu-id="8850c-126">Click Save.</span></span>
18. <span data-ttu-id="8850c-127">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="8850c-127">Close the page.</span></span>
