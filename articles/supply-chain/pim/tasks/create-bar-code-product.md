# <a name="create-a-bar-code-for-a-product"></a>Crear un código de barras para un producto

[!include [banner](../../includes/banner.md)]

En este tema se muestra cómo crear manualmente un código de barras mediante el número de artículo M0001 como ejemplo. La empresa de datos de prueba utilizada para crear este procedimiento es USMF.

1. Seleccione **Mantenimiento de producto emitido** en la página principal.
2. Vaya a **Productos > Productos emitidos** en la sección **Vínculos**.
3. En la lista, busque y seleccione el registro deseado. Para este ejemplo, seleccione el código de artículo **M0001**.
4. En el panel de acciones, seleccione **Administrar inventario**.
5. Seleccione **Códigos de barras**.
6. Seleccione **Nuevo**.
7. Marque la fila seleccionada que se crea en la lista mostrada a continuación.
8. En el campo **Configuración de código de barras**, especifique o seleccione un valor.
9. En el campo **Código de barras**, especifique o seleccione un valor.
10. En el campo **Código de barras**, escriba un valor.  
11. En el campo **Cantidad**, especifique un número.
12. Seleccione **Guardar**.
13. Cierre la página.
