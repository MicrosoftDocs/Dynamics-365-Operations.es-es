# <a name="create-a-bar-code-for-a-product"></a>Crear un código de barras para un producto

[!include [banner](../../includes/banner.md)]

En este tema se muestra cómo crear manualmente un código de barras mediante el número de artículo M0001 como ejemplo. La empresa de datos de prueba utilizada para crear este procedimiento es USMF.

1. Vaya a **Gestión de información de productos \> Productos \> Productos despachados**.
1. En la lista, busque y seleccione el registro deseado. Para este ejemplo, seleccione el código de artículo **M0001**.
1. En el panel de acciones, seleccione **Administrar inventario**.
1. Seleccione **Códigos de barras**.
1. Seleccione **Nuevo**.
1. Marque la fila seleccionada que se crea en la lista mostrada a continuación.
1. En el campo **Configuración de código de barras**, especifique o seleccione un valor.
1. En el campo **Código de barras**, especifique o seleccione un valor.
1. En el campo **Código de barras**, escriba un valor.  
1. En el campo **Cantidad**, especifique un número.
1. Seleccione **Guardar**.
1. Cierre la página.
