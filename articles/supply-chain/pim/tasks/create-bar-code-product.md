[!include [banner](../../includes/banner.md)]

<span data-ttu-id="9bca2-104">En este tema se muestra cómo crear manualmente un código de barras mediante el número de artículo M0001 como ejemplo.</span><span class="sxs-lookup"><span data-stu-id="9bca2-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="9bca2-105">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="9bca2-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="9bca2-106">Seleccione **Mantenimiento de producto emitido** en la página principal.</span><span class="sxs-lookup"><span data-stu-id="9bca2-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="9bca2-107">Vaya a **Productos > Productos emitidos** en la sección **Vínculos**.</span><span class="sxs-lookup"><span data-stu-id="9bca2-107">Go to **Products > Released products** under the **Links** section.</span></span>

4. <span data-ttu-id="9bca2-110">En el panel de acciones, seleccione **Administrar inventario**.</span><span class="sxs-lookup"><span data-stu-id="9bca2-110">On the Action Pane, select **Manage inventory**.</span></span>

13. <span data-ttu-id="9bca2-119">Cierre la página.</span><span class="sxs-lookup"><span data-stu-id="9bca2-119">Close the page.</span></span>
