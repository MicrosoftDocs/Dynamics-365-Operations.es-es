# <a name="define-configuration-groups"></a>Definir grupos de configuración

[!include [banner](../../includes/banner.md)]

Este procedimiento muestra cómo definir grupos de configuración de productos creando dos grupos en la empresa de demostración USMF. Este es el cuarto procedimiento de ocho que explica cómo crear combinaciones para la configuración basada en dimensiones.

1. Vaya a Gestión de información de productos > Configuración > Listas de materiales y fórmulas > Grupos de configuración.
2. Haga clic en Nuevo.
3. En el campo Grupo de configuración, escriba un valor.
    * Escriba un nombre para el grupo de configuración que se usará para las líneas de la lista de materiales.  
4. En el campo Nombre, escriba un valor.
5. Haga clic en Nuevo.
6. En el campo Grupo de configuración, escriba un valor.
    * Escriba un nombre para el grupo de configuración que se usará para las líneas de la lista de materiales.  
7. En el campo Nombre, escriba un valor.
8. Cierre la página.
