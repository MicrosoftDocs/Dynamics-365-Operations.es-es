# <a name="transportation-management-discounts"></a><span data-ttu-id="bad47-103">Descuentos de la administración de transporte</span><span class="sxs-lookup"><span data-stu-id="bad47-103">Transportation management discounts</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="bad47-104">La gestión de transporte le permite configurar descuentos que se aplicarán además del precio de transporte calculado por el motor de clasificación.</span><span class="sxs-lookup"><span data-stu-id="bad47-104">Transportation management lets you set up discounts to be applied on top of the transportation price calculated by the rating engine.</span></span> <span data-ttu-id="bad47-105">Puede especificar cada descuento como un porcentaje de los cargos o como una cantidad fija.</span><span class="sxs-lookup"><span data-stu-id="bad47-105">You can specify each discount either as a percentage of charges or as a fixed amount.</span></span>

<span data-ttu-id="bad47-106">Para configurar descuentos para la gestión del transporte, vaya a **Gestión de transporte \> Preparar \> Clasificación \> Descuentos**.</span><span class="sxs-lookup"><span data-stu-id="bad47-106">To set up discounts for transportation management, go to **Transportation management \> Setup \> Rating \> Discounts**.</span></span>

<span data-ttu-id="bad47-107">Puede elegir pasar el descuento a su cliente (configurando el tipo de descuento al cliente) o retenerlo en la empresa (configurando el tipo de descuento como remitente).</span><span class="sxs-lookup"><span data-stu-id="bad47-107">You can choose whether to pass the discount on to your customer (by setting the discount type to customer) or to retain it in the company (by setting the discount type to shipper).</span></span> <span data-ttu-id="bad47-108">Esto le permite definir una configuración en la que gana un margen sobre los gastos de transporte calculados.</span><span class="sxs-lookup"><span data-stu-id="bad47-108">This lets you define a setup in which you earn a margin on the calculated freight charges.</span></span>

<span data-ttu-id="bad47-109">Puede configurar varios calificadores para cada descuento, como transportista, servicio de transportista, condiciones de entrega, cliente, destino y más.</span><span class="sxs-lookup"><span data-stu-id="bad47-109">You can set up multiple qualifiers for each discount, such as carrier, carrier service, delivery terms, customer, destination, and more.</span></span>
