# <a name="order-promising"></a><span data-ttu-id="1da3a-104">Compromisos de pedidos</span><span class="sxs-lookup"><span data-stu-id="1da3a-104">Order promising</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1da3a-105">En este artículo se proporciona información acerca de los compromisos de entrega.</span><span class="sxs-lookup"><span data-stu-id="1da3a-105">This article provides information about order promising.</span></span> <span data-ttu-id="1da3a-106">Los compromisos de entrega le ayuda a comprometer fechas de entrega de manera confiable a sus clientes y le ofrecen flexibilidad para que pueda cumplir esas fechas.</span><span class="sxs-lookup"><span data-stu-id="1da3a-106">Order promising helps you reliably promise delivery dates to your customers and gives you flexibility so that you can meet those dates.</span></span>

<span data-ttu-id="1da3a-107">Los compromisos de pedidos calculan las fechas de envío y recepción más tempranas, y se basan en el método de control de fecha de entrega y los días de transporte.</span><span class="sxs-lookup"><span data-stu-id="1da3a-107">Order promising calculates the earliest ship and receipt dates, and is based on the delivery date control method and transport days.</span></span> <span data-ttu-id="1da3a-108">Puede seleccionar entre cuatro métodos de control de fecha de entrega:</span><span class="sxs-lookup"><span data-stu-id="1da3a-108">You can select among four delivery date control methods:</span></span>

-   <span data-ttu-id="1da3a-109">**Plazo de ventas**: el plazo de ventas es el tiempo entre la creación del pedido de ventas y el envío de los artículos.</span><span class="sxs-lookup"><span data-stu-id="1da3a-109">**Sales lead time** – Sales lead time is the time between creation of the sales order and shipment of the items.</span></span> <span data-ttu-id="1da3a-110">El cálculo de la fecha de entrega se basa en un número predeterminado de días y no considera la disponibilidad de existencias, la demanda conocida o el suministro planificado.</span><span class="sxs-lookup"><span data-stu-id="1da3a-110">The delivery date calculation is based on a default number of days, and does not consider stock availability, known demand, or planned supply.</span></span>
-   <span data-ttu-id="1da3a-111">**NNC (neto no comprometido)**: el NNC es la cantidad de un artículo que esté disponible y se pueda prometer a un cliente en una fecha específica.</span><span class="sxs-lookup"><span data-stu-id="1da3a-111">**ATP (available-to-promise)** – ATP is the quantity of an item that is available and can be promised to a customer on a specific date.</span></span> <span data-ttu-id="1da3a-112">El cálculo del NNC incluye inventario no comprometido, fechas disponibles, recepciones planificadas y emisiones.</span><span class="sxs-lookup"><span data-stu-id="1da3a-112">The ATP calculation includes uncommitted inventory, lead times, planned receipts, and issues.</span></span>
-   <span data-ttu-id="1da3a-113">**NNC + Días de emisión**: la fecha de envío será igual que la fecha de neto no comprometido (NNC) más el margen de emisión del artículo.</span><span class="sxs-lookup"><span data-stu-id="1da3a-113">**ATP + Issue margin** – The shipping date is equal to the ATP date plus the issue margin for the item.</span></span> <span data-ttu-id="1da3a-114">El margen de emisión es el tiempo que necesario para preparar los artículos para el envío.</span><span class="sxs-lookup"><span data-stu-id="1da3a-114">The issue margin is the time that is required to prepare the items for shipment.</span></span>
-   <span data-ttu-id="1da3a-115">**CTP (capaz de comprometer)**: la disponibilidad se calcula a través de la expansión.</span><span class="sxs-lookup"><span data-stu-id="1da3a-115">**CTP (capable-to-promise)** – Availability is calculated through explosion.</span></span>

## <a name="atp-calculations"></a><span data-ttu-id="1da3a-116">Cálculos de NNC</span><span class="sxs-lookup"><span data-stu-id="1da3a-116">ATP calculations</span></span>
<span data-ttu-id="1da3a-117">La cantidad de NNC se calcula mediante el método “NNC acumulativo con previsión futura”.</span><span class="sxs-lookup"><span data-stu-id="1da3a-117">The ATP quantity is calculated by using the “cumulative ATP with look-ahead” method.</span></span> <span data-ttu-id="1da3a-118">La principal ventaja de este cálculo de NNC es que permite controlar las ocasiones en las que la suma de las emisiones entre recepciones supera la última recepción; por ejemplo, cuando es necesario usar una cantidad de una recepción anterior para satisfacer un requisito.</span><span class="sxs-lookup"><span data-stu-id="1da3a-118">The main advantage of this ATP calculation method is that it can handle cases where the sum of issues among receipts is more than the latest receipt (for example, when a quantity from an earlier receipt must be used to meet a requirement).</span></span> <span data-ttu-id="1da3a-119">El método de cálculo “NNC acumulativo con previsión futura” incluye todas las emisiones hasta que la cantidad acumulativa que se recibe excede la cantidad acumulada que se emite.</span><span class="sxs-lookup"><span data-stu-id="1da3a-119">The “cumulative ATP with look-ahead” calculation method includes all issues until the cumulative quantity to receive exceeds the cumulative quantity to issue.</span></span> <span data-ttu-id="1da3a-120">Por lo tanto, este método de cálculo de NNC evalúa si parte de la cantidad de un período anterior se puede utilizar en un período posterior.</span><span class="sxs-lookup"><span data-stu-id="1da3a-120">Therefore, this ATP calculation method evaluates whether some of the quantity from an earlier period can be used in a later period.</span></span>  

<span data-ttu-id="1da3a-121">La cantidad de NNC es el saldo no comprometido de inventario en el primer período.</span><span class="sxs-lookup"><span data-stu-id="1da3a-121">The ATP quantity is the uncommitted inventory balance in the first period.</span></span> <span data-ttu-id="1da3a-122">Normalmente, se calcula para cada período en el que se programa una recepción.</span><span class="sxs-lookup"><span data-stu-id="1da3a-122">Typically, it's calculated for each period in which a receipt is scheduled.</span></span> <span data-ttu-id="1da3a-123">El programa calcula el período de NNC en días y la fecha actual como la primera fecha para la cantidad de NNC.</span><span class="sxs-lookup"><span data-stu-id="1da3a-123">The program calculates the ATP period in days and calculates the current date as the first date for the ATP quantity.</span></span> <span data-ttu-id="1da3a-124">En el primer período, NNC incluye el inventario disponible menos los pedidos de cliente vencidos.</span><span class="sxs-lookup"><span data-stu-id="1da3a-124">In the first period, ATP includes on-hand inventory minus customer orders that are due and overdue.</span></span>  

<span data-ttu-id="1da3a-125">La ATP se calcula mediante la fórmula siguiente:</span><span class="sxs-lookup"><span data-stu-id="1da3a-125">ATP is calculated by using the following formula:</span></span>  

<span data-ttu-id="1da3a-126">ATP = ATP para el período anterior + Recepciones del período actual - Emisiones del período actual - Cantidad neta de emisión para los períodos futuros hasta el período en el que la suma de las recepciones de todos los períodos futuros (hasta el período futuro, incluido éste) sea superior a la suma de las emisiones (hasta el período futuro, incluido este).</span><span class="sxs-lookup"><span data-stu-id="1da3a-126">ATP = ATP for the previous period + Receipts for the current period – Issues for the current period – Net issue quantity for each future period until the period when the sum of receipts for all future periods, up to and including the future period, exceeds the sum of issues up to and including the future period.</span></span>  

<span data-ttu-id="1da3a-127">Cuando ya no queden emisiones o recepciones para tener en cuenta, la cantidad ATP de las fechas siguientes será la misma que la última cantidad ATP calculada.</span><span class="sxs-lookup"><span data-stu-id="1da3a-127">When there are no more issues or receipts to consider, the ATP quantity for the following dates is the same as the latest calculated ATP quantity.</span></span>  

<span data-ttu-id="1da3a-128">Si no se proporcionan todas las dimensiones que se usan para un artículo a la hora de realizar la comprobación NNC, es posible que aún se especifiquen en la emisión y las recepciones.</span><span class="sxs-lookup"><span data-stu-id="1da3a-128">If not all the dimensions that are used for an item are given when the ATP check is completed, they can still be specified on the issue and receipts.</span></span> <span data-ttu-id="1da3a-129">En este caso, el cálculo de NNC, las recepciones y las emisiones deberán agregarse a las dimensiones existentes con el fin de reducir el número de líneas de recepción y emisión que se usan en el cálculo de NNC.</span><span class="sxs-lookup"><span data-stu-id="1da3a-129">In this case, in the ATP calculation, the receipts and issues must be aggregated to the existing dimensions to reduce the number of receipt and issue lines that are used in the ATP calculation.</span></span>  

<span data-ttu-id="1da3a-130">La cantidad de NNC que se muestra siempre es mayor o igual que 0 (cero).</span><span class="sxs-lookup"><span data-stu-id="1da3a-130">The ATP quantity that is shown is always greater than or equal to 0 (zero).</span></span> <span data-ttu-id="1da3a-131">Si el cálculo devuelve una cantidad negativa de NNC (por ejemplo, si la cantidad que se prometió anteriormente excede la cantidad disponible), el programa establece automáticamente la cantidad en **0**.</span><span class="sxs-lookup"><span data-stu-id="1da3a-131">If the calculation returns a negative ATP quantity (for example, if the quantity that was previously promised exceeds the available quantity), the program automatically sets the quantity to **0**.</span></span>

<span data-ttu-id="1da3a-140">Un pedido de compra de 200 piezas de un producto que se debería haber recibido hace tres días todavía no se recibió.</span><span class="sxs-lookup"><span data-stu-id="1da3a-140">A purchase order for 200 pieces of a product that should have been received three days ago hasn't been received yet.</span></span> <span data-ttu-id="1da3a-141">Por consiguiente, una línea de pedido de ventas de 75 piezas del mismo producto que se debería haber enviado ayer no se ha enviado.</span><span class="sxs-lookup"><span data-stu-id="1da3a-141">Therefore, a sales order line for 75 pieces of the same product that should have been shipped yesterday hasn't been shipped.</span></span>
