# <a name="turn-on-sensor-data-intelligence-for-your-system"></a>Active Sensor Data Intelligence para su sistema

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]

Antes de poder usar el Sensor Data Intelligence, debe estar activado para su sistema.

1. Vaya a **Administración del sistema \> Espacios de trabajo \> Administración de características**. (Para más información, consulte [Resumen de administración de características](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).)
1. En la pestaña **Todos**, utilice el campo **Filtro** para buscar la característica que se llama *Sensor Data Intelligence*.
1. Si la característica *Sensor Data Intelligence* está habilitada en su sistema, selecciónela en la lista y luego seleccione **Deshabilitar** para deshabilitarla. No puede usar esta versión anterior de la función junto con la nueva versión de vista previa que habilitará en el siguiente paso.
1. Use el campo **Filtro** para buscar la característica que se llama *(Vista previa) Sensor Data Intelligence*.
1. Seleccione la característica en la lista y luego seleccione **Habilitar ahora** para habilitarla.
