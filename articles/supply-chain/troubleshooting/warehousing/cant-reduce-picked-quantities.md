# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>No puede mover el inventario a una ubicación que esté controlada por matrículas.

## <a name="symptoms"></a>Síntomas

No puede reducir las cantidades recogidas en una carga.

## <a name="resolution"></a>Resolución

En versiones anteriores, no podía reducir las cantidades recogidas en una carga. Sin embargo, ahora puede anular la selección a una ubicación controlada por matrícula. Debe especificar tanto un valor de **Ubicación** como un valor de **Matrícula** para la línea de carga en la página **Reducir la cantidad recogida**.
