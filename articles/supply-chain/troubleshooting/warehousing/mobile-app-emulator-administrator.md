# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Solo puede usar el rol de administrador para obtener acceso al emulador de la aplicación del dispositivo móvil

## <a name="symptoms"></a>Síntomas

Al configurar almacenes en Supply Chain Management, no puede usar ningún rol excepto el rol de administrador para acceder al emulador de la aplicación de dispositivo móvil.

## <a name="resolution"></a>Resolución

El emulador de aplicaciones para dispositivos móviles está configurado para funcionar solo con la cuenta de administrador. Para todos los propósitos de pruebas y procesos en vivo, le recomendamos que utilice la propia aplicación móvil Warehouse Management.
