# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>No se puede realizar un seguimiento de los números de serie de los bienes terminados con los de los bienes consumidos

## <a name="symptoms"></a>Síntomas

No puede rastrear los números de serie de los productos terminados con los números de serie de los materiales que consume una orden de producción para fabricar esos productos terminados.

## <a name="workaround"></a>Solución alternativa

Para una solución, cree órdenes de producción por una cantidad de 1.
