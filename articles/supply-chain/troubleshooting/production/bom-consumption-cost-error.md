# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>No se puede finalizar un pedido de producción debido a un error de valor de consumptionCost de L. MAT.

## <a name="symptoms"></a>Síntomas

Cuando intenta finalizar un pedido de producción, recibe el siguiente mensaje de error:

> Cálculo del consumo de la lista de materiales El valor del costo debe ser negativo en el momento de la emisión del inventario.

## <a name="resolution"></a>Resolución

Este problema se solucionó en la versión 10.0.15.
