# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Los usuarios pueden deshacer las líneas de material de los productos que se han notificado como terminados

Número de KB: 4614721

## <a name="symptoms"></a>Síntomas

El sistema permite a los usuarios deshacer las líneas de materias primas de los productos que se han notificado como terminados.

## <a name="resolution"></a>Resolución

El sistema se está comportando según lo diseñado. El consumo de materias primas se puede ajustar hasta que el pedido de producción alcance un estado de *Finalizado*.
