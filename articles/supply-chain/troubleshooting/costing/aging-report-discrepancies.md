# <a name="inventory-aging-report-discrepancies"></a>Discrepancias de informe de vencimiento de inventario

## <a name="symptoms"></a>Síntomas

El **Informe de antigüedad de inventario** muestra diferentes valores cuando se ve en diferentes dimensiones de almacenamiento (como sitio o almacén).

## <a name="resolution"></a>Resolución

Para obtener más información sobre la lógica de informes, consulte [Ejemplos y lógica de informes de antigüedad de inventario](../../cost-management/inventory-aging-report.md).
