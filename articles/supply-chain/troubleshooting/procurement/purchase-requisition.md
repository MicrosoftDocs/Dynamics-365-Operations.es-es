# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>No puede agregar una línea a una solicitud de compra después de solicitar un cambio

Número de KB: 4611211

## <a name="symptoms"></a>Síntomas

El sistema no le permite agregar una línea a una solicitud de compra después de solicitar un cambio.

## <a name="resolution"></a>Resolución

Debe recuperar el flujo de trabajo. Cuando la solicitud de compra esté en estado *Borrador*, puede continuar editándola o agregarle una línea.
