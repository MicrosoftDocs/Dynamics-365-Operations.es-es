# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>El informe de lista de selección no muestra las notas nuevas que se adjuntan a un diario

Número de KB: 4611660

## <a name="symptoms"></a>Síntomas

El informe de lista de selección no muestra las notas nuevas que se adjuntan a un diario de lista de selección.

## <a name="resolution"></a>Resolución

El informe de lista de selección muestra solo las notas que se adjuntan en la línea del pedido de ventas y el encabezado del pedido de ventas. No muestra ninguna nota nueva que se agregue al diario de facturas.
