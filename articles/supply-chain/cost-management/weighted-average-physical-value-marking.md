# <a name="weighted-average-with-physical-value-and-marking"></a><span data-ttu-id="d6e47-103">Media ponderada con valor físico y marcado</span><span class="sxs-lookup"><span data-stu-id="d6e47-103">Weighted average with physical value and marking</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="d6e47-104">La media ponderada es un modelo de inventario que se basa en el principio de media ponderada, donde las emisiones de inventario se tasan en el valor medio de los artículos recibidos en inventario durante el período de cierre de inventario, más cualquier inventario disponible del período anterior.</span><span class="sxs-lookup"><span data-stu-id="d6e47-104">Weighted average is an inventory model based on the weighted average principle, where issues from inventory are valued at the average value of the items that are received into inventory during the inventory closing period, plus any on-hand inventory from the previous period.</span></span>

<span data-ttu-id="d6e47-105">Cuando se ejecuta un cierre de inventario, todas las recepciones se liquidan contra una emisión virtual, que mantiene la cantidad recibida total y el valor.</span><span class="sxs-lookup"><span data-stu-id="d6e47-105">When you run an inventory closing, all receipts are settled against a virtual issue, which holds the total received quantity and value.</span></span> <span data-ttu-id="d6e47-106">Esta emisión virtual tiene una recepción virtual correspondiente a partir de la cual se liquidan las emisiones.</span><span class="sxs-lookup"><span data-stu-id="d6e47-106">This virtual issue has a corresponding virtual receipt from which the issues are settled.</span></span> <span data-ttu-id="d6e47-107">De este modo, todas las emisiones obtienen el mismo coste medio.</span><span class="sxs-lookup"><span data-stu-id="d6e47-107">In this manner, all issues get the same average cost.</span></span> <span data-ttu-id="d6e47-108">La emisión y recepción virtual puede verse como una transferencia virtual, llamada “transferencia de cierre de inventario de media ponderada”.</span><span class="sxs-lookup"><span data-stu-id="d6e47-108">The virtual issue and receipt can be seen as a virtual transfer, which is named the weighted average inventory closing transfer.</span></span>

<span data-ttu-id="d6e47-109">Si solo existe una recepción, todas las emisiones se pueden liquidar a partir de ella y no se creará la transferencia virtual.</span><span class="sxs-lookup"><span data-stu-id="d6e47-109">If there is only one receipt, all issues can be settled from it and the virtual transfer will not be created.</span></span> 

<span data-ttu-id="d6e47-110">Al usar la media ponderada, puede marcar las transacciones de inventario de modo que se liquide una recepción de artículo específica con una emisión específica, en lugar de usar la regla de media ponderada.</span><span class="sxs-lookup"><span data-stu-id="d6e47-110">When using weighted average, you can mark inventory transactions so that a specific item receipt is settled against a specific issue, instead of using the weighted average rule.</span></span> 

<span data-ttu-id="d6e47-111">Recomendamos un cierre de inventario mensual si se utiliza el modelo de inventario de media ponderada.</span><span class="sxs-lookup"><span data-stu-id="d6e47-111">We recommend a monthly inventory closing when you use the weighted average inventory model.</span></span>
