## <a name="avoiding-pricing-amplification"></a><span data-ttu-id="64aef-126">Evitar amplificación de precio</span><span class="sxs-lookup"><span data-stu-id="64aef-126">Avoiding pricing amplification</span></span>

3.  <span data-ttu-id="64aef-133">Financieramente recibe una cantidad de 101 a 202,00 USD.</span><span class="sxs-lookup"><span data-stu-id="64aef-133">You physically receive a quantity of 101 at USD 202.00.</span></span>

<span data-ttu-id="64aef-134">Cuando se examina el precio de coste promedio móvil estimado para el artículo, se espera un precio de coste de 1,51 dólares.</span><span class="sxs-lookup"><span data-stu-id="64aef-134">When you examine the estimated running average cost price for the item, you expect a cost price of USD 1.51.</span></span> <span data-ttu-id="64aef-135">En su lugar, encuentra un promedio móvil estimado de 102,00 USD, basado en la siguiente fórmula: Precio estimado = \[202 + (-100)\] ÷ \[101 + (-100)\] = 102 ÷ 1 = 102. Esta amplificación de precios se produce porque, cuando se emiten 200 artículos financieramente en el paso 2, el sistema debe fijar el precio de 100 de los artículos antes de tener los recibos correspondientes.</span><span class="sxs-lookup"><span data-stu-id="64aef-135">Instead, you find an estimated running average of USD 102.00, which is based on the following formula: Estimated price = \[202 + (-100)\] ÷ \[101 + (-100)\] = 102 ÷ 1 = 102 This pricing amplification occurs because, when 200 items are issued financially in step 2, the system must price 100 of the items before it has any corresponding receipts.</span></span> <span data-ttu-id="64aef-136">Esta situación genera inventario negativo.</span><span class="sxs-lookup"><span data-stu-id="64aef-136">This situation causes negative inventory.</span></span> <span data-ttu-id="64aef-137">El sistema estima a continuación un precio unitario de 1,00 USD, como se podría esperar.</span><span class="sxs-lookup"><span data-stu-id="64aef-137">The system then estimates a unit price of USD 1.00, as we might expect.</span></span> <span data-ttu-id="64aef-138">Sin embargo, cuando llegan los 100 recibos correspondientes, se encuentran en un precio unitario de 2,00 USD cada uno.</span><span class="sxs-lookup"><span data-stu-id="64aef-138">However, when the corresponding 100 receipts arrive, they are at a unit price of USD 2.00 each.</span></span> 

<span data-ttu-id="64aef-139">**Nota:** Aunque las emisiones provocan un inventario negativo, el inventario es positivo cuando se calcula el precio de emisión.</span><span class="sxs-lookup"><span data-stu-id="64aef-139">**Note:** Although the issues create negative inventory, inventory is positive when the issue price is calculated.</span></span> <span data-ttu-id="64aef-140">Por tanto, el precio de coste promedio móvil se usa en lugar del precio del maestro de artículos.</span><span class="sxs-lookup"><span data-stu-id="64aef-140">Therefore, the running average cost price is used instead of the price on the item master.</span></span> <span data-ttu-id="64aef-141">En este momento, el sistema tiene una compensación de valor de inventario de 100,00 dólares.</span><span class="sxs-lookup"><span data-stu-id="64aef-141">At this point, the system has an inventory value offset of USD 100.00.</span></span> <span data-ttu-id="64aef-142">Aunque la compensación se ha creado para 100 artículos, donde había una compensación unitaria de 1,00 dólar cada uno, ahora solo hay una unidad en el inventario.</span><span class="sxs-lookup"><span data-stu-id="64aef-142">Although that offset was built up over 100 pieces, where there was a unit offset of USD 1.00 each, we now have only one piece in inventory.</span></span> <span data-ttu-id="64aef-143">Por tanto, la compensación de 100,00 dólares se asigna a esta única unidad.</span><span class="sxs-lookup"><span data-stu-id="64aef-143">Therefore, the offset of USD 100.00 is allocated to this one piece.</span></span> <span data-ttu-id="64aef-144">El resultado es el precio de coste estimado excesivo.</span><span class="sxs-lookup"><span data-stu-id="64aef-144">The result is the overly inflated estimated cost price.</span></span>

<span data-ttu-id="64aef-151">No obstante, si el modelo empresarial permite las siguientes prácticas, pueden ayudarle a evitar las cantidades negativas que hacen posible el escenario de amplificación de precios:</span><span class="sxs-lookup"><span data-stu-id="64aef-151">However, if your business model allows for the following practices, they can help you avoid the negative quantities that make the pricing amplification scenario possible:</span></span>
