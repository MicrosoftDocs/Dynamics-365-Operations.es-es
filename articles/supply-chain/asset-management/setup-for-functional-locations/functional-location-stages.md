[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="d0c5a-104">Este tema describe cómo configurar los estados y modelos de ciclo de vida de la ubicación funcional en Administración de activos.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="d0c5a-105">Los estados de ciclo de vida de una ubicación funcional definen los estados por los que puede pasar una ubicación funcional, por ejemplo: creada, activa y finalizada.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="d0c5a-106">Puede ver todas las ubicaciones funcionales, independientemente de su estado de ciclo de vida, en la página de lista **Todas las ubicaciones funcionales**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="d0c5a-107">Puede cambiar el estado de una ubicación funcional seleccionándola en la página de lista **Todas las ubicaciones funcionales** y seleccionando **Actualizar estado de ubicación funcional**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="d0c5a-108">Configurar estados de ciclo de vida de ubicación técnica</span><span class="sxs-lookup"><span data-stu-id="d0c5a-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="d0c5a-109">Seleccione **Administración de activos** > **Configuración** > **Ubicaciones funcionales** > **Estados de ciclo de vida**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="d0c5a-110">Seleccione **Nuevo** para crear un nuevo estado de ubicación funcional.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="d0c5a-111">Inserte el identificador de estado en el campo **Estado de ciclo de vida** y un nombre para el estado de la ubicación funcional en el campo **Nombre**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="d0c5a-112">En el campo **Modelos del ciclo de vida**, puede ver el número de modelos de ciclo de vida de ubicación funcional que usan el estado de ubicación funcional.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>

5. <span data-ttu-id="d0c5a-114">Seleccione "Sí" en el botón de alternar **Crear activos** si es posible crear automáticamente un activo con el mismo nombre que la ubicación funcional e instalarlo en la ubicación funcional en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="d0c5a-115">Este botón de alternar está relacionado con el campo **Tipo de activo** en el FastTab **General** del formulario **Tipos de ubicación funcional** (**Administración de activos** > **Configuración** > **Ubicaciones funcionales** > **Tipos de ubicación funcional**).</span><span class="sxs-lookup"><span data-stu-id="d0c5a-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="d0c5a-116">Seleccione "Sí" en el botón de alternar **Cambiar el nombre de la ubicación** si es posible modificar el nombre de la ubicación funcional en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="d0c5a-117">Seleccione "Sí" en el botón de alternar **Nuevas sububicaciones** si es posible agregar nuevas sububicaciones a la ubicación funcional en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="d0c5a-118">Seleccione "Sí" en el botón de alternar **Instalar activos** si es posible instalar activos en la ubicación funcional en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="d0c5a-119">Seleccione "Sí" en el botón de alternar **Eliminar ubicación funcional** si es posible eliminar la ubicación funcional en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="d0c5a-120">Seleccione un estado de activo en el campo **Estado del ciclo de vida** si desea que el estado de ciclo de vida de activo de todos los activos instalados en la ubicación funcional se actualicen automáticamente en este estado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="d0c5a-121">Ejemplo: Si cierra una ubicación funcional y establece el estado de ciclo de vida de la ubicación en “Finalizada”, podría cambiar automáticamente el estado del ciclo de vida de los activos instalados en dicha ubicación funcional a "No en uso".</span><span class="sxs-lookup"><span data-stu-id="d0c5a-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="d0c5a-122">Los estados de ciclo de vida, los modelos de ciclo de vida y los tipos de ubicación funcional están relacionados y se utilizan de la misma forma que los estados de ciclo de vida de la orden de trabajo, los modelos del ciclo de vida de la orden de trabajo y los tipos de orden de trabajo.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="d0c5a-123">Configurar modelos de ciclo de vida de ubicación técnica</span><span class="sxs-lookup"><span data-stu-id="d0c5a-123">Set up functional location lifecycle models</span></span>

2. <span data-ttu-id="d0c5a-128">Seleccione **Nuevo** para crear un nuevo modelo de ciclo de vida.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-128">Select **New** to create a new lifecycle model.</span></span>
3. <span data-ttu-id="d0c5a-129">Inserte el identificador de modelo de ciclo de vida en el campo **Modelo de ciclo de vida** y un nombre para el modelo de ciclo de vida en el campo **Nombre**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="d0c5a-130">En los campos **Tipos de ubicación funcional** y **Estados de ciclo de vida** puede ver el número de tipos de ubicación funcional que usan el modelo del ciclo de vida y el número de estados seleccionados en el modelo de ciclo de vida.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>

5. <span data-ttu-id="d0c5a-133">Si desea seleccionar todos los estados disponibles para un modelo, haga clic en el botón ![seleccionar todos los estados disponibles](media/03-setup-for-functional-locations.png) .</span><span class="sxs-lookup"><span data-stu-id="d0c5a-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="d0c5a-134">Transfiere todos los estados a la sección **Estados de ciclo de vida seleccionados**.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-134">All states are transferred to the **Lifecycle states selected** section.</span></span>

7. <span data-ttu-id="d0c5a-136">Seleccione **Actualizaciones de estado de ciclo de vida** para definir qué estados de ciclo de vida pueden seguir un estado seleccionado.</span><span class="sxs-lookup"><span data-stu-id="d0c5a-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
