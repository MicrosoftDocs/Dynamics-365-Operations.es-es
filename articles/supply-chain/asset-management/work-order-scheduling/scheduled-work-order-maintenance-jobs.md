# <a name="scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="4331e-103">Trabajos de mantenimiento de órdenes de trabajo programados</span><span class="sxs-lookup"><span data-stu-id="4331e-103">Scheduled work order maintenance jobs</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="4331e-104">La página **Trabajos de mantenimiento de órdenes de trabajo programadas** muestra una visión general de las órdenes de trabajo asignadas a un recurso.</span><span class="sxs-lookup"><span data-stu-id="4331e-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="4331e-105">Las órdenes de trabajo mediante los tipos de recursos "Recursos Humanos", "Herramienta" "Máquina" se muestran.</span><span class="sxs-lookup"><span data-stu-id="4331e-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="4331e-106">Por ejemplo, si un trabajador de mantenimiento llama porque está enfermo, puede usar esta página para encontrar rápidamente las órdenes de trabajo asignadas al trabajador y luego asignar otro trabajador de mantenimiento al trabajo.</span><span class="sxs-lookup"><span data-stu-id="4331e-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="4331e-107">Ver trabajos de mantenimiento de órdenes de trabajo programadas</span><span class="sxs-lookup"><span data-stu-id="4331e-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="4331e-108">Haga clic en **Administración de activos** > **Común** > **Órdenes de trabajo** > **Trabajos de mantenimiento de órdenes de trabajo activas programadas**.</span><span class="sxs-lookup"><span data-stu-id="4331e-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="4331e-109">Verá una lista de todas las órdenes de trabajo establecidas para el estado de ciclo de vida "Programada" o "En curso" de la orden de trabajo.</span><span class="sxs-lookup"><span data-stu-id="4331e-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="4331e-110">Puede ordenar la lista, por ejemplo, por trabajador de mantenimiento.</span><span class="sxs-lookup"><span data-stu-id="4331e-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="4331e-111">También puede utilizar el filtro para restringir la lista a órdenes de trabajo asignadas a un determinado recurso o trabajador de mantenimiento.</span><span class="sxs-lookup"><span data-stu-id="4331e-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="4331e-112">Puede ver notas en la orden de trabajo y, si procede, agregar nuevas notas seleccionando una tarea de la orden de trabajo y haciendo clic en **Notas**.</span><span class="sxs-lookup"><span data-stu-id="4331e-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="4331e-113">Si desea asignar un trabajador de mantenimiento a una orden de trabajo, seleccione la orden de trabajo y haga clic en **Orden de trabajo**.</span><span class="sxs-lookup"><span data-stu-id="4331e-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="4331e-114">Se abre la página **Orden de trabajo**.</span><span class="sxs-lookup"><span data-stu-id="4331e-114">The **Work order** page opens.</span></span> <span data-ttu-id="4331e-115">Haga clic en **Distribuir** para programar la orden de trabajo a un determinado trabajador de mantenimiento.</span><span class="sxs-lookup"><span data-stu-id="4331e-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
