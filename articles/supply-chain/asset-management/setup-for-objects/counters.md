# <a name="asset-measures"></a><span data-ttu-id="fcbe3-103">Medidas de activo</span><span class="sxs-lookup"><span data-stu-id="fcbe3-103">Asset measures</span></span>

<span data-ttu-id="fcbe3-104">En este tema se explica cómo crear tipos de medidas de activo en Administración de activos.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-104">The topic explains how to create asset measure types in Asset Management.</span></span> <span data-ttu-id="fcbe3-105">Los tipos de medida de activo se usan para realizar registros de medida en activos (por ejemplo, sobre el número de horas de producción, o la cantidad producida en el activo).</span><span class="sxs-lookup"><span data-stu-id="fcbe3-105">Asset measure types are used to make measurement registrations on assets, for example, regarding number of production hours, or quantity produced on the asset.</span></span> <span data-ttu-id="fcbe3-106">Los tipos de activo están relacionados con los tipos de medidas de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-106">Asset types are related to the asset measure types.</span></span> <span data-ttu-id="fcbe3-107">Esto significa que solo se puede usar una medida de activo en un activo si la medida de activo está configurada en el tipo de activo que se utiliza en el activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-107">This means that an asset measure can only be used on an asset if the asset measure is set up on the asset type used on the asset.</span></span>

<span data-ttu-id="fcbe3-108">Para poder realizar registros de medida en activos, debe crear primero los tipos de medida de activo que desea usar en **Contadores**.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-108">Before you can make measurement registrations on assets, you first create the asset measure types you want to use in **Counters**.</span></span> <span data-ttu-id="fcbe3-109">A continuación, puede crear registros de medida para activos en **Medidas de activo**.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-109">Next, you can create measurement registrations on assets in **Asset measures**.</span></span> 

<span data-ttu-id="fcbe3-110">Las medidas de activo se pueden usar en los planes de mantenimiento.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-110">Asset measures can be used on maintenance plans.</span></span> <span data-ttu-id="fcbe3-111">Una línea de plan de mantenimiento puede ser de tipo "Contador" (relacionada, por ejemplo, con el número de horas de producción o la cantidad producida).</span><span class="sxs-lookup"><span data-stu-id="fcbe3-111">A maintenance plan line can be of type "Counter", for example, relating to number of production hours or quantity produced.</span></span> 

<span data-ttu-id="fcbe3-112">Un registro de la medida de activo se puede actualizar manual o automáticamente en función de las horas de producción o la cantidad producida.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-112">An asset measurement registration can be updated manually or automatically based on production hours or quantity produced.</span></span> <span data-ttu-id="fcbe3-113">Se puede configurar una medida de activo para utilizar uno de los tres métodos de actualización (se seleccionan en el campo **Actualización** de **Contadores**):</span><span class="sxs-lookup"><span data-stu-id="fcbe3-113">An asset measure can be set up to use one of three update methods (selected in the **Update** field in **Counters**):</span></span>
  
- <span data-ttu-id="fcbe3-114">Manual: debe registrar manualmente los valores de medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-114">Manual - you must manually register asset measurement values.</span></span>  
- <span data-ttu-id="fcbe3-115">Horas de producción: el contador se actualiza automáticamente en función del número de horas de producción.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-115">Production hours - the counter is automatically updated based on number of production hours.</span></span>  
- <span data-ttu-id="fcbe3-116">Cantidad de producción: el contador se actualiza automáticamente en función de la cantidad producida.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-116">Production quantity - the counter is automatically updated based on number of quantity produced.</span></span>  

>[!NOTE]
><span data-ttu-id="fcbe3-117">Si se utiliza la cantidad producida, *todos* los artículos registrados se incluyen en el registro de la medida, tanto la cantidad sin errores como la cantidad con errores.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-117">If quantity produced is used, *all* registered items are included in the measurement registration, good quantity as well as error quantity.</span></span> <span data-ttu-id="fcbe3-118">Si es necesario, siempre es posible realizar registros manuales de medidas de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-118">It is always possible to make manual asset measurement registrations, if required.</span></span>

## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="fcbe3-119">Crear tipos de contador para registros de contador de activo</span><span class="sxs-lookup"><span data-stu-id="fcbe3-119">Create counter types for asset counter registrations</span></span>

1. <span data-ttu-id="fcbe3-120">Seleccione **Administración de activos** > **Configuración** > **Tipos de activos** > **Contadores**.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-120">Select **Asset management** > **Setup** > **Asset types** > **Counters**.</span></span>
2. <span data-ttu-id="fcbe3-121">Seleccione **Nuevo** para crear un nuevo tipo de medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-121">Select **New** to create a new asset measure type.</span></span>
3. <span data-ttu-id="fcbe3-122">Inserte un identificador en el campo **Contador**, y un nombre de contador en el campo **Nombre**.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-122">Insert an ID in the **Counter** field, and a counter name in the **Name** field.</span></span>
4. <span data-ttu-id="fcbe3-123">En la ficha desplegable **General**, seleccione una unidad de medida en el campo **Unidad**.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-123">On the **General** FastTab, select a measurement unit in the **Unit** field.</span></span>
5. <span data-ttu-id="fcbe3-124">En el campo **Actualización**, seleccione el método de actualización que desea utilizar para la medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-124">In the **Update** field, select the update method to be used for the asset measure.</span></span>
6. <span data-ttu-id="fcbe3-125">Seleccione "Sí" en el botón de alternancia **Heredar valores de contador** si los activos secundarios de una estructura de activo deben heredar automáticamente los registros de medida de activo realizados en el activo principal.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit asset measure registrations made on the parent asset.</span></span>
7. <span data-ttu-id="fcbe3-126">En el campo **Agregado total**, seleccione el método de suma que se debe usar para una medida de activo que use este tipo de medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-126">In the **Total aggregate** field, select the summation method to be used for an asset measure using this asset measure type.</span></span> <span data-ttu-id="fcbe3-127">"Suma" es la selección estándar que se usa para sumar continuamente los valores registrado y el valor total.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-127">"Sum" is the standard selection used to continuously add registered values to the total value.</span></span> <span data-ttu-id="fcbe3-128">Se puede usar "Promedio" si una medida de activo está configurada para supervisar un umbral (por ejemplo, de temperatura, vibraciones o desgaste de un activo).</span><span class="sxs-lookup"><span data-stu-id="fcbe3-128">"Average" can be used if an asset measure is set up to monitor a threshold, for example, regarding temperature, vibrations, or wear and tear on an asset.</span></span> 
8. <span data-ttu-id="fcbe3-129">En el campo **Desviación por arriba**, inserte el nivel superior en porcentaje para validar si los registros de medida de activo manuales se encuentran dentro de un intervalo esperado.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-129">In the **Deviation over** field, insert the upper level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="fcbe3-130">La validación se basa en un aumento lineal de los registros de medida de activo existentes.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-130">The validation is based on a linear increase in existing asset measure registrations.</span></span>
9. <span data-ttu-id="fcbe3-131">En el campo **Desviación por debajo**, inserte el nivel inferior en porcentaje para validar si los registros de medida de activo manuales se encuentran dentro de un intervalo esperado.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-131">In the **Deviation under** field, insert the lower level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="fcbe3-132">La validación se basa en una disminución lineal de los registros de medida de activo existentes.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-132">The validation is based on a linear decrease in existing asset measure registrations.</span></span>
10. <span data-ttu-id="fcbe3-133">En el campo **Tipo**, seleccione el tipo de mensaje (información, advertencia, error) que se debe mostrar si se producen desviaciones fuera del intervalo definido al crear registros de medida de activo manuales.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual asset measure registrations.</span></span>
11. <span data-ttu-id="fcbe3-134">En la ficha desplegable **Tipos de activos**, agregue los tipos de activos que deben poder usar la medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-134">On the **Asset types** FastTab, add the asset types that should be able to use the asset measure.</span></span>
12. <span data-ttu-id="fcbe3-135">En la ficha desplegable **Medidas de activo relacionadas**, agregue las medidas de activo que desea que se actualicen automáticamente cuando se actualice esta medida de activo.</span><span class="sxs-lookup"><span data-stu-id="fcbe3-135">On the **Related asset measures** FastTab, add the asset measures that you want to be automatically updated when this asset measure is updated.</span></span>
