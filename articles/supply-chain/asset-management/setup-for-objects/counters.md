6. <span data-ttu-id="46875-125">Seleccione "Sí" en el botón de alternancia **Heredar valores de contador** si los activos secundarios de una estructura de activo deben heredar automáticamente los registros de contador realizados en el activo principal.</span><span class="sxs-lookup"><span data-stu-id="46875-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit counter registrations made on the parent asset.</span></span>

9. <span data-ttu-id="46875-131">En el campo **Desviación por debajo**, inserte el nivel inferior en porcentaje para validar si los registros de contador manuales se encuentran dentro de un intervalo esperado.</span><span class="sxs-lookup"><span data-stu-id="46875-131">In the **Deviation under** field, insert the lower level in percent for validating if manual counter registrations are within an expected range.</span></span> <span data-ttu-id="46875-132">La validación se basa en un disminución lineal de los registros de contador existentes.</span><span class="sxs-lookup"><span data-stu-id="46875-132">The validation is based on a linear decrease in existing counter registrations.</span></span>
10. <span data-ttu-id="46875-133">En el campo **Tipo**, seleccione el tipo de mensaje (información, advertencia, error) que se debe mostrar si se producen desviaciones fuera del intervalo definido al crear registros de contador manuales.</span><span class="sxs-lookup"><span data-stu-id="46875-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual counter registrations.</span></span>
