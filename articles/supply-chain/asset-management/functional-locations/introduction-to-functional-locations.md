# <a name="introduction-to-functional-locations"></a><span data-ttu-id="95357-103">Introducción a las ubicaciones funcionales</span><span class="sxs-lookup"><span data-stu-id="95357-103">Introduction to functional locations</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="95357-104">Este tema proporciona una visión general de las ubicaciones funcionales en Administración de activos.</span><span class="sxs-lookup"><span data-stu-id="95357-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="95357-105">Las ubicaciones funcionales son elementos de una estructura técnica, como las unidades funcionales de un sistema.</span><span class="sxs-lookup"><span data-stu-id="95357-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="95357-106">Las ubicaciones funcionales se crean jerárquicamente y en ellas se instalan activos.</span><span class="sxs-lookup"><span data-stu-id="95357-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="95357-107">La configuración de las ubicaciones funcionales de la empresa depende de los requisitos de la misma.</span><span class="sxs-lookup"><span data-stu-id="95357-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="95357-108">A continuación se muestran algunos ejemplos de cómo se pueden usar las ubicaciones funcionales:</span><span class="sxs-lookup"><span data-stu-id="95357-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="95357-109">**Funcional**: las ubicaciones funcionales pueden estar dirigidas al usuario y usarse para gestionar activos que tienen un comportamiento similar.</span><span class="sxs-lookup"><span data-stu-id="95357-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>
- <span data-ttu-id="95357-110">**Relacionada con el proceso**: las ubicaciones funcionales pueden estar orientadas al flujo de trabajo.</span><span class="sxs-lookup"><span data-stu-id="95357-110">**Process-related** – The functional locations can be workflow-oriented.</span></span>

<span data-ttu-id="95357-112">Cada ubicación funcional se administra de forma independiente en Administración de activos.</span><span class="sxs-lookup"><span data-stu-id="95357-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="95357-113">Estas son algunas de las características útiles de las ubicaciones funcionales:</span><span class="sxs-lookup"><span data-stu-id="95357-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="95357-114">Configurar especificaciones de ubicaciones funcionales.</span><span class="sxs-lookup"><span data-stu-id="95357-114">Set up functional location specifications.</span></span>
- <span data-ttu-id="95357-115">Configurar requisitos de especificación de activos.</span><span class="sxs-lookup"><span data-stu-id="95357-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="95357-116">Configurar secuencias de mantenimiento preventivo y reactivo.</span><span class="sxs-lookup"><span data-stu-id="95357-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="95357-117">Gestionar activos instalados.</span><span class="sxs-lookup"><span data-stu-id="95357-117">Manage installed assets.</span></span>
- <span data-ttu-id="95357-118">Realizar un seguimiento de las solicitudes y las órdenes de trabajo activas relacionadas con los activos instalados.</span><span class="sxs-lookup"><span data-stu-id="95357-118">Track active requests and work orders that are related to installed assets.</span></span>
- <span data-ttu-id="95357-119">Realizar un seguimiento de los errores que se registran en los activos.</span><span class="sxs-lookup"><span data-stu-id="95357-119">Track faults that are registered on assets.</span></span>
- <span data-ttu-id="95357-120">Realizar un seguimiento de los costes de mantenimiento de los activos relacionados con una ubicación funcional en un momento dado.</span><span class="sxs-lookup"><span data-stu-id="95357-120">Track maintenance costs on the assets that are related to a functional location at any given time.</span></span>

<span data-ttu-id="95357-121">Las ubicaciones funcionales ofrecen la trazabilidad de los activos en relación con solicitudes, órdenes de trabajo, registros de errores, evaluaciones de condición, registros de detención de producción y registros de contador de activos.</span><span class="sxs-lookup"><span data-stu-id="95357-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>

> [!NOTE]
> <span data-ttu-id="95357-122">Incluso si un activo está instalado en varias ubicaciones funcionales durante su vida útil, los costes se pueden relacionar con cada ubicación.</span><span class="sxs-lookup"><span data-stu-id="95357-122">Even if an asset is installed on various functional locations during its lifetime, the costs can be related to each location.</span></span> <span data-ttu-id="95357-123">En otras palabras, los costes del activo siempre están relacionados con la ubicación funcional en la que se instaló el activo en un momento dado.</span><span class="sxs-lookup"><span data-stu-id="95357-123">In other words, asset costs are always related to the functional location that the asset was installed on at a given time.</span></span>

<span data-ttu-id="95357-124">Las ubicaciones funcionales **no** son flexibles.</span><span class="sxs-lookup"><span data-stu-id="95357-124">Functional locations are **not** flexible.</span></span> <span data-ttu-id="95357-125">Por lo tanto, después de configurar una jerarquía de ubicaciones funcionales, no puede mover las ubicaciones en ella.</span><span class="sxs-lookup"><span data-stu-id="95357-125">Therefore, after you set up a functional location hierarchy, you can't move locations around in it.</span></span> 

<span data-ttu-id="95357-126">Después de crear una jerarquía de ubicaciones funcionales, el paso siguiente es instalar activos en ella.</span><span class="sxs-lookup"><span data-stu-id="95357-126">After you create a functional location hierarchy, the next step is to install assets on it.</span></span> <span data-ttu-id="95357-127">Para más información, consulte [Instalar activos en ubicaciones técnicas](../functional-locations/install-objects-on-functional-locations.md).</span><span class="sxs-lookup"><span data-stu-id="95357-127">For more information, see [Install assets on functional locations](../functional-locations/install-objects-on-functional-locations.md).</span></span>
