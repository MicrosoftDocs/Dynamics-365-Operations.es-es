[!include [banner](includes/banner.md)]

<span data-ttu-id="44490-104">La operación de **recuperación de pedidos** en el punto de venta comercial (POS) proporciona funciones de filtrado y búsqueda de pedidos actualizadas e información específica del pedido.</span><span class="sxs-lookup"><span data-stu-id="44490-104">The **Recall order** operation in the Commerce point of sale (POS) provides updated order search and filtering features and order-specific information.</span></span> <span data-ttu-id="44490-105">Esta función está disponible en las versiones de Commerce 10.0.15 y posteriores.</span><span class="sxs-lookup"><span data-stu-id="44490-105">This feature is available in Commerce versions 10.0.15 and later.</span></span>

<span data-ttu-id="44490-110">Las opciones de visualización son las siguientes.</span><span class="sxs-lookup"><span data-stu-id="44490-110">The display options are as follows.</span></span>
- <span data-ttu-id="44490-111">**Ninguna** - Esta opción despliega la operación sin visualización específica.</span><span class="sxs-lookup"><span data-stu-id="44490-111">**None** – This option deploys the operation with no specific display.</span></span> <span data-ttu-id="44490-112">Cuando un usuario abre la operación con esta configuración, se le pedirá que busque y encuentre pedidos o elija entre un filtro de pedidos predefinido.</span><span class="sxs-lookup"><span data-stu-id="44490-112">When a user opens the operation with this configuration, they will be prompted to search and find orders or choose from a pre-defined order filter.</span></span>
- <span data-ttu-id="44490-113">**Órdenes a cumplir**: cuando un usuario inicia la operación, se ejecutará automáticamente una consulta para buscar y mostrar una lista de pedidos que debe atender la tienda del usuario actual.</span><span class="sxs-lookup"><span data-stu-id="44490-113">**Orders to fulfill** – When a user launches the operation, a query will run automatically to search and display a list of orders that are to be fulfilled by the user's current store.</span></span> <span data-ttu-id="44490-114">Estos pedidos están configurados para recogida en tienda o envío en tienda y las líneas de estos pedidos aún no se han recogido ni embalado.</span><span class="sxs-lookup"><span data-stu-id="44490-114">These orders are configured for in-store pickup or store shipment and the lines of these orders have not yet been picked or packed.</span></span>

- <span data-ttu-id="44490-116">**Órdenes a enviar** - Cuando un usuario inicia la operación, se ejecutará automáticamente una consulta para buscar y mostrar una lista de pedidos que están configurados para el envío desde la tienda actual del usuario.</span><span class="sxs-lookup"><span data-stu-id="44490-116">**Orders to ship** - When a user launches the operation, a query will run automatically to search and display a list of orders that are configured for shipment from the user's current store.</span></span>

<span data-ttu-id="44490-117">Al iniciar la operación **Recuperar pedido** desde el PDV, si la pantalla está configurada para **Ninguno**, un usuario podrá buscar y recuperar los pedidos de una de las siguientes formas.</span><span class="sxs-lookup"><span data-stu-id="44490-117">When launching the **Recall order** operation from POS, if the display is configured to **None**, a user will be able to search and retrieve orders in one of the following ways.</span></span>

- <span data-ttu-id="44490-120">Seleccione **Buscar órdenes** o **Buscar y filtrar** en la barra de aplicaciones para usar el mecanismo de filtrado para localizar pedidos que cumplan con los criterios de filtrado.</span><span class="sxs-lookup"><span data-stu-id="44490-120">Select **Search orders** or **Search and filter** icon on the AppBar to use the filtering mechanism to locate orders that meet the filter criteria.</span></span>

<span data-ttu-id="44490-127">Un usuario puede seleccionar un pedido de la lista para ver detalles adicionales.</span><span class="sxs-lookup"><span data-stu-id="44490-127">A user can select an order on the list to view additional details.</span></span> <span data-ttu-id="44490-128">El panel de información en el lado derecho de la pantalla muestra detalles sobre el pedido seleccionado, incluidos los detalles de la línea del pedido, los detalles de la entrega y los detalles de cumplimiento.</span><span class="sxs-lookup"><span data-stu-id="44490-128">The information panel on the right side of the screen displays specifics on the selected order, including order line details, delivery details, and fulfillment details.</span></span>
