## <a name="overview"></a><span data-ttu-id="dd7b5-105">Visión general</span><span class="sxs-lookup"><span data-stu-id="dd7b5-105">Overview</span></span>

<span data-ttu-id="dd7b5-106">Un módulo de colocación de contenido es un contenedor especial en el que otros módulos se pueden colocar dentro en un diseño específico.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-106">A content placement module is a special container that other modules can be put inside in a specific layout.</span></span> <span data-ttu-id="dd7b5-107">Los módulos de colocación de contenido admiten los siguientes tipos de módulos como módulos secundarios: elemento de colocación de contenido, elemento de bienvenida de cuenta, elemento de pedido de cuenta, elemento de lista de deseos de cuenta y elemento de perfil de cuenta.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-107">Content placement modules support the following types of modules as child modules: content placement item, account welcome item, account order item, account wish list item, and account profile item.</span></span>

<span data-ttu-id="dd7b5-108">Los módulos de colocación de contenido obtienen datos de los módulos secundarios que admiten.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-108">Content placement modules derive data from the child modules that they support.</span></span> <span data-ttu-id="dd7b5-109">Por lo tanto, los módulos de colocación de contenido se pueden usar de diversas maneras, en función de los módulos que se agregan.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-109">Therefore, content placement modules can be used in various ways, depending on the modules that are added to it.</span></span>

<span data-ttu-id="dd7b5-110">Los módulos de elementos de colocación de contenido usan tanto imágenes como texto para mostrar promociones, directivas y características de producto.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-110">Content placement item modules use both images and text to showcase promotions, policies, and product features.</span></span> <span data-ttu-id="dd7b5-111">Por ejemplo, un módulo de elementos de colocación de contenido se puede usar en una página principal para mostrar directivas de la tienda o información de envío.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-111">For example, a content placement item module can be used on a home page to show store policies or shipping information.</span></span> <span data-ttu-id="dd7b5-112">Los módulos de elementos de colocación de contenido se controlan por datos desde el sistema de gestión de contenidos (CMS) y se pueden colocar en cualquier página.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-112">Content placement item modules are driven by data from the content management system (CMS) and can be put on any page.</span></span> <span data-ttu-id="dd7b5-113">Sin embargo, solo se pueden usar dentro de un módulo de colocación de contenido.</span><span class="sxs-lookup"><span data-stu-id="dd7b5-113">However, they can be used only inside a content placement module.</span></span>
