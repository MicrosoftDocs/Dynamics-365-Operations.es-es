<span data-ttu-id="836b9-104">En este tema se tratan los módulos de navegación y se describe cómo agregarlos a las páginas de sitio en Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="836b9-104">This topic covers breadcrumb modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="836b9-105">Los módulos de navegación se utilizan para proporcionar navegación secundaria en las páginas del sitio.</span><span class="sxs-lookup"><span data-stu-id="836b9-105">Breadcrumb modules are used to provide secondary navigation on site pages.</span></span> <span data-ttu-id="836b9-106">Por lo general, se muestran en la parte superior de las páginas, debajo del encabezado.</span><span class="sxs-lookup"><span data-stu-id="836b9-106">They are typically shown at the top of a page, below the header.</span></span> <span data-ttu-id="836b9-107">Aunque los módulos de navegación se pueden agregar a cualquier página, se usan con mayor frecuencia en páginas de detalles de productos (PDP), para mostrar la jerarquía de categorías de productos y proporcionar una forma rápida de moverse por un sitio.</span><span class="sxs-lookup"><span data-stu-id="836b9-107">Although breadcrumb modules can be added to any page, they are most often used on product details pages (PDPs), to show the product category hierarchy and provide a quick way to move around a site.</span></span> <span data-ttu-id="836b9-108">Un módulo de navegación también se puede utilizar para mostrar un vínculo "Volver a los resultados" cuando los usuarios abren un PDP desde una página de búsqueda o lista.</span><span class="sxs-lookup"><span data-stu-id="836b9-108">A breadcrumb module can also be used to show a "Back to results" link when users open a PDP from a search or list page.</span></span> <span data-ttu-id="836b9-109">De esta manera, los usuarios pueden regresar rápidamente a su página de lista filtrada para continuar comprando.</span><span class="sxs-lookup"><span data-stu-id="836b9-109">In this way, users can quickly return to their filtered list page to continue shopping.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="836b9-116">Configuración del módulo de navegación</span><span class="sxs-lookup"><span data-stu-id="836b9-116">Breadcrumb module settings</span></span>

<span data-ttu-id="836b9-117">El módulo de navegación se basa en la opción **Tipo de visualización de navegación en PDP**, que se define en **Configuración del sitio \> Extensiones** en el generador de sitios.</span><span class="sxs-lookup"><span data-stu-id="836b9-117">The breadcrumb module relies on the **Breadcrumb display type on PDP** setting, which is defined at **Site Settings \> Extensions** in site builder.</span></span> <span data-ttu-id="836b9-118">Esta opción tiene tres valores posibles:</span><span class="sxs-lookup"><span data-stu-id="836b9-118">This setting has three possible values:</span></span>

- <span data-ttu-id="836b9-119">**Mostrar jerarquía de categoría**: cuando se selecciona este valor, el módulo de navegación mostrará la jerarquía de categoría completa del producto que se visualiza en el PDP.</span><span class="sxs-lookup"><span data-stu-id="836b9-119">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="836b9-120">**Mostrar de nuevo los resultados**: cuando se selecciona este valor, el módulo de navegación mostrará un vínculo "Volver a resultados" en un PDP si el usuario abrió el PDP desde un módulo que permite un vínculo "Volver a resultados".</span><span class="sxs-lookup"><span data-stu-id="836b9-120">**Show back to results** – When this value is selected, the breadcrumb está disponible cuando los usuarios navegan desde páginas de categorías, búsquedas, listas y listas de recomendaciones.</span><span class="sxs-lookup"><span data-stu-id="836b9-121">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="836b9-122">Para admitir esta funcionalidad, los módulos de colección de functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="836b9-123">Esta propiedad le brinda la flexibilidad de definir qué módulos deben admitir la funcionalidad de vínculo "Volver a los resultados" en el PDP.</span><span class="sxs-lookup"><span data-stu-id="836b9-123">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="836b9-124">Por ejemplo, cuando se selecciona **Mostrar de nuevo los resultados** para la opción **Tipo de visualización de navegación en PDP** del módulo de navegación y se selecciona **Permitir volver a los resultados en PDP** para el módulo de resultados de búsqueda de la página de búsqueda, se mostrará un vínculo "Volver a los resultados" cuando los usuarios naveguen desde la página de búsqueda a un PDP.</span><span class="sxs-lookup"><span data-stu-id="836b9-124">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="836b9-125">**Mostrar jerarquía de categoría y volver a los resultados**: este valor es una combinación de los dos anteriores.</span><span class="sxs-lookup"><span data-stu-id="836b9-125">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="836b9-126">Cuando se selecciona este valor, el módulo de navegación mostrará tanto la jerarquía de categoría completa como un vínculo "Volver a resultados" (si está configurado) en un PDP.</span><span class="sxs-lookup"><span data-stu-id="836b9-126">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>

> [!IMPORTANT]
> <span data-ttu-id="836b9-127">Estos ajustes están disponibles en Dynamics 365 Commerce 10.0.12.</span><span class="sxs-lookup"><span data-stu-id="836b9-127">These settings are available in the Dynamics 365 Commerce 10.0.12 release.</span></span> <span data-ttu-id="836b9-128">Si está actualizando desde una versión anterior de Dynamics 365 Commerce, debe actualizar manualmente el archivo appsettings.json.</span><span class="sxs-lookup"><span data-stu-id="836b9-128">If you are updating from an older version of Dynamics 365 Commerce, you must manually update the appsettings.json file.</span></span> <span data-ttu-id="836b9-129">Para obtener instrucciones sobre cómo actualizar el archivo appsettings.json, consulte [Actualizaciones de SDK y biblioteca de módulos](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span><span class="sxs-lookup"><span data-stu-id="836b9-129">For instructions on updating the appsettings.json file, see [SDK and module library updates](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span></span>

1. <span data-ttu-id="836b9-152">En el cuadro de diálogo **Agregar módulo**, seleccione el módulo **Navegación** y, a continuación, **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="836b9-152">In the **Add Module** dialog box, select the **Breadcrumb** module, and then select **OK**.</span></span>

1. <span data-ttu-id="836b9-154">En el cuadro de diálogo **Texto del vínculo**, introduzca **Inicio** y luego, en **Víncular objetivo**, seleccione **Agregar un vínculo**.</span><span class="sxs-lookup"><span data-stu-id="836b9-154">In the **Link text** dialog box, enter **Home**, and then, under **Link target**, select **Add a link**.</span></span>
