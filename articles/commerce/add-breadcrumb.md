## <a name="overview"></a><span data-ttu-id="b5291-105">Información general</span><span class="sxs-lookup"><span data-stu-id="b5291-105">Overview</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="b5291-116">Configuración del módulo de navegación</span><span class="sxs-lookup"><span data-stu-id="b5291-116">Breadcrumb module settings</span></span>

<span data-ttu-id="b5291-117">El módulo de navegación se basa en la opción **Tipo de visualización de navegación en PDP**, que se define en **Configuración del sitio \> Extensiones** en el generador de sitios.</span><span class="sxs-lookup"><span data-stu-id="b5291-117">The breadcrumb module relies on the **Breadcrumb display type on PDP** setting, which is defined at **Site Settings \> Extensions** in site builder.</span></span> <span data-ttu-id="b5291-118">Esta opción tiene tres valores posibles:</span><span class="sxs-lookup"><span data-stu-id="b5291-118">This setting has three possible values:</span></span>

- <span data-ttu-id="b5291-119">**Mostrar jerarquía de categoría**: cuando se selecciona este valor, el módulo de navegación mostrará la jerarquía de categoría completa del producto que se visualiza en el PDP.</span><span class="sxs-lookup"><span data-stu-id="b5291-119">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="b5291-120">**Mostrar de nuevo los resultados**: cuando se selecciona este valor, el módulo de navegación mostrará un vínculo "Volver a resultados" en un PDP si el usuario abrió el PDP desde un módulo que permite un vínculo "Volver a resultados".</span><span class="sxs-lookup"><span data-stu-id="b5291-120">**Show back to results** – When this value is selected, the breadcrumb module will show a "Back to results" link on a PDP if the user opened the PDP from a module that allows for a "Back to results" link.</span></span> <span data-ttu-id="b5291-121">Esta funcionalidad está disponible cuando los usuarios navegan desde páginas de categorías, búsquedas, listas y listas de recomendaciones.</span><span class="sxs-lookup"><span data-stu-id="b5291-121">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="b5291-122">Para admitir esta funcionalidad, los módulos de colección de productos y resultados de búsqueda tienen una propiedad que se denomina **Permitir volver a los resultados en PDP**.</span><span class="sxs-lookup"><span data-stu-id="b5291-122">To support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="b5291-123">Esta propiedad le brinda la flexibilidad de definir qué módulos deben admitir la funcionalidad de vínculo "Volver a los resultados" en el PDP.</span><span class="sxs-lookup"><span data-stu-id="b5291-123">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="b5291-124">Por ejemplo, cuando se selecciona **Mostrar de nuevo los resultados** para la opción **Tipo de visualización de navegación en PDP** del módulo de navegación y se selecciona **Permitir volver a los resultados en PDP** para el módulo de resultados de búsqueda de la página de búsqueda, se mostrará un vínculo "Volver a los resultados" cuando los usuarios naveguen desde la página de búsqueda a un PDP.</span><span class="sxs-lookup"><span data-stu-id="b5291-124">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="b5291-125">**Mostrar jerarquía de categoría y volver a los resultados**: este valor es una combinación de los dos anteriores.</span><span class="sxs-lookup"><span data-stu-id="b5291-125">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="b5291-126">Cuando se selecciona este valor, el módulo de navegación mostrará tanto la jerarquía de categoría completa como un vínculo "Volver a resultados" (si está configurado) en un PDP.</span><span class="sxs-lookup"><span data-stu-id="b5291-126">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>
