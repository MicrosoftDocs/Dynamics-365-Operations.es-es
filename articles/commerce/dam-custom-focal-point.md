# <a name="customize-image-focal-points"></a><span data-ttu-id="15143-103">Personalizar puntos focales de imagen</span><span class="sxs-lookup"><span data-stu-id="15143-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="15143-104">En este tema se describe cómo personalizar puntos focales de imagen en el generador de sitios de Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="15143-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="15143-105">Información general</span><span class="sxs-lookup"><span data-stu-id="15143-105">Overview</span></span>

<span data-ttu-id="15143-106">Cuando se carga una imagen en la Biblioteca multimedia del generador de sitios de Commerce, el sistema intenta determinar el punto focal de la imagen.</span><span class="sxs-lookup"><span data-stu-id="15143-106">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="15143-107">Por ejemplo, si la imagen tiene una persona, el sistema establecerá el punto focal en la cara de la persona de forma predeterminada.</span><span class="sxs-lookup"><span data-stu-id="15143-107">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="15143-108">En la mayoría de los casos, el punto focal configurado automáticamente funciona bien para todas las ventanas gráficas, pero a veces es posible que desee ajustar el punto focal para asegurarse de que una parte específica de la imagen esté siempre visible.</span><span class="sxs-lookup"><span data-stu-id="15143-108">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="15143-109">Definir un punto focal personalizado para una imagen</span><span class="sxs-lookup"><span data-stu-id="15143-109">Define a custom focal point for an image</span></span>

<span data-ttu-id="15143-110">Para definir un punto focal personalizado para una imagen, siga estos pasos.</span><span class="sxs-lookup"><span data-stu-id="15143-110">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="15143-111">En el panel de navegación izquierdo del generador de sitios de Commerce, seleccione **Biblioteca multimedia**.</span><span class="sxs-lookup"><span data-stu-id="15143-111">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="15143-112">En la ventana principal, seleccione la imagen que desea modificar.</span><span class="sxs-lookup"><span data-stu-id="15143-112">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="15143-113">En la barra de comandos, seleccione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="15143-113">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="15143-114">Seleccione la imagen para pasar al **Modo de edición**.</span><span class="sxs-lookup"><span data-stu-id="15143-114">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="15143-115">En **Modo de edición**, seleccione **Cambiar punto focal**.</span><span class="sxs-lookup"><span data-stu-id="15143-115">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="15143-116">Aparece un control de punto focal circular sobre la imagen.</span><span class="sxs-lookup"><span data-stu-id="15143-116">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="15143-117">Seleccione el control del punto focal para moverlo sobre el punto focal deseado.</span><span class="sxs-lookup"><span data-stu-id="15143-117">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="15143-118">Cuando haya terminado, en la barra de comandos seleccione **Guardar** y luego seleccione **Finalizar edición**.</span><span class="sxs-lookup"><span data-stu-id="15143-118">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
