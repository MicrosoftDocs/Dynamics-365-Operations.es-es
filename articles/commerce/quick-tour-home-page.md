- <span data-ttu-id="24d21-113">**Elemento principal**: normalmente, el primer artículo de la parte superior de la sección principal muestra una o más imágenes de “elemento principal” que resaltan nuevos productos y promociones de la tienda.</span><span class="sxs-lookup"><span data-stu-id="24d21-113">**Hero** – Typically, the first item at the top of the main section shows one or more "hero" images that highlight new products and promotions in the store.</span></span> <span data-ttu-id="24d21-114">Si hay varias imágenes de elemento principal, se hospedan en un módulo de carrusel de modo que los usuarios puedan examinarlos.</span><span class="sxs-lookup"><span data-stu-id="24d21-114">If there are multiple hero images, they are hosted in a carousel module so that users can browse them.</span></span>

- <span data-ttu-id="24d21-121">**Icono**: un diseño de icono de un módulo de bloque de contenido se usa para mostrar varios productos o categoría de productos mediante una combinación de imágenes y texto en un diseño de varias columnas.</span><span class="sxs-lookup"><span data-stu-id="24d21-121">**Tile** – A tile layout of a content block module is used to showcase multiple products or category of products by using a combination of images and text in a multicolumn layout.</span></span> <span data-ttu-id="24d21-122">En la ilustración de una página principal que aparece anteriormente en este tema, se usa un diseño de icono para la representación de tres columnas de los elementos **Tienda de mujeres**, **Tienda de hombres** y **Tienda de accesorios**.</span><span class="sxs-lookup"><span data-stu-id="24d21-122">In the illustration of a home page that appears earlier in this topic, a tile  layout is used for the three-column rendering of the **Shop Women**, **Shop Men**, and **Shop Accessories** items.</span></span>

- <span data-ttu-id="24d21-125">**Bloque de texto**: un módulo de enriquecimiento de contenido se puede usar para presentar contenido de texto en la página principal en un diseño de varias columnas o de columna única.</span><span class="sxs-lookup"><span data-stu-id="24d21-125">**Text block** – A content rich block module can be used to present text content on the home page in a single-column or multicolumn layout.</span></span>
- <span data-ttu-id="24d21-126">**Recomendaciones de producto**: los módulos de recomendaciones de producto se usan para mostrar listas, como **Nuevos**, **Tendencias** y **Más vendidos** en la página principal.</span><span class="sxs-lookup"><span data-stu-id="24d21-126">**Product recommendations** – Product recommendations modules are used to show lists, such as **New**, **Trending**, and **Best Selling** on the home page.</span></span> <span data-ttu-id="24d21-127">Estas listas muestran productos basados en tendencias de compras y se pueden generar de manera algorítmica o mantener manualmente.</span><span class="sxs-lookup"><span data-stu-id="24d21-127">These lists showcase products based on shopping trends, and they can be algorithmically generated or manually curated.</span></span> <span data-ttu-id="24d21-128">Ayudan a los clientes a detectar rápidamente los productos principales y después continuar comprando.</span><span class="sxs-lookup"><span data-stu-id="24d21-128">They help customers quickly discover top products and then continue to shop.</span></span>
