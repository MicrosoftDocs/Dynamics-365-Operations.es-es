<span data-ttu-id="3c0fd-106">La Biblioteca de medios del generador de sitios de Commerce le permite recortar imágenes para optimizarlas para diferentes tipos de módulos y ventanas gráficas.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="3c0fd-107">Recortar una imagen</span><span class="sxs-lookup"><span data-stu-id="3c0fd-107">Crop an image</span></span>

<span data-ttu-id="3c0fd-108">Para recortar una imagen al generador de sitios, siga estos pasos.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="3c0fd-109">En el panel de navegación izquierdo del generador de sitios de Commerce, seleccione **Biblioteca multimedia**.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="3c0fd-110">En la ventana principal, seleccione la imagen que desea modificar.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="3c0fd-111">En la barra de comandos, seleccione **Editar** para revisar el archivo.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-111">On the command bar, select **Edit** to check out the file.</span></span>
1. <span data-ttu-id="3c0fd-112">Seleccione la imagen para pasar al **Modo de edición**.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="3c0fd-113">En el **Modo de edición**, seleccione **Editar vista por módulo**.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="3c0fd-114">Desde el menú desplegable **Módulo**, seleccione el tipo de módulo.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="3c0fd-115">Desde el menú desplegable **Tipo de vista**, seleccione el tipo de vista.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="3c0fd-116">En el menú desplegable **Posición**, seleccione la posición de la imagen.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="3c0fd-117">Desde el menú desplegable **Ventanilla**, seleccione el tamaño de la ventanilla.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="3c0fd-118">La imagen se superpone con el área que representa la región de recorte.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="3c0fd-119">Mueva y cambie el tamaño de la región de recorte según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="3c0fd-120">La relación de aspecto se mantendrá automáticamente.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="3c0fd-121">Cuando haya terminado, en la barra de comandos, seleccione **Guardar** y luego seleccione **Finalizar edición**.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="3c0fd-122">Una vez que se completa el recorte personalizado, las modificaciones de la imagen surtirán efecto casi de inmediato.</span><span class="sxs-lookup"><span data-stu-id="3c0fd-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
