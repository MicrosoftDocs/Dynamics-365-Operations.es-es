1. <span data-ttu-id="d35f3-115">En el panel de navegación, seleccione los puntos suspensivos (**...**) junto a **Pie de página** y, a continuación, seleccione **Agregar módulo**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>

1. <span data-ttu-id="d35f3-117">En el panel de navegación, seleccione los puntos suspensivos (...) junto a **Categoría de pie de página** y, a continuación, seleccione **Agregar módulo**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="d35f3-118">En el cuadro de diálogo, seleccione **Bloque de texto** y después seleccione **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="d35f3-119">En el panel de navegación, seleccione **Bloque de texto**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="d35f3-120">En el panel de propiedades de la derecha, en el campo **Párrafo**, agregue su mensaje de derechos de autor.</span><span class="sxs-lookup"><span data-stu-id="d35f3-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="d35f3-121">Por ejemplo, especifique **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="d35f3-122">Seleccione **Guardar**, **Finalizar edición** y **Publicar**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="d35f3-123">Vaya a **Plantillas**, seleccione la plantilla y, a continuación, seleccione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="d35f3-124">En **página Esquema**, expanda **Cuerpo** y, a continuación, expanda **Página predeterminada**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="d35f3-125">Seleccione el botón de puntos suspensivos junto a **Espacio de pie de página** y, a continuación, seleccione **Agregar fragmento**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="d35f3-126">Seleccione el fragmento que haya creado anteriormente y después, **Seleccionar**.</span><span class="sxs-lookup"><span data-stu-id="d35f3-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="d35f3-127">Seleccione **Finalizar edición** para proteger la plantilla y luego seleccione **Publicar** para publicarla.</span><span class="sxs-lookup"><span data-stu-id="d35f3-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="d35f3-128">El pie de página que contiene el aviso de derechos de autor aparece automáticamente en la parte inferior de todas las páginas que utilicen la plantilla seleccionada.</span><span class="sxs-lookup"><span data-stu-id="d35f3-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
