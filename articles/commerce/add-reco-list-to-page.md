# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>Agregar listas de recomendaciones de producto a páginas de comercio electrónico

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

Este tema explica cómo agregar una lista de recomendaciones de productos a una página de comercio electrónico.

1. En una página existente de creación de comercio electrónico, seleccione **Agregar módulo**.
2. Busque el módulo **colección de productos**.
3. En el asistente para el módulo, seleccione la lista para agregar a la página.
4. Guarde los cambios.

Para obtener más información acerca de los tipos de listas que puede agregar a una página, consulte [Visión general de recomendaciones de producto](product-recommendations.md).
