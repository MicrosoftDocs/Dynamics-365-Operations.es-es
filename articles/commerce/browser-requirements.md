# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Requisitos del explorador para sitios de Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

Este tema cubre los requisitos del navegador para sitios de Microsoft Dynamics 365 Commerce.

Los sitios de Dynamics 365 Commerce son más compatibles con los siguientes exploradores web. Para buscar la versión más reciente para cada explorador web, vaya al sitio web del fabricante de software.

- [Microsoft Edge](https://www.microsoft.com/edge) (hasta la versión actual menos 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (hasta la versión actual menos 1)
- [Google Chrome](https://www.google.com/chrome) (hasta la versión actual menos 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (hasta la versión actual menos 1)

> [!NOTE]
> Internet Explorer no se admite. Si utiliza Internet Explorer, es posible que algunas características no funcionen y que los usuarios tengan una experiencia deficiente.
