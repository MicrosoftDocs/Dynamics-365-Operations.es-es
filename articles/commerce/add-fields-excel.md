# <a name="add-fields-to-an-excel-workbook-to-edit-retail-transactions"></a>Agregar campos a un libro de trabajo de Excel para editar transacciones minoristas

[!include [banner](../includes/banner.md)]

En este tema se describe cómo agregar campos a un libro de Microsoft Excel para poder editar transacciones minoristas en Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Visión general

Cuando genera un archivo de Excel para poder editar transacciones minoristas, el archivo se rellena con algunos campos predeterminados. Si un campo que debe actualizarse no está visible de forma predeterminada en el archivo de Excel generado, puede agregarlo.

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a>Agregar campos a una hoja de cálculo de un libro de Excel

Para agregar campos a un libro de Excel para que se puedan editar transacciones minoristas, siga estos pasos.

1. Descargue y abra el archivo de Excel en la página **Extractos**, la página **Transacciones comerciales** o el icono **Errores de validación de transacciones** en el espacio de trabajo **Operaciones financieras de tienda**.
1. Seleccione **Diseño**.
1. Seleccione el símbolo del lápiz para la tabla que desee y, a continuación, en la lista de campos disponibles, busque y seleccione el campo que desea agregar.
1. Seleccione **Agregar** y, a continuación, **Actualizar**. Puede reordenar los campos.
1. Una vez completada la actualización, seleccione **Actualizar** para capturar los datos para la nueva columna.

El nuevo campo y los datos para él deberían estar ahora disponibles para su edición en Excel.
