## <a name="overview"></a><span data-ttu-id="b12b1-105">Visión general</span><span class="sxs-lookup"><span data-stu-id="b12b1-105">Overview</span></span>

<span data-ttu-id="b12b1-106">Cuando genera un archivo de Excel para poder editar transacciones minoristas, el archivo se rellena con algunos campos predeterminados.</span><span class="sxs-lookup"><span data-stu-id="b12b1-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="b12b1-107">Si un campo que debe actualizarse no está visible de forma predeterminada en el archivo de Excel generado, puede agregarlo.</span><span class="sxs-lookup"><span data-stu-id="b12b1-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="b12b1-108">Agregar campos a una hoja de cálculo de un libro de Excel</span><span class="sxs-lookup"><span data-stu-id="b12b1-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="b12b1-109">Para agregar campos a un libro de Excel para que se puedan editar transacciones minoristas, siga estos pasos.</span><span class="sxs-lookup"><span data-stu-id="b12b1-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="b12b1-110">Descargue y abra el archivo de Excel en la página **Extractos**, la página **Transacciones comerciales** o el icono **Errores de validación de transacciones** en el espacio de trabajo **Operaciones financieras de tienda**.</span><span class="sxs-lookup"><span data-stu-id="b12b1-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="b12b1-111">Seleccione **Diseño**.</span><span class="sxs-lookup"><span data-stu-id="b12b1-111">Select **Design**.</span></span>
1. <span data-ttu-id="b12b1-112">Seleccione el símbolo del lápiz para la tabla que desee y, a continuación, en la lista de campos disponibles, busque y seleccione el campo que desea agregar.</span><span class="sxs-lookup"><span data-stu-id="b12b1-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="b12b1-113">Seleccione **Agregar** y, a continuación, **Actualizar**.</span><span class="sxs-lookup"><span data-stu-id="b12b1-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="b12b1-114">Puede reordenar los campos.</span><span class="sxs-lookup"><span data-stu-id="b12b1-114">You can reorder fields.</span></span>
1. <span data-ttu-id="b12b1-115">Una vez completada la actualización, seleccione **Actualizar** para capturar los datos para la nueva columna.</span><span class="sxs-lookup"><span data-stu-id="b12b1-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="b12b1-116">El nuevo campo y los datos para él deberían estar ahora disponibles para su edición en Excel.</span><span class="sxs-lookup"><span data-stu-id="b12b1-116">The new field and data for it should now be available for editing in Excel.</span></span>
