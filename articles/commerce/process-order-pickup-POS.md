# <a name="process-customer-order-pickups-in-pos"></a><span data-ttu-id="be1e0-103">Procesar recogidas de pedidos de clientes en PDV</span><span class="sxs-lookup"><span data-stu-id="be1e0-103">Process customer order pickups in POS</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="be1e0-104">Cuando se crea un [pedido de cliente](customer-orders-overview.md) para la recogida en la tienda, un usuario de la tienda puede usar la aplicación de punto de venta (PDV) para iniciar la recogida del inventario.</span><span class="sxs-lookup"><span data-stu-id="be1e0-104">When a [customer order](customer-orders-overview.md) is created for store pickup, a store user can use the point of sale (POS) application to start the pickup of inventory.</span></span> <span data-ttu-id="be1e0-105">PDV ejecutará la captura de pago final según sea necesario.</span><span class="sxs-lookup"><span data-stu-id="be1e0-105">POS will run the final payment capture as required.</span></span> <span data-ttu-id="be1e0-106">También completará el inventario y el registro financiero de las cantidades recogidas.</span><span class="sxs-lookup"><span data-stu-id="be1e0-106">It will also complete the inventory and financial posting for the quantities that are picked up.</span></span>

<span data-ttu-id="be1e0-107">Si es un usuario de la tienda, puede realizar la recogida mediante la operación **Recuperar pedido** o la operación **Cumplimiento de pedido** en PDV.</span><span class="sxs-lookup"><span data-stu-id="be1e0-107">If you're a store user, you can perform the pickup by using either the **Recall order** operation or the **Order fulfillment** operation in POS.</span></span> <span data-ttu-id="be1e0-108">Para hacer que la operación **Recoger** esté disponible, primero hay que seguir uno de estos pasos:</span><span class="sxs-lookup"><span data-stu-id="be1e0-108">To make the **Pick up** operation available, you must first follow one of these steps:</span></span>

- <span data-ttu-id="be1e0-109">Para usar la operación **Recuperar pedido**, busque y seleccione el pedido que se va a recoger.</span><span class="sxs-lookup"><span data-stu-id="be1e0-109">To use the **Recall order** operation, search for and select the order that will be picked up.</span></span>
- <span data-ttu-id="be1e0-110">Para usar la operación **Cumplimiento de pedido**, busque y seleccione una o varias líneas de pedido.</span><span class="sxs-lookup"><span data-stu-id="be1e0-110">To use the **Order fulfillment** operation, search for and select one or more order lines.</span></span>

<span data-ttu-id="be1e0-111">Si el pedido seleccionado o las líneas de pedido no están configuradas para su recogida en esa tienda concreta, o si ya se ha recogido el pedido por completo, la operación **Recoger** no estará disponible.</span><span class="sxs-lookup"><span data-stu-id="be1e0-111">If the selected order or order lines aren't configured for pickup at that specific store, or if the order has already been fully picked up, the **Pick up** operation will be unavailable.</span></span>

![Operación de recogida](media/pickupoperation.png)

<span data-ttu-id="be1e0-113">En Microsoft Dynamics 365 Commerce, versión 10.0.17 y posteriores, se puede activar la característica **Experiencia de usuario mejorada para el procesamiento de pedidos de recogida en el punto de venta** a través de Administración de características en la sede de Commerce.</span><span class="sxs-lookup"><span data-stu-id="be1e0-113">In Microsoft Dynamics 365 Commerce version 10.0.17 and later, the **Improved user experience for pick up order processing in Point of Sale** feature can be turned on through Feature management in Commerce headquarters.</span></span> <span data-ttu-id="be1e0-114">Si esta característica está desactivada, los usuarios no podrán seleccionar cantidades para recoger.</span><span class="sxs-lookup"><span data-stu-id="be1e0-114">If this feature is turned off, users can't select pickup quantities.</span></span> <span data-ttu-id="be1e0-115">De forma predeterminada, la cantidad total solicitada para la línea es la cantidad que se va a recoger.</span><span class="sxs-lookup"><span data-stu-id="be1e0-115">By default, the full quantity that was ordered for the line is the quantity that will be picked up.</span></span> <span data-ttu-id="be1e0-116">Esta experiencia puede ser problemática, ya que los usuarios pueden olvidarse de seleccionar algunos artículos para recoger cuando realizan la recogida a través del cumplimiento del pedido.</span><span class="sxs-lookup"><span data-stu-id="be1e0-116">This experience can be problematic, because users might forget to select some items for pickup when they perform the pickup through order fulfillment.</span></span>

<span data-ttu-id="be1e0-117">La característica **Experiencia de usuario mejorada para el procesamiento de pedidos de recogida en el punto de venta** ofrece a los usuarios más control sobre la selección de productos que se va a recoger y la cantidad de esos productos que se va a recoger.</span><span class="sxs-lookup"><span data-stu-id="be1e0-117">The **Improved user experience for pick up order processing in Point of Sale** feature gives users more control over the selection of products that will be picked up and the quantity of those products that will be picked up.</span></span> <span data-ttu-id="be1e0-118">Los usuarios no tienen que seleccionar todas las líneas del pedido de ventas en la página de cumplimiento del pedido antes de seleccionar **Recoger**.</span><span class="sxs-lookup"><span data-stu-id="be1e0-118">Users don't have to select every line of the sales order on the order fulfillment page before they select **Pick up**.</span></span> <span data-ttu-id="be1e0-119">Se mostrarán todos los artículos que se pueden recoger.</span><span class="sxs-lookup"><span data-stu-id="be1e0-119">All the items that can be picked up will be shown.</span></span> <span data-ttu-id="be1e0-120">Los usuarios pueden especificar varias líneas para recoger incluso si solo se selecciona una línea de producto.</span><span class="sxs-lookup"><span data-stu-id="be1e0-120">Users can specify multiple lines for pickup even if only one product line is selected.</span></span>

<span data-ttu-id="be1e0-121">Cuando la característica **Experiencia de usuario mejorada para el procesamiento de pedidos de recogida en el punto de venta** está activada y se selecciona la operación **Recoger**, aparece el cuadro de diálogo **Recoger**.</span><span class="sxs-lookup"><span data-stu-id="be1e0-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="be1e0-122">En este cuadro puede seleccionar los artículos y las cantidades que se recogerán.</span><span class="sxs-lookup"><span data-stu-id="be1e0-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="be1e0-123">De forma predeterminada, cualquier cantidad pedida que tenga inventario en estado de recogida o empaquetado se considera apta para recogida.</span><span class="sxs-lookup"><span data-stu-id="be1e0-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="be1e0-124">De forma predeterminada, esa cantidad se establece como la cantidad de recogida.</span><span class="sxs-lookup"><span data-stu-id="be1e0-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="be1e0-125">Puede cambiar la cantidad introducida siempre que esta cantidad no sea 0 (cero) y no exceda de la cantidad total abierta (es decir, no facturada) para la línea seleccionada.</span><span class="sxs-lookup"><span data-stu-id="be1e0-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>

<span data-ttu-id="be1e0-127">Después de seleccionar las cantidades que se van a recoger y de seleccionar **Recoger**, aparece la página de transacción.</span><span class="sxs-lookup"><span data-stu-id="be1e0-127">After you select the quantities that will be picked up and then select **Pick up**, the transaction page appears.</span></span> <span data-ttu-id="be1e0-128">Si la característica [pagos de omnicanal](omni-channel-payments.md) está activada y hay registrados pagos con tarjeta de crédito preautorizados, debe aplicar el pago.</span><span class="sxs-lookup"><span data-stu-id="be1e0-128">If the [omni-channel payments](omni-channel-payments.md) feature is turned on, and there are pre-authorized credit card payments on file, you must apply the payment.</span></span>

<span data-ttu-id="be1e0-129">En la página de transacción, el sistema calcula los importes adeudados calculando el total adeudado por los artículos para recoger seleccionados y restando a continuación los depósitos aplicados previamente o los pagos autorizados con tarjeta de crédito.</span><span class="sxs-lookup"><span data-stu-id="be1e0-129">On the transaction page, the system calculates the amounts that are due by calculating the total that is due for the selected pickup items and then subtracting any previously applied deposits or authorized credit card payments.</span></span> <span data-ttu-id="be1e0-130">Debe procesar el pago para completar la transacción de recogida.</span><span class="sxs-lookup"><span data-stu-id="be1e0-130">You must process payment to complete the pickup transaction.</span></span> <span data-ttu-id="be1e0-131">Si el [diseño de pantalla](pos-screen-layouts.md) de la página de transacción está configurado de forma que incluya la operación **Concluir transacción** y no se adeuda ningún importe, puede completar la transacción sin seleccionar un método de pago.</span><span class="sxs-lookup"><span data-stu-id="be1e0-131">If the [screen layout](pos-screen-layouts.md) of the transaction page is configured so that it includes the **Conclude transaction** operation, and no amount is due, you can complete the transaction without selecting a payment method.</span></span> <span data-ttu-id="be1e0-132">Si la operación **Concluir transacción** no está disponible, puede seleccionar vínculo **Importe de 0,00 $ adeudado** en el panel **Totales** para concluir la transacción sin tener que seleccionar un método de pago.</span><span class="sxs-lookup"><span data-stu-id="be1e0-132">If the **Conclude transaction** operation isn't available, you can select the **$0.00 amount due** link in the **Totals** pane to conclude the transaction without having to select a payment method.</span></span>
