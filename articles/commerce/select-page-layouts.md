# <a name="select-page-layouts"></a>Seleccionar diseños de página


[!include [banner](includes/banner.md)]

Este tema explica cómo crear y seleccionar diseños de página en Microsoft Dynamics 365 Commerce.

## <a name="create-layouts-for-an-existing-page"></a>Crear diseños para una página existente

> [!NOTE]
> Puede crear diseños para una página existente solo si esa página tiene al menos dos módulos en la franja principal.

Para crear diseños para una página existente, siga estos pasos.

1. Vaya a **Páginas** y busque la página existente en la lista. Use la característica de búsqueda según sea necesario.
1. Seleccione la página, desprótejala y, a continuación, selecciónela para abrirla. Anote el orden del módulo.
1. Seleccione **Guardar como nuevo diseño**.
1. Especifique un nombre para el diseño y después seleccione **Aceptar**.
1. Seleccione **Convertir en diseño incrustado**.
1. Cambiar el orden de los módulos según sea necesario y anote el nuevo orden.
1. Seleccione **Guardar como nuevo diseño**.
1. Especifique un nombre para el diseño y después seleccione **Aceptar**.
1. Seleccione **Cambiar diseño**, seleccione el primer diseño que ha creado y, a continuación, **Aceptar**. Anote el orden del módulo. Cámbielo para que coincida con el orden del módulo que se guardó con el diseño.

## <a name="select-a-different-layout-for-an-existing-page"></a>Seleccione un diseño diferente para una página existente

> [!NOTE]
> Puede seleccionar otro diseño para una página existente si la plantilla que se usó para crear esa página tiene más de un diseño.

Para seleccionar un diseño diferente para una página existente, siga estos pasos.

1. Vaya a **Páginas** y busque la página existente en la lista. Use la característica de búsqueda según sea necesario.
1. Seleccione la página, desprótejala y, a continuación, selecciónela para abrirla.
1. Seleccione **Cambiar diseño**.
1. Seleccione el nuevo diseño de la página y, a continuación, **Aceptar**. El editor de páginas se actualiza para mostrar el nuevo diseño.
