# <a name="ways-to-add-content"></a>Métodos para agregar contenido

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

Este tema proporciona información acerca de cómo agregar y administrar contenido en su sitio de Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Visión general

Hay muchas maneras de cambiar el aspecto y el contenido del sitio. En función del nivel necesario de personalización, muchos de estos cambios se pueden implementar por no desarrolladores. Por ejemplo, no se tiene que escribir ningún código para crear plantillas, seleccionar temas, y seleccionar y configurar módulos. Por el contrario, se requieren aptitudes de desarrollo para crear un nuevo tema o módulo, ya que deben usarse el kit de desarrollo de software (SDK) de comercio electrónico y el flujo de trabajo de implementación de Lifecycle Services (LCS) de Microsoft Dynamics.

Los temas siguientes ofrecen información detallada acerca de cómo agregar y administrar contenido del sitio. Se centran en áreas de su sitio que no requieren un desarrollador. Según sea necesario, designan tareas que requieran un trabajo del SDK.

- Para cambiar el texto, las imágenes o el vídeo en una página de sitio existente, consulte [Trabajar con módulos](work-with-modules.md).
- Para ayudar a garantizar una experiencia infalible, de acuerdo con los valores de la marca para autores de contenido web, consulte [Trabajar con plantillas](work-with-templates.md).
- Para reorganizar secciones sobre una página de sitio, consulte [Trabajar con diseños](work-with-layouts.md).
- Para cambiar las fuentes, los colores y el aspecto general de las páginas del sitio, consulte [Seleccionar un tema de sitio](select-site-theme.md).
