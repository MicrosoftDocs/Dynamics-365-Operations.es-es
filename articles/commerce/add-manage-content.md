# <a name="ways-to-add-content"></a>Métodos para agregar contenido

[!include [banner](includes/banner.md)]

En este tema se proporciona una descripción general y vínculos a la documentación sobre cómo administrar contenido mediante el conjunto de herramientas de creación web del configurador de sitios de Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Visión general

Hay muchas maneras de cambiar el aspecto y el contenido del sitio. En función del nivel de personalización requerido, muchos de estos cambios pueden ser implementados por personas sin conocimientos de programación en el configurador de sitios, el conjunto de herramientas de creación web incluido en Dynamics 365 Commerce. El configurador de sitios le permite crear plantillas, seleccionar temas y seleccionar y configurar módulos sin tener que escribir código. Por el contrario, se requieren aptitudes de desarrollo para crear un nuevo tema o módulo, ya que deben usarse el kit de desarrollo de software (SDK) de comercio electrónico y el flujo de trabajo de implementación de Lifecycle Services (LCS) de Microsoft Dynamics.

Los siguientes temas son buenos puntos de partida para comenzar a comprender cómo se agrega y administra el contenido del sitio. La mayoría de los temas mostrados se centran en áreas de su sitio que no requieren un desarrollador. Algunos abordan la edición básica de contenido, mientras que otros se centran en las tareas de administrador del sitio. Cada uno de estos temas denotará tareas específicas que pueden requerir trabajar con el SDK. En cada tema se supone que ya ha aprovisionado un sitio y se le ha otorgado acceso al conjunto de herramientas del configurador de sitios para su sitio.

Seleccione uno de los siguientes temas para empezar.

- Para familiarizarse con la terminología de administración de contenido utilizada en el configurador de sitios y en esta documentación, consulte el [Glosario del modelo de página](page-elements-overview.md).
- Para comprender cómo funcionan los módulos en los flujos de trabajo de administración de contenido, consulte [Trabajar con módulos](work-with-modules.md).
- Para cambiar el texto, las imágenes o el vídeo en una página de sitio existente, consulte [Trabajar con módulos](work-with-modules.md).
- Para ver cómo los fragmentos pueden hacer que la administración de contenido sea más eficiente y flexible, consulte [Trabajar con fragmentos](work-with-fragments.md).
- Para ayudar a garantizar una experiencia de autoría de marca exitosa para los autores de contenido web, consulte [Visión general de plantillas y diseños](templates-layouts-overview.md) y [Trabajar con plantillas](work-with-templates.md).
- Para reorganizar secciones sobre una página de sitio, consulte [Trabajar con diseños](work-with-layouts.md).
- Para cambiar las fuentes, los colores y el aspecto general de las páginas del sitio, consulte [Seleccionar un tema de sitio](select-site-theme.md) o [Trabajar con archivos de anulación de CSS](css-override-files.md).
- Para reorganizar o agregar nuevas opciones de navegación, consulte [Personalizar navegación del sitio](customize-site-navigation.md).
- Para aprender a organizar, previsualizar y publicar un amplio conjunto de cambios simultáneos de contenido web, consulte [Trabajar con grupos de publicación](publish-groups.md).
