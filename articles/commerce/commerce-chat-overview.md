# <a name="commerce-chat-features-overview"></a>Información general de las características de chat de Commerce

[!include [banner](includes/banner.md)]

Al integrar la Plataforma omnicanal para Customer Service de Microsoft Dynamics 365 y las aplicaciones de Power Virtual Agents con sitios web de comercio electrónico de Dynamics 365 Commerce, las empresas pueden empoderar a sus compradores en línea a través de capacidades de chat. Este artículo proporciona una descripción general de las funciones de chat de Commerce que están disponibles para los compradores en línea cuando la Plataforma omnicanal para Customer Service y las aplicaciones de Power Virtual Agents se integran con los sitios web de comercio electrónico de Dynamics 365 Commerce.

## <a name="omnichannel-for-customer-service"></a>Plataforma omnicanal para Customer Service

La Plataforma omnicanal para Customer Service es una aplicación robusta que amplía el poder de Dynamics 365 Customer Service. Permite a las organizaciones conectarse e interactuar instantáneamente con sus clientes a través de canales como chat en vivo, voz y servicio de mensajes cortos (SMS). Para obtener más información sobre la Plataforma omnicanal para Customer Service, consulte [Introducción a la Plataforma omnicanal para Customer Service](/dynamics365/customer-service/introduction-omnichannel). Para obtener información sobre cómo integrar la Plataforma omnicanal para Customer Service con un sitio web de comercio electrónico de Dynamics 365 Commerce, consulte [Chat de comercio con omnicanal para el módulo de servicio al cliente](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

Power Virtual Agents le permite crear potentes chatbots impulsados por IA para una variedad de solicitudes. Para obtener más información acerca de Power Virtual Agents, consulte [Información general de Power Virtual Agents](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Para obtener información sobre cómo integrar Power Virtual Agents con un sitio web de comercio electrónico de Dynamics 365 Commerce, consulte [Chat de Commerce con Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> Power Virtual Agents ya está incluido en la Plataforma omnicanal para Customer Service. Si necesita capacidades tanto de agente virtual (chatbot) como de agente en vivo, use la Plataforma omnicanal para Customer Service.
