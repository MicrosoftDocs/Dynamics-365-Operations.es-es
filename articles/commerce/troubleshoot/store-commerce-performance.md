# <a name="troubleshoot-store-commerce-performance-issues"></a>Solucionar problemas de rendimiento de Store Commerce

[!include [banner](../includes/banner.md)]

Este artículo explica cómo solucionar problemas de rendimiento en la aplicación Store Commerce de Microsoft Dynamics 365 Commerce.

- Cuando solucione problemas de rendimiento para Store Commerce, evite el inicio de sesión remoto. En su lugar, inicie sesión directamente en el dispositivo Store Commerce.
- Si la aplicación Store Commerce es lenta, verifique el uso de la CPU, la memoria, la red y el disco mediante el Administrador de tareas o el Monitor de recursos. Asegúrese de que no haya un consumo significativo de los recursos del dispositivo.
- Verifique el estado de conectividad de su red y confirme que las solicitudes/respuestas HTTPS no toman más de unos segundos.
