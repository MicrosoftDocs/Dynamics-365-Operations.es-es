## <a name="overview"></a><span data-ttu-id="e757d-105">Visión general</span><span class="sxs-lookup"><span data-stu-id="e757d-105">Overview</span></span>

<span data-ttu-id="e757d-106">En un entorno minorista, siempre es difícil asegurarse de que las tareas sean realizadas por la persona adecuada en el momento adecuado.</span><span class="sxs-lookup"><span data-stu-id="e757d-106">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="e757d-107">Los minoristas deben poder notificar a los trabajadores sobre las próximas tareas y proporcionar un contexto comercial relacionado, para que las tareas se puedan completar correctamente y a tiempo.</span><span class="sxs-lookup"><span data-stu-id="e757d-107">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="e757d-108">La gestión de tareas es una característica de productividad en Dynamics 365 Commerce que permite a los gerentes y trabajadores crear listas de tareas, administrar criterios de asignación, rastrear el estado de la tarea e integrar estas operaciones entre las aplicaciones administrativas y de punto de venta (PDV) de Commerce.</span><span class="sxs-lookup"><span data-stu-id="e757d-108">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="e757d-109">Las personas de la sede pueden usar la gestión de tareas para crear listas de tareas para tiendas minoristas y para rastrear el estado por tienda o trabajador.</span><span class="sxs-lookup"><span data-stu-id="e757d-109">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="e757d-110">También pueden crear tareas recurrentes (por ejemplo, "Lista de verificación de cierre del jueves por la noche").</span><span class="sxs-lookup"><span data-stu-id="e757d-110">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="e757d-111">Los gerentes de tienda pueden usar la administración de tareas para asignar tareas a trabajadores individuales, enviar notificaciones sobre tareas futuras o tareas vencidas, actualizar el estado de la tarea y crear tareas de propósito único en la aplicación PDV.</span><span class="sxs-lookup"><span data-stu-id="e757d-111">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="e757d-112">Los trabajadores pueden ver notificaciones, ver detalles de la tarea y actualizar el estado de la tarea en el PDV.</span><span class="sxs-lookup"><span data-stu-id="e757d-112">Workers can then see notifications, view task details, and update task status at the POS.</span></span>
