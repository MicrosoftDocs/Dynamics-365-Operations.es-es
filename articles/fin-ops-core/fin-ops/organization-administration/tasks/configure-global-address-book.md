[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="c5ddc-107">En el panel de exploración, vaya a **Módulos > Administración de la organización > Libreta de direcciones global > Parámetros de libreta de direcciones global**.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>

8. <span data-ttu-id="c5ddc-114">En el campo **Cliente**, seleccione una opción.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="c5ddc-115">En el campo **Proveedor**, seleccione una opción.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="c5ddc-116">En el campo **Cliente potencial**, seleccione una opción.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="c5ddc-117">En el campo **Competidor**, seleccione una opción.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="c5ddc-118">Haga clic en la pestaña **Seguridad de ubicación privada**.</span><span class="sxs-lookup"><span data-stu-id="c5ddc-118">Click the **Private location security** tab.</span></span>
