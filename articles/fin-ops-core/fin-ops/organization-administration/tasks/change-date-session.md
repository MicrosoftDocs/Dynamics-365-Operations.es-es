# <a name="change-the-date-for-a-session"></a>Cambiar la fecha de una sesión

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

Este artículo explica cómo cambiar la fecha de una sesión. De forma predeterminada, se utiliza la fecha actual al especificar y registrar entradas del diario y documentos de origen. Puede cambiar la fecha que se usa para la sesión actual. Utilice esta función para atrasar la fecha de las entradas del diario o de los documentos de origen, en caso necesario.

1. En el panel de exploración, vaya a **Módulos > Común > Común > Fecha y hora de la sesión**.
2. En el campo **Fecha**, escriba una fecha.
3. Seleccione **Aceptar**.
