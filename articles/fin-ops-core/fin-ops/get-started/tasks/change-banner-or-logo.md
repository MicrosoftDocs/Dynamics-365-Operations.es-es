# <a name="change-the-banner-or-logo"></a><span data-ttu-id="9104d-103">Cambiar el encabezado o el logotipo</span><span class="sxs-lookup"><span data-stu-id="9104d-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="9104d-104">El procedimiento siguiente muestra los pasos que los administradores del sistema pueden usar para actualizar la imagen del banner o del logotipo que se muestra para una entidad jurídica.</span><span class="sxs-lookup"><span data-stu-id="9104d-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="9104d-105">La empresa de datos de prueba utilizada para crear este procedimiento es USMF.</span><span class="sxs-lookup"><span data-stu-id="9104d-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="9104d-106">Vaya a **Panel de navegación > Módulos > Administración de la organización > Organizaciones > Entidades jurídicas**.</span><span class="sxs-lookup"><span data-stu-id="9104d-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="9104d-107">En la lista de la izquierda, seleccione la entidad jurídica para la que desea actualizar el banner o el logotipo.</span><span class="sxs-lookup"><span data-stu-id="9104d-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="9104d-108">Si ya está seleccionada, vaya al paso siguiente.</span><span class="sxs-lookup"><span data-stu-id="9104d-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="9104d-109">Expanda la ficha **Imagen de panel de información**.</span><span class="sxs-lookup"><span data-stu-id="9104d-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="9104d-110">Seleccione **Cambiar**.</span><span class="sxs-lookup"><span data-stu-id="9104d-110">Select **Change**.</span></span>
    
    <span data-ttu-id="9104d-111">La resolución ideal para una imagen de banner es 1920 x 281 píxeles.</span><span class="sxs-lookup"><span data-stu-id="9104d-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="9104d-112">El ancho ideal para una imagen de logotipo es 350 píxeles.</span><span class="sxs-lookup"><span data-stu-id="9104d-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="9104d-113">Seleccione **Guardar**.</span><span class="sxs-lookup"><span data-stu-id="9104d-113">Select **Save**.</span></span>
6. <span data-ttu-id="9104d-114">Vaya a **Panel de navegación > Módulos > Común > Común > Panel de información predeterminado**.</span><span class="sxs-lookup"><span data-stu-id="9104d-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="9104d-115">Debe ver la nueva imagen de banner o logotipo en el panel de información.</span><span class="sxs-lookup"><span data-stu-id="9104d-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="9104d-116">Es posible que tenga que cambiar su empresa, con el selector de empresa, a aquella para la que cargó el banner.</span><span class="sxs-lookup"><span data-stu-id="9104d-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
