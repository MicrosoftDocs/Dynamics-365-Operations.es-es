[!include[banner](../includes/banner.md)]

<span data-ttu-id="b14ca-104">A medida que configura las asignaciones o los formatos de modelos electrónicos (ER), puede [agregar](#review) los orígenes de datos requeridos del tipo **Unión**.</span><span class="sxs-lookup"><span data-stu-id="b14ca-104">While configuring Electronic reporting (ER) model mappings or formats, you can [add](#review) required data sources of the **Join** type.</span></span> <span data-ttu-id="b14ca-105">En el tiempo de diseño, un origen datos **Unión** se configura como un conjunto de varios orígenes de datos que devuelve una lista de registros.</span><span class="sxs-lookup"><span data-stu-id="b14ca-105">At design time, a **Join** data source is configured as a set of several data sources each of which returns a list of records.</span></span> <span data-ttu-id="b14ca-106">Para cada origen de datos excepto la primera, es necesario definir condiciones necesarias para unir los registros de los orígenes de datos actual y anterior.</span><span class="sxs-lookup"><span data-stu-id="b14ca-106">For every data source except the first one, you need to define necessary conditions to join records of the current and previous data sources.</span></span> <span data-ttu-id="b14ca-107">En el tiempo de ejecución, un origen de datos tipo **Unión** [devuelve](#executeERformat) una sola lista de registros participada que contiene campos de los registros de orígenes de datos anidados.</span><span class="sxs-lookup"><span data-stu-id="b14ca-107">At runtime, a configured data source of **Join** type [returns](#executeERformat) a single joined list of records containing fields from the records of nested data sources.</span></span>

<span data-ttu-id="b14ca-108">Actualmente se admiten los siguientes tipos de uniones:</span><span class="sxs-lookup"><span data-stu-id="b14ca-108">The following types of joins are currently supported:</span></span>

- <span data-ttu-id="b14ca-109">Unión externa (izquierda):</span><span class="sxs-lookup"><span data-stu-id="b14ca-109">Outer (left) join:</span></span>
    - <span data-ttu-id="b14ca-110">Une a todos los registros del primer origen de datos (de más a la izquierda) y después cualquier coincidencia de acuerdo a las condiciones configuradas de los registros de la segunda fuente de datos (de más a la derecha).</span><span class="sxs-lookup"><span data-stu-id="b14ca-110">Join all records of the first (left-most) data source and then any matching in accordance to configured conditions records of the second (right-most) data source.</span></span>
- <span data-ttu-id="b14ca-111">Unión interna (derecha):</span><span class="sxs-lookup"><span data-stu-id="b14ca-111">Inner (right) join:</span></span>
    - <span data-ttu-id="b14ca-112">Une solo los registros del primer origen de datos (de más a la izquierda) y solo los registros del segundo origen de datos (de más a la derecha) que coincidan según las condiciones configuradas.</span><span class="sxs-lookup"><span data-stu-id="b14ca-112">Join only records of the first (left-most) data source and only records of the second (right-most) data source matching to each other in accordance to configured conditions.</span></span>
