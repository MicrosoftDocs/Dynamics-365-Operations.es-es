# <a name="file-destination"></a>Destino de archivo

[!include [banner](../includes/banner.md)]

Puede configurar un destino de archivo para cada componente FOLDER o FILE de un formato de informe electrónico (ER) que está configurado para generar documentos salientes. Según la configuración del destino, un documento generado está disponible para la descarga desde el navegador web.

En la página **Configuración de destino**, si establece **Habilitado** en **Sí** se abre un cuadro de diálogo para abrir o guardar cuando la configuración termina de ejecutarse.
