# <a name="list-of-er-functions-in-the-mathematical-category"></a>Lista de funciones ER en la categoría matemática.

[!include [banner](../includes/banner.md)]

Las funciones matemáticas de informes electrónicos (ER) se pueden usar para hacer muchos cálculos matemáticos comunes. Este artículo proporciona un resumen de estas funciones.

## <a name="list-of-supported-functions"></a>Lista de funciones permitidas.

| Función | Descripción |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Esta función devuelve el valor absoluto (módulo) de la cifra especificada como un número *Real*. Es decir, devuelve el número sin su signo. |
| [Potencia](er-functions-mathematical-power.md)         | Esta función devuelve un valor *Real* que representa el resultado de elevar el número positivo especificado a la potencia especificada. |
| [Round](er-functions-mathematical-round.md)         | Esta función devuelve el número especificado como un valor *Real* después de que se haya redondeado al número especificado de posiciones decimales. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Esta función devuelve el número especificado como un valor *Real* después de que se haya redondeado hacia abajo al número especificado de posiciones decimales. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Esta función devuelve el número especificado como un valor *Real* después de que se haya redondeado hacia arriba al número especificado de posiciones decimales. |
