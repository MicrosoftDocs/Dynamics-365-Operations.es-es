# <a name="ScreenDestinationType">Destino de la pantalla</a>

[!include [banner](../includes/banner.md)]

Puede configurar un destino de pantalla para cada componente de carpeta o archivo de un formato de informe electrónico (ER) que está configurado para generar documentos salientes. Según la configuración del destino, un documento generado se abre para su vista previa en una pestaña del navegador independiente.

Si establece **Habilitado** en **Sí**, se crea una vista previa de la salida. Puede ver algunos tipos de archivo, como XML, TXT o PDF, directamente en una ventana de explorador. Para otros tipos de archivo, como Microsoft Excel o Word, se usa el servicio de Microsoft Office Online.
