    2. <span data-ttu-id="87856-117">Seleccione **LCS** como el tipo de repositorio.</span><span class="sxs-lookup"><span data-stu-id="87856-117">Select **LCS** as the repository type.</span></span>
    3. <span data-ttu-id="87856-118">Haga clic en **Crear repositorio**.</span><span class="sxs-lookup"><span data-stu-id="87856-118">Click **Create repository**.</span></span>
    4. <span data-ttu-id="87856-119">Si se le solicite, siga las instrucciones de autorización.</span><span class="sxs-lookup"><span data-stu-id="87856-119">If prompted, follow the authorization instructions.</span></span>
