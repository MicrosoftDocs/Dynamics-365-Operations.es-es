# <a name="create-a-record-template-to-facilitate-data-entry"></a><span data-ttu-id="20a75-103">Crear una plantilla de registro para facilitar la entrada de datos</span><span class="sxs-lookup"><span data-stu-id="20a75-103">Create a record template to facilitate data entry</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="20a75-104">Este tema demuestra la creación de plantillas de registro para no tener que especificar los valores de campo que se utilizan a menudo explícitamente para cada registro nuevo.</span><span class="sxs-lookup"><span data-stu-id="20a75-104">This topic demonstrates how to create a record template so that field values that are used often do not have to be entered explicitly for each new record.</span></span> <span data-ttu-id="20a75-105">En este procedimiento, usted creará un registro nuevo para los nuevos portátiles que deberán agregarse a sus activos fijos.</span><span class="sxs-lookup"><span data-stu-id="20a75-105">In this procedure, you'll create a new record for new laptops that should be added to your fixed assets.</span></span> <span data-ttu-id="20a75-106">Este procedimiento usa la empresa de muestra USMF.</span><span class="sxs-lookup"><span data-stu-id="20a75-106">This procedure uses the USMF sample company.</span></span>

1. <span data-ttu-id="20a75-107">En el panel de exploración, vaya a **Módulos > activos fijos > activos fijos > activos fijos**.</span><span class="sxs-lookup"><span data-stu-id="20a75-107">In the navigation pane, go to **Modules > Fixed assets > Fixed assets > Fixed assets**.</span></span>
2. <span data-ttu-id="20a75-108">Seleccione **Nuevo**.</span><span class="sxs-lookup"><span data-stu-id="20a75-108">Select **New**.</span></span>
3. <span data-ttu-id="20a75-109">En el campo **Grupo de activos fijos**, escriba o seleccione un valor.</span><span class="sxs-lookup"><span data-stu-id="20a75-109">In the **Fixed asset group** field, enter or select a value.</span></span>
4. <span data-ttu-id="20a75-110">En el campo **Nombre**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="20a75-110">In the **Name** field, type a value.</span></span> <span data-ttu-id="20a75-111">Por ejemplo, especifique **Portátil corporativo del cliente potencial**.</span><span class="sxs-lookup"><span data-stu-id="20a75-111">For example, enter **Corporate lead laptop**.</span></span>  
5. <span data-ttu-id="20a75-112">En el campo **Nombre de búsqueda**, introduzca un valor.</span><span class="sxs-lookup"><span data-stu-id="20a75-112">In the **Search name** field, type a value.</span></span> <span data-ttu-id="20a75-113">Por ejemplo, especifique **portátil**.</span><span class="sxs-lookup"><span data-stu-id="20a75-113">For example, enter **laptop**.</span></span>  
6. <span data-ttu-id="20a75-114">Expanda la sección **Información técnica**.</span><span class="sxs-lookup"><span data-stu-id="20a75-114">Expand the **Technical information** section.</span></span>
7. <span data-ttu-id="20a75-115">Especifique valores en los campos **Marca**, **Modelo** y **Año de modelo**.</span><span class="sxs-lookup"><span data-stu-id="20a75-115">In the **Make**, **Model**, and **Model year** fields, type values.</span></span>
8. <span data-ttu-id="20a75-116">En el panel de acciones, seleccione **Opciones**.</span><span class="sxs-lookup"><span data-stu-id="20a75-116">On the Action Pane, select **Options**.</span></span>
9. <span data-ttu-id="20a75-117">Seleccione **Información de registro**.</span><span class="sxs-lookup"><span data-stu-id="20a75-117">Select **Record info**.</span></span>
10. <span data-ttu-id="20a75-118">Seleccionar **Plantilla de usuario**.</span><span class="sxs-lookup"><span data-stu-id="20a75-118">Select **User template**.</span></span>
11. <span data-ttu-id="20a75-119">En el campo **Nombre**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="20a75-119">In the **Name** field, type a value.</span></span>
12. <span data-ttu-id="20a75-120">En el campo **Descripción**, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="20a75-120">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="20a75-121">Seleccione **Aceptar**.</span><span class="sxs-lookup"><span data-stu-id="20a75-121">Select **OK**.</span></span>
14. <span data-ttu-id="20a75-122">Seleccione **Cerrar**.</span><span class="sxs-lookup"><span data-stu-id="20a75-122">Select **Close**.</span></span>
