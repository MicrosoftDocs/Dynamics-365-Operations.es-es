# <a name="access-to-finance-and-tax-reference-data"></a>Acceso a datos financieros y de referencia fiscal

[!include [banner](../../includes/banner.md)]



Cada empresa trabaja con un conjunto básico de datos financieros, como el año calendario fiscal, la moneda en la que se realiza la transacción comercial, las cuentas en las que entra o sale el dinero para administrar la empresa, las tasas impositivas y las remesas. Estos datos residen en aplicaciones Finance and Operations. Sin embargo, está expuesto a Common Data Service para que las aplicaciones basadas en modelos en Microsoft Dynamics 365 puedan tener una única fuente de datos financieros y fiscales. De esta manera, los datos son uniformes en todo el ecosistema empresarial. 

Los datos financieros y de impuestos se integran mediante las siguientes asignaciones:

+ [Libro mayor integrado](ledger-mapping.md)
+ [Maestro de impuestos integrado](tax-mapping.md)
