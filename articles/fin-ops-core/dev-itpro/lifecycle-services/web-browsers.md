# <a name="supported-web-browsers-for-lifecycle-services"></a>Navegadores web compatibles para Lifecycle Services 

[!include[banner](../includes/banner.md)]

Los usuarios pueden acceder a Microsoft Dynamics Lifecyle Services (LCS) mediante el uso de las versiones más recientes de estos navegadores populares: 

- Microsoft Edge (recomendado: [Chromium Edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Para buscar la versión más reciente para cada explorador web, vaya al sitio web del fabricante de software. Para un rendimiento óptimo y una experiencia óptima, le recomendamos que utilice la última versión de un navegador moderno, especialmente Microsoft Edge.
