# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="bdf59-103">Configurar un alias de cuenta contable</span><span class="sxs-lookup"><span data-stu-id="bdf59-103">Set up a ledger account alias</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="bdf59-104">Este procedimiento muestra cómo crear un alias de cuenta que proporcione un método abreviado para especificar un número de cuenta.</span><span class="sxs-lookup"><span data-stu-id="bdf59-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="bdf59-105">Este procedimiento usa la empresa de datos de demostración USMF.</span><span class="sxs-lookup"><span data-stu-id="bdf59-105">This procedure users demo data company USMF.</span></span>

1. <span data-ttu-id="bdf59-106">Vaya a Contabilidad general > Plan contable > Cuentas > Alias de cuenta contable.</span><span class="sxs-lookup"><span data-stu-id="bdf59-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="bdf59-107">Haga clic en Nuevo.</span><span class="sxs-lookup"><span data-stu-id="bdf59-107">Click New.</span></span>
3. <span data-ttu-id="bdf59-108">En el campo Alias de cuenta contable, escriba un valor.</span><span class="sxs-lookup"><span data-stu-id="bdf59-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="bdf59-109">En el campo Estructura contable, seleccione la estructura a la que pertenecen la cuenta y las dimensiones.</span><span class="sxs-lookup"><span data-stu-id="bdf59-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="bdf59-110">En el campo Empresa, haga clic en el botón desplegable para abrir la búsqueda.</span><span class="sxs-lookup"><span data-stu-id="bdf59-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="bdf59-111">En la lista, busque y seleccione la empresa a la que se aplica el alias.</span><span class="sxs-lookup"><span data-stu-id="bdf59-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="bdf59-112">En la lista, haga clic en el vínculo de la fila seleccionada.</span><span class="sxs-lookup"><span data-stu-id="bdf59-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="bdf59-113">En el campo Definición del alias de la cuenta contable, especifique la cuenta y las dimensiones.</span><span class="sxs-lookup"><span data-stu-id="bdf59-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="bdf59-114">La cuenta y las dimensiones se rellenarán al usar el método abreviado.</span><span class="sxs-lookup"><span data-stu-id="bdf59-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="bdf59-115">En el campo Enfoque inicial, seleccione la dimensión que tendrá enfoque cuando se usa el alias.</span><span class="sxs-lookup"><span data-stu-id="bdf59-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="bdf59-116">Después de escribir el método abreviado, y de que se rellenen la cuenta y las dimensiones, el campo Enfoque inicial es donde se moverá el cursor o el enfoque.</span><span class="sxs-lookup"><span data-stu-id="bdf59-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
