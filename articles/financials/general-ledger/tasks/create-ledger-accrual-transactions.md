    * <span data-ttu-id="5cd9a-110">En este ejemplo, estamos definiendo el gasto para el seguro.</span><span class="sxs-lookup"><span data-stu-id="5cd9a-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="5cd9a-111">Se convertirá en un importe de gasto periódico.</span><span class="sxs-lookup"><span data-stu-id="5cd9a-111">It will become a periodic expense amount.</span></span>
