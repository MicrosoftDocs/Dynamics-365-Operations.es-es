[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="fa313-104">Este procedimiento muestra cómo usar la consulta de transacciones de asiento para buscar los movimientos del diario o las transacciones.</span><span class="sxs-lookup"><span data-stu-id="fa313-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="fa313-105">Vaya a Contabilidad general > Consultas e informes > Transacciones de asiento.</span><span class="sxs-lookup"><span data-stu-id="fa313-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="fa313-106">Seleccione el campo para el que desea definir un criterio de filtro.</span><span class="sxs-lookup"><span data-stu-id="fa313-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="fa313-107">Introduzca sus criterios de filtro para el campo seleccionado.</span><span class="sxs-lookup"><span data-stu-id="fa313-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="fa313-108">Podría filtrar en un único valor o en un intervalo.</span><span class="sxs-lookup"><span data-stu-id="fa313-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="fa313-109">Al definir un intervalo, asegúrese de que se use la sintaxis correcta.</span><span class="sxs-lookup"><span data-stu-id="fa313-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="fa313-110">Los valores se deben separar con doble punto (..).</span><span class="sxs-lookup"><span data-stu-id="fa313-110">The values should be separated by a double period (..).</span></span>

7. <span data-ttu-id="fa313-114">Haga clic en Cancelar si decide no agregar una tabla adicional.</span><span class="sxs-lookup"><span data-stu-id="fa313-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="fa313-115">Haga clic en la ficha Intervalo.</span><span class="sxs-lookup"><span data-stu-id="fa313-115">Click the Range tab.</span></span>
9. <span data-ttu-id="fa313-116">Haga clic en Aceptar para ejecutar la consulta.</span><span class="sxs-lookup"><span data-stu-id="fa313-116">Click OK to run the query.</span></span>
