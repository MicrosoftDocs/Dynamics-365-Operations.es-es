# <a name="ledger-allocation-rules"></a><span data-ttu-id="275a8-104">Reglas de asignación contable</span><span class="sxs-lookup"><span data-stu-id="275a8-104">Ledger allocation rules</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="275a8-105">Este artículo ofrece información general relativa a las reglas de asignación contable.</span><span class="sxs-lookup"><span data-stu-id="275a8-105">This article provides information about ledger allocation rules.</span></span> <span data-ttu-id="275a8-106">Describe los diversos componentes de estas reglas de asignación y los métodos de asignación que se pueden usar para ellos.</span><span class="sxs-lookup"><span data-stu-id="275a8-106">It describes the various components of these allocation rules and the allocation methods that can be used for them.</span></span>

<span data-ttu-id="275a8-107">Las reglas de asignación de contabilidad se usan para calcular y para generar automáticamente diarios y asientos contables para asignar saldos contables o importes fijos.</span><span class="sxs-lookup"><span data-stu-id="275a8-107">Ledger allocation rules are used to automatically calculate and generate allocation journals and account entries for the allocation of ledger balances or fixed amounts.</span></span> <span data-ttu-id="275a8-108">Los métodos de asignación pueden ser variables o fijos.</span><span class="sxs-lookup"><span data-stu-id="275a8-108">Allocation methods can be variable or fixed.</span></span> <span data-ttu-id="275a8-109">Se pueden usar los siguientes métodos de asignación para las reglas de asignación de contabilidad:</span><span class="sxs-lookup"><span data-stu-id="275a8-109">The following allocation methods can be used for ledger allocation rules:</span></span>

-   <span data-ttu-id="275a8-110">**Base**: este método variable se usa cuando la asignación depende de los saldos contables reales, en función de los criterios de filtro.</span><span class="sxs-lookup"><span data-stu-id="275a8-110">**Basis** – This variable method is used when the allocation depends on the actual ledger balance, based on filter criteria.</span></span> <span data-ttu-id="275a8-111">Por ejemplo, los costes de publicidad se pueden asignar según las ventas de cada departamento en proporción con el total de ventas departamentales.</span><span class="sxs-lookup"><span data-stu-id="275a8-111">For example, advertising expenses can be allocated based on each department's sales in proportion to the total departmental sales.</span></span>
-   <span data-ttu-id="275a8-112">**Porcentaje fijo** y **Peso fijo**: para estos métodos, el porcentaje de asignación o el peso se define directamente para la regla.</span><span class="sxs-lookup"><span data-stu-id="275a8-112">**Fixed percentage** and **Fixed weight** – For these methods, the allocation percentage or weight is defined directly for the rule.</span></span> <span data-ttu-id="275a8-113">Por ejemplo, se pueden asignar gastos de publicidad de modo que el departamento A reciba el 70 por ciento de los gastos de publicidad y el departamento B reciba el 30 por ciento.</span><span class="sxs-lookup"><span data-stu-id="275a8-113">For example, advertising expenses can be allocated so that Department A receives 70 percent of the advertising expense and Department B receives 30 percent.</span></span>
-   <span data-ttu-id="275a8-114">**Equitativamente**: este método distribuye el importe de manera equitativa a cada destino definido.</span><span class="sxs-lookup"><span data-stu-id="275a8-114">**Equally** – This method distributes the amount equally to each defined destination.</span></span> <span data-ttu-id="275a8-115">Por ejemplo, si se definen destinos para el departamento A y el departamento B, los gastos de publicidad se pueden asignar de modo que el departamento A y el departamento B reciban el 50 por ciento de los gastos de publicidad.</span><span class="sxs-lookup"><span data-stu-id="275a8-115">For example, if destinations are defined for Department A and Department B, advertising expenses can be allocated so that both Department A and Department B receive 50 percent of the advertising expense.</span></span>

<span data-ttu-id="275a8-116">Si se usa Base como método de asignación para una regla de asignación, entonces también deberá definir una regla de base de asignación contable independiente.</span><span class="sxs-lookup"><span data-stu-id="275a8-116">If Basis is used as the allocation method for an allocation rule, you must also define separate ledger allocation basis rules.</span></span> <span data-ttu-id="275a8-117">El proceso "Solicitud de asignación de proceso" permite a los usuarios procesar la regla de asignación de contabilidad y ver las entradas de asignación del diario resultantes antes de que se registren o eliminen las asignaciones calculadas.</span><span class="sxs-lookup"><span data-stu-id="275a8-117">The "Process allocation request" process lets users process the ledger allocation rule and preview the resulting allocation journal entries before they either post or delete the calculated allocations.</span></span>

## <a name="components-of-ledger-allocation-rules"></a><span data-ttu-id="275a8-118">Componentes de reglas de asignación contable</span><span class="sxs-lookup"><span data-stu-id="275a8-118">Components of ledger allocation rules</span></span>
<span data-ttu-id="275a8-119">Cada regla de asignación tiene cuatro componentes: general, origen, destino y contrapartida.</span><span class="sxs-lookup"><span data-stu-id="275a8-119">Each allocation rule has four components: general, source, destination, and offset.</span></span> <span data-ttu-id="275a8-120">Se requiere un componente adicional, reglas de bases de asignación contable, si se usa Base como método de asignación.</span><span class="sxs-lookup"><span data-stu-id="275a8-120">An additional component, ledger allocation bases rules, is required if Basis is used as the allocation method.</span></span> <span data-ttu-id="275a8-121">Cada componente proporciona una parte crítica de la información necesaria para procesar asignaciones.</span><span class="sxs-lookup"><span data-stu-id="275a8-121">Each component provides a critical piece of the information that is required in order to process allocations.</span></span>

-   <span data-ttu-id="275a8-122">**General**: el componente general es donde el usuario especifica opciones como el método de asignación, la configuración de las reglas de empresas vinculadas y si la regla está activa.</span><span class="sxs-lookup"><span data-stu-id="275a8-122">**General** – This component is where the user specifies options such as the allocation method, intercompany rule settings, and whether the rule is active.</span></span>
-   <span data-ttu-id="275a8-123">**Origen**: este componente es donde el usuario especifica los datos de origen para la asignación.</span><span class="sxs-lookup"><span data-stu-id="275a8-123">**Source** – This component is where the user specifies the source data for the allocation.</span></span> <span data-ttu-id="275a8-124">La asignación se puede basar en saldos contables (**Origen de datos** = **Libro mayor**) o importes fijos (**Origen de datos** = **Valor fijo**).</span><span class="sxs-lookup"><span data-stu-id="275a8-124">Allocation can be based on ledger balances (**Data source** = **Ledger**) or fixed amounts (**Data source** = **Fixed value**).</span></span> <span data-ttu-id="275a8-125">Cuando **Origen de datos** se establece en **Libro mayor**, se deben definir criterios de filtro de suministro para la regla de asignación de contabilidad (por ejemplo, para los gastos de publicidad).</span><span class="sxs-lookup"><span data-stu-id="275a8-125">When **Data source** is set to **Ledger**, source filter criteria must be defined for the ledger allocation rule (for example, for the advertising expenses).</span></span>
-   <span data-ttu-id="275a8-126">**Destino**: este componente define cómo se debe distribuir y contabilizar el resultado del cálculo de asignación.</span><span class="sxs-lookup"><span data-stu-id="275a8-126">**Destination** – This component defines how the result of the allocation calculation should be distributed and accounted for.</span></span> <span data-ttu-id="275a8-127">Por ejemplo, puede haber una línea de destino para cada departamento.</span><span class="sxs-lookup"><span data-stu-id="275a8-127">For example, there can be one destination line for each department.</span></span>
-   <span data-ttu-id="275a8-128">**Compensación**: este componente define cómo se deben determinar las cuentas principales y las dimensiones para las entradas de contrapartida que equilibran las entradas de destino.</span><span class="sxs-lookup"><span data-stu-id="275a8-128">**Offset** – This component defines how main accounts and dimensions should be determined for the offset entries that balance the destination entries.</span></span> <span data-ttu-id="275a8-129">Normalmente se usan opciones definidas por el usuario en lugar de cuentas o dimensiones basadas en el origen.</span><span class="sxs-lookup"><span data-stu-id="275a8-129">User-defined options are typically used instead of accounts and dimensions that are based on the source.</span></span> <span data-ttu-id="275a8-130">Cuando **Origen de datos** se establece en **Valor fijo**, **Origen** no se puede utilizar como opción.</span><span class="sxs-lookup"><span data-stu-id="275a8-130">When **Data source** is set to **Fixed value**, **Source** can't be used as an option.</span></span>
-   <span data-ttu-id="275a8-131">**Reglas de la base de asignación contable**: estas reglas usan sus propios criterios de filtro de origen para determinar qué saldos contables se deben usar para la asignación (por ejemplo, los ingresos por departamento).</span><span class="sxs-lookup"><span data-stu-id="275a8-131">**Ledger allocation basis rules** – These rules use their own source filter criteria to determine which ledger balances should be used for allocation (for example, the revenue per department).</span></span> <span data-ttu-id="275a8-132">Cada regla base de asignación se puede usar con varias reglas de asignación.</span><span class="sxs-lookup"><span data-stu-id="275a8-132">Each allocation basis rule can be used with multiple allocation rules.</span></span>
