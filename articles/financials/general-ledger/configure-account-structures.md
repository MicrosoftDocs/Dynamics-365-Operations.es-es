# <a name="configure-account-structures"></a><span data-ttu-id="05835-103">Configurar estructuras contables</span><span class="sxs-lookup"><span data-stu-id="05835-103">Configure account structures</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="05835-104">Las estructuras contables utilizan la cuenta principal y las dimensiones financieras para crear un conjunto de reglas que determinan el pedido y los valores que se utilizaron para introducir el número de cuenta.</span><span class="sxs-lookup"><span data-stu-id="05835-104">Account structures use the main account and financial dimensions to create a set of rules that determine the order and values used when entering the account number.</span></span> <span data-ttu-id="05835-105">Puede configurar tantas estructuras contables como necesite para su negocio.</span><span class="sxs-lookup"><span data-stu-id="05835-105">You can set up as many account structures as you need for your business.</span></span> <span data-ttu-id="05835-106">Las estructuras contables se asignan a la configuración del libro mayor de una empresa, de manera que se puedan compartir.</span><span class="sxs-lookup"><span data-stu-id="05835-106">The account structures are assigned to a company’s ledger setup, so they can be shared.</span></span>

<span data-ttu-id="05835-107">Al crear una estructura contable, el número máximo de segmentos es 11.</span><span class="sxs-lookup"><span data-stu-id="05835-107">When creating an account structure, the maximum number of segments is 11.</span></span> <span data-ttu-id="05835-108">Si necesita más segmentos, evalúe exhaustivamente la configuración y los requisitos, ya que afectará a la experiencia del usuario.</span><span class="sxs-lookup"><span data-stu-id="05835-108">If you need more segments than this, thoroughly evaluate your setup and requirements, as it will impact the user experience.</span></span> <span data-ttu-id="05835-109">Considere si un segmento podría derivar en un escenario de informes a través de una jerarquía en lugar durante la entrada de datos, o mediante un campo definido por el usuario.</span><span class="sxs-lookup"><span data-stu-id="05835-109">Consider if a segment could be derived in a reporting scenario using a hierarchy instead of during data entry, or by using a user-defined field.</span></span> <span data-ttu-id="05835-110">Por ejemplo, si desea informar sobre la ubicación, pero puede representar la ubicación por departamento o centro de coste, no necesitará la ubicación como una dimensión financiera.</span><span class="sxs-lookup"><span data-stu-id="05835-110">For example, if you want to report on location, but you can figure location by department or cost center, you would not need location as a financial dimension.</span></span> <span data-ttu-id="05835-111">Si después de la evaluación decide que son necesarios más de 11 segmentos, puede agregar segmentos adicionales mediante reglas avanzadas.</span><span class="sxs-lookup"><span data-stu-id="05835-111">If after evaluation you do determine more than 11 segments are needed, you can add additional segments using advanced rules.</span></span>

<span data-ttu-id="05835-112">Las estructuras contables requieren la cuenta principal.</span><span class="sxs-lookup"><span data-stu-id="05835-112">Account structures require the main account.</span></span> <span data-ttu-id="05835-113">La cuenta principal no necesita ser el primer segmento en la estructura, pero identifica qué estructura contable se está utilizando durante la entrada del número de cuenta.</span><span class="sxs-lookup"><span data-stu-id="05835-113">The main account does not need to be the first segment in the structure, but it does identify what account structure is being used during the account number entry.</span></span> <span data-ttu-id="05835-114">Debido a esto, un valor de la cuenta principal solo puede existir en una estructura asignada al libro mayor para que no se superpongan.</span><span class="sxs-lookup"><span data-stu-id="05835-114">Because of this, a main account value can only exist in one structure assigned to the ledger so that they do not overlap.</span></span> <span data-ttu-id="05835-115">Después de que se identifique la estructura contable, la lista de valores permitida se filtra para guiar al usuario solo a través de la selección de valores de dimensión válidos, lo que reduce la posibilidad de un movimiento de diario incorrecto.</span><span class="sxs-lookup"><span data-stu-id="05835-115">After the account structure is identified, the allowed values list is filtered to guide the user through picking only valid dimension values, decreasing the possibility of an incorrect journal entry.</span></span>

> [!NOTE] 
> <span data-ttu-id="05835-116">Si planea presupuestar una dimensión financiera, esta deberá formar parte de una estructura contable.</span><span class="sxs-lookup"><span data-stu-id="05835-116">If you plan to budget against a financial dimension, it will need to be part of an account structure.</span></span> <span data-ttu-id="05835-117">La gestión presupuestaria no utiliza actualmente reglas avanzadas.</span><span class="sxs-lookup"><span data-stu-id="05835-117">Budgeting does not currently utilize advanced rules.</span></span>
