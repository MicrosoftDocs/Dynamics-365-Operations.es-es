# <a name="configure-base-entities"></a>Configurar entidades base 

[!include[banner](../includes/banner.md)]


La contabilidad de costes es un módulo independiente que requiere que las entidades base siguientes se configuren correctamente en Dynamics 365 for Finance and Operations, Enterprise Edition.

-  Divisa
-  Tipo de cambio
-  Calendario fiscal  

Antes de utilizar el módulo de contabilidad de costes, debe tener una divisa, un tipo de cambio y un calendario fiscal definidos para la empresa. 

Para obtener más información acerca de estas entidades, haga el curso de formación de eLearning [Configuración de la divisa y el tipo de cambio del módulo 2](https://mbspartner.microsoft.com/AX/CourseModules/1215) (requiere una cuenta de CustomerSource).
