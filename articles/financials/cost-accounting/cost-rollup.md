# <a name="cost-rollup-policy-and-overhead-calculation"></a><span data-ttu-id="1b93a-103">Directiva de acumulación de costes y cálculo de costes generales</span><span class="sxs-lookup"><span data-stu-id="1b93a-103">Cost rollup policy and overhead calculation</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1b93a-104">La contabilidad de costes permite obtener detalles sobre cómo en el flujo de costes está relacionado con los productos y los servicios que se entregan en una organización.</span><span class="sxs-lookup"><span data-stu-id="1b93a-104">Cost accounting lets you gain insight into how the cost flow relates to the products and services that are delivered within an organization.</span></span> <span data-ttu-id="1b93a-105">Para ver la transparencia de los costes, es esencial lograr una asignación de costes entre los objetos de coste basada en una base de asignación adecuada.</span><span class="sxs-lookup"><span data-stu-id="1b93a-105">To see the cost transparency, it's crucial to achieve cost allocation between cost objects based on an appropriate allocation basis.</span></span> <span data-ttu-id="1b93a-106">De forma predeterminada, la asignación de costes se consigue para el elemento de costes principal, lo que se desea en algunos casos, pero tiene ciertas implicaciones que deben considerarse.</span><span class="sxs-lookup"><span data-stu-id="1b93a-106">By default, the cost allocation is achieved for the primary cost element, which is desired in some situations, but it has a few implications that should be considered.</span></span>

-   <span data-ttu-id="1b93a-107">Los objetos de coste auxiliares terminan con un saldo de cero para el elemento de coste principal después del cálculo de gastos generales.</span><span class="sxs-lookup"><span data-stu-id="1b93a-107">Auxiliary cost objects will end with a zero balance for the primary cost element after overhead calculation.</span></span>

<span data-ttu-id="1b93a-110">Para evitar estas implicaciones, la contabilidad de costes le permite configurar la asignación de costes para se ajuste a los requisitos de generación de informes ejecutivos de la organización.</span><span class="sxs-lookup"><span data-stu-id="1b93a-110">To avoid these implications, Cost accounting lets you configure cost allocation to fit into your organization managerial reporting requirements.</span></span> <span data-ttu-id="1b93a-111">Este tema describe cómo puede determinar el nivel correcto de elementos de coste secundarios y crear reglas de acumulación que se ajusten a los informes de la organización y la rastreabilidad de costes.</span><span class="sxs-lookup"><span data-stu-id="1b93a-111">This topic discusses how you can determine the correct level of secondary cost elements and create cost rollup rules that fit into organization reporting and cost traceability.</span></span>

> <span data-ttu-id="1b93a-112">Puede cambiar las configuraciones si cambian los requisitos de los informes.</span><span class="sxs-lookup"><span data-stu-id="1b93a-112">You can change configurations if reporting requirements change.</span></span>
