# <a name="project-quotations"></a><span data-ttu-id="7e715-104">Presupuestos de proyecto</span><span class="sxs-lookup"><span data-stu-id="7e715-104">Project quotations</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="7e715-105">Este artículo presenta el concepto de presupuestos de proyecto, que puede usar para realizar una oferta atractiva a un cliente como el primer paso de la fase de proyecto.</span><span class="sxs-lookup"><span data-stu-id="7e715-105">This article introduces the concept of project quotations, which you can use to make an attractive offer to a customer as the first step of the project phase.</span></span> <span data-ttu-id="7e715-106">Un presupuesto de proyecto puede incluir los artículos y los servicios que se van a presupuestar, la información básica de contacto, los acuerdos comerciales especiales y los descuentos, así como posibles impuestos y suplementos.</span><span class="sxs-lookup"><span data-stu-id="7e715-106">A project quotation might include the items and services that are quoted, basic contact information, special trade agreements and discounts, and possible taxes and surcharges.</span></span> 

<span data-ttu-id="7e715-107">La habilidad para realizar un seguimiento, revisar y controlar el proceso de presupuestos y pedidos de proyecto es una parte importante de la gestión del proyecto.</span><span class="sxs-lookup"><span data-stu-id="7e715-107">The ability to monitor, review, and control the pipeline of project quotations and orders is an important part of project management.</span></span> <span data-ttu-id="7e715-108">Varias herramientas de Microsoft Dynamics 365 for Finance and Operations, Enterprise Edition pueden ayudarle con estas tareas.</span><span class="sxs-lookup"><span data-stu-id="7e715-108">Various tools in Microsoft Dynamics 365 for Finance and Operations, Enterprise edition can help with these tasks.</span></span> <span data-ttu-id="7e715-109">Por ejemplo, la de corrección de las definiciones de datos de referencia (tipos y origen de presupuestos, así como pronóstico y probabilidad), le ayudan a analizar el proceso.</span><span class="sxs-lookup"><span data-stu-id="7e715-109">For example, correct reference data definitions (quotation types, quotation origin, and prognosis and probability) help you analyze the pipeline.</span></span> <span data-ttu-id="7e715-110">Puede usar estas herramientas para categorizar las razones por las que un presupuesto de proyecto ha sido un éxito o no, y para determinar el valor posible de un presupuesto.</span><span class="sxs-lookup"><span data-stu-id="7e715-110">You can use these tools to categorize the reasons why a project quotation was won or lost, and to determine the potential value of a quotation.</span></span> 

<span data-ttu-id="7e715-111">En un presupuesto de proyecto, puede especificar los servicios, la información básica de contacto, los acuerdos comerciales especiales y los descuentos, así como los impuestos y suplementos estimados para un proyecto.</span><span class="sxs-lookup"><span data-stu-id="7e715-111">In a project quotation, you enter the services, basic contact information, special trade agreements and discounts, and estimated taxes and surcharges for a project.</span></span> <span data-ttu-id="7e715-112">También puede seleccionar las actividades o las tareas de un proyecto, y crear una jerarquía de tareas y subtareas.</span><span class="sxs-lookup"><span data-stu-id="7e715-112">You can also select the activities or tasks for a project, and create a hierarchy of tasks and subtasks.</span></span> <span data-ttu-id="7e715-113">Para cada actividad, puede especificar detalles acerca de los tiempos y la duración de la actividad, así como de las aptitudes y la experiencia que necesitan los trabajadores que la realizan.</span><span class="sxs-lookup"><span data-stu-id="7e715-113">For each activity, you can enter details about the timing and duration of the activity, and about the skills and experience that are required for workers who perform the activity.</span></span> 

<span data-ttu-id="7e715-114">El presupuesto de proyecto es una estimación no vinculante del trabajo que se debe realizar.</span><span class="sxs-lookup"><span data-stu-id="7e715-114">The project quotation is a non-binding estimate of the work that must be performed.</span></span> <span data-ttu-id="7e715-115">Sin embargo, cuando la información del presupuesto se copia en un proyecto que está asociado a un contrato de proyecto, esa información se convierte en parte de un acuerdo vinculante entre las dos partes.</span><span class="sxs-lookup"><span data-stu-id="7e715-115">However, when the information in the quotation is copied to a project that is associated with a project contract, that information becomes part of a binding agreement between two parties.</span></span> 

<span data-ttu-id="7e715-116">Si el cliente aprueba el presupuesto de proyecto, puede copiar la información del presupuesto de proyecto en un proyecto.</span><span class="sxs-lookup"><span data-stu-id="7e715-116">If the customer approves the project quotation, you can copy the information in the project quotation to a project.</span></span> <span data-ttu-id="7e715-117">También puede copiar la información del presupuesto de proyecto en una previsión de proyecto al mismo tiempo.</span><span class="sxs-lookup"><span data-stu-id="7e715-117">You can also copy the project quotation information to a project forecast at the same time.</span></span>
