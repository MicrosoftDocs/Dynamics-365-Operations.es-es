<span data-ttu-id="47023-107">Las distribuciones contables son una capacidad de todo el programa que se usa y se extiende por cada documento de origen, como un pedido de compra, una factura de proveedor, un informe de gastos y una factura de servicios.</span><span class="sxs-lookup"><span data-stu-id="47023-107">Accounting distributions are a program-wide capability that is used and extended by each source document, such as a purchase order, vendor invoice, expense report, and free text invoice.</span></span> <span data-ttu-id="47023-108">De forma predeterminada, se genera una distribución contable predeterminada para cada línea del documento de origen y el importe monetario, y se habilita de manera condicional para la modificación.</span><span class="sxs-lookup"><span data-stu-id="47023-108">By default, a default accounting distribution is generated for each source document line and monetary amount, and is conditionally enabled for modification.</span></span>

    -   <span data-ttu-id="47023-128">La cuenta principal y la dimensión financiera pueden ser editables también para los elementos secundarios.</span><span class="sxs-lookup"><span data-stu-id="47023-128">The main account and financial dimension might be editable for children too.</span></span>
