# <a name="replenishment"></a><span data-ttu-id="3b2ec-103">Reabastecimiento</span><span class="sxs-lookup"><span data-stu-id="3b2ec-103">Replenishment</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="3b2ec-104">Este tema describe las estrategias de reabastecimiento que están disponibles para los almacenes que utilizan la funcionalidad que está disponible en Administración de almacenes.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="3b2ec-105">La información no se aplica a la solución de almacenamiento que está disponible en Gestión de inventarios.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="3b2ec-106">Las siguientes estrategias de reabastecimiento están disponibles:</span><span class="sxs-lookup"><span data-stu-id="3b2ec-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="3b2ec-107">**Reabastecimiento de demanda de oleada**: esta estrategia crea el trabajo de reabastecimiento para los pedidos o las cargas de salida si el inventario no está disponible cuando la oleada crea el trabajo.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="3b2ec-108">Por ejemplo, el trabajo de reabastecimiento se puede crear si la cantidad necesaria para un pedido de ventas no está disponible cuando se procesa una oleada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
-   <span data-ttu-id="3b2ec-109">**Reabastecimiento mínimo/máximo**: esta estrategia usa los límites de existencias mínimos y máximos para determinar cuándo se deben reabastecer las ubicaciones.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="3b2ec-110">Los criterios de artículo y ubicación definen el inventario que se evalúa para el reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="3b2ec-111">Las plantillas de reabastecimiento mínimo/máximo son el mecanismo principal para mantener niveles óptimos en ubicaciones de picking.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="3b2ec-112">Para garantizar que hay suficiente inventario de frente de recogida disponible para satisfacer la demanda de oleada, puede utilizar el reabastecimiento de demanda como suplemento entre los ciclos de reabastecimiento mínimo o máximo.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-112">To ensure that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
-   <span data-ttu-id="3b2ec-113">**Reabastecimiento de demanda de carga**: esta estrategia suma la demanda para varias cargas y crea el trabajo de reabastecimiento que es necesario para almacenar las ubicaciones de picking pertinentes.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="3b2ec-114">Esta estrategia ayuda a garantizar que las cargas que se crean se pueden seleccionar en el almacén tras su emisión.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-114">This strategy helps to ensure that the loads that are created can be picked in the warehouse after they are released.</span></span>

<span data-ttu-id="3b2ec-115">Las tres estrategias crean trabajo de reabastecimiento, en función de una plantilla de reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-115">All three strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="3b2ec-116">Reabastecimiento de la demanda de oleada</span><span class="sxs-lookup"><span data-stu-id="3b2ec-116">Wave demand replenishment</span></span>

<span data-ttu-id="3b2ec-117">El reabastecimiento de demanda de oleada crea el trabajo de reabastecimiento, basado en demanda, si la cantidad necesaria para los pedidos de producción, kanbans, pedidos o cargas de salida no está disponible cuando la oleada crea el trabajo.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-117">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="3b2ec-118">La plantilla de reabastecimiento contiene información sobre los criterios del artículo, la unidad de medida, el incremento de la demanda y la ubicación.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-118">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="3b2ec-119">Las directivas de ubicación se utilizan para determinar qué ubicación se debe reabastecer.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-119">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="3b2ec-120">Vincula estas directivas de ubicación a la plantilla de reabastecimiento mediante el campo **Código de directiva**.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-120">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="3b2ec-121">Si el campo **Código de directiva** no está establecido, las consultas se utilizan para determinar qué directiva de ubicación se debe utilizar.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-121">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="3b2ec-122">Tenga en cuenta que si no se especifica un código de directiva en la plantilla de reabastecimiento y la directiva de ubicación tiene un código de directiva, se omitirá la directiva de ubicación, incluso si la consulta sobre la directiva de ubicación es correcta.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-122">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="3b2ec-123">Las directivas de ubicación de picking se utilizan para determinar dónde obtener inventario para el reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-123">Pick location directives are used to determine where to get inventory for the replenishment.</span></span> 

<span data-ttu-id="3b2ec-124">Además de crear una plantilla, debe especificar algunos valores de reabastecimiento en la plantilla de oleada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-124">In addition to creating a template, you must specify some replenishment settings in the wave template.</span></span> <span data-ttu-id="3b2ec-125">La plantilla de oleada debe contener un paso de oleada para reabastecimiento que solo se ejecuta si la asignación de un artículo no es correcta.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-125">The wave template should contain a wave step for replenishment that is run only if allocation of an item isn't successful.</span></span> <span data-ttu-id="3b2ec-126">Este paso de oleada de reabastecimiento utiliza un código de paso de oleada para determinar qué plantilla de reabastecimiento debe utilizarse.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-126">This replenishment wave step uses a wave step code to determine which replenishment template should be used.</span></span> <span data-ttu-id="3b2ec-127">Además de tener un paso de oleada para reabastecimiento, tiene que asegurarse de que la opción **Reabastecer** está seleccionada en la sección **Métodos** de la plantilla de oleada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-127">In addition to having a wave step for replenishment, you need to make sure that **Replenish** is selected in the **Methods** section of the wave template.</span></span> 

<span data-ttu-id="3b2ec-128">La página **Plantilla de reabastecimiento** incluye una casilla **Permitir una demanda de oleadas para usar las cantidades que no estén reservadas**.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-128">The **Replenishment template** page includes an **Allow wave demand to use unreserved quantities** check box.</span></span> <span data-ttu-id="3b2ec-129">Debe activar esta casilla si desea permitir al reabastecimiento de la demanda deduzca cantidades sin reservas del trabajo que se genera a partir de la plantilla de reabastecimiento seleccionada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-129">You should select this check box if you want to allow demand replenishment to deduct unreserved quantities from work that is generated from the selected replenishment template.</span></span> <span data-ttu-id="3b2ec-130">Para que las plantillas de reabastecimiento de la demanda puedan usar esta lógica, seleccione esta casilla para cada plantilla de reabastecimiento existente.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-130">To enable demand replenishment templates to use this logic, select this check box for every existing replenishment template.</span></span> <span data-ttu-id="3b2ec-131">Cuando el reabastecimiento de la demanda se active en el almacén, deducirá la demanda del trabajo de reabastecimiento existente con cantidades sin reservas, si el trabajo se origina a partir de plantillas de reabastecimiento donde la casilla **Permitir que la demanda de oleada use cantidades sin reservas** está activada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-131">When demand replenishment is triggered in the warehouse, it will deduct the demand from existing replenishment work that has unreserved quantities, if the work originates from replenishment templates where the **Allow wave demand to use unreserved quantities** check box is selected.</span></span>


<span data-ttu-id="3b2ec-132">El reabastecimiento de la demanda se admite para los pedidos de ventas, pedidos de transferencia, pedidos de producción y kanbans.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-132">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="3b2ec-133">Reabastecimiento mínimo/máximo</span><span class="sxs-lookup"><span data-stu-id="3b2ec-133">Min/Max replenishment</span></span>
<span data-ttu-id="3b2ec-134">En el reabastecimiento mínimo/máximo, las existencias se reabastecen para que se encuentren entre los límites mínimo y máximo que se han establecido.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-134">In Min/Max replenishment, stock is replenished so that it's between the minimum and maximum limits that have been set.</span></span> <span data-ttu-id="3b2ec-135">Normalmente, este proceso se produce una vez al día para garantizar que todas las ubicaciones de picking se rellenan hasta el nivel máximo antes de que se inicie el picking.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-135">Typically, this process occurs one time every day to ensure that all picking locations are filled to the maximum level before picking starts.</span></span> 

<span data-ttu-id="3b2ec-136">Los importes mínimo y máximo se establecen en una plantilla de reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-136">The minimum and maximum amounts are set in a replenishment template.</span></span> <span data-ttu-id="3b2ec-137">Muchos de los demás valores de la plantilla son similares a la configuración de las plantillas que se utilizan en el reabastecimiento de la demanda de oleada.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-137">Many of the other settings in the template resemble the settings in templates that are used in Wave demand replenishment.</span></span> <span data-ttu-id="3b2ec-138">La plantilla debe contener una línea para cada artículo y ubicación.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-138">The template should contain one line for each item and location.</span></span> <span data-ttu-id="3b2ec-139">Cuando ejecute el reabastecimiento mediante el trabajo por lotes, Finance and Operations evalua si el reabastecimiento es necesario en la secuencia en la que se organizan las líneas.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-139">When you run replenishment by using the batch job, Finance and Operations evaluates whether replenishment is required, in the sequence that the lines are organized in.</span></span> 

<span data-ttu-id="3b2ec-140">Tenga en cuenta que la estrategia de reabastecimiento mínima/máxima no puede reabastecer una ubicación vacía a menos que la ubicación se establezca como la ubicación fija para el artículo.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-140">Note that the Min/Max replenishment strategy can't replenish an empty location unless the location is set as the fixed location for the item.</span></span> <span data-ttu-id="3b2ec-141">Si la ubicación que debe reabastecerse no es una ubicación fija, no se puede determinar qué elemento se va a reabastecer.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-141">If the location that must be replenished isn't a fixed location, it's not possible to determine which item to replenish.</span></span> <span data-ttu-id="3b2ec-142">Por tanto, se requiere al menos alguna cantidad disponible para que se produzca el reabastecimiento.</span><span class="sxs-lookup"><span data-stu-id="3b2ec-142">Therefore, at least some on-hand quantity is required before replenishment occurs.</span></span>
