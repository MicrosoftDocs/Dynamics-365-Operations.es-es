# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Crear una fórmula copiando una fórmula existente

[!include[task guide banner](../../includes/task-guide-banner.md)]

Este procedimiento muestra cómo crear rápidamente una fórmula nueva copiando de una fórmula existente. Puede usar las empresas de datos de demostración USMF, USP2 o USPI para crear este procedimiento.


## <a name="create-a-new-formula"></a>Crear una nueva fórmula
1. Vaya a Fórmulas.
2. Haga clic en Nuevo.
3. En el campo Fórmula, escriba un valor.
4. En el campo Nombre, escriba un valor.
5. En el campo Sitio, especifique o seleccione un valor.
6. En el campo Grupo de artículos, especifique o seleccione un valor.
7. Haga clic en Guardar.

## <a name="copy-from-an-existing-formula"></a>Copiar de una fórmula existente
1. Haga clic en Copiar.
2. En el campo Número de artículo, especifique o seleccione un valor.
3. En el campo Versión de fórmula, especifique o seleccione un valor.
4. Haga clic en Aceptar

## <a name="revise-the-copied-formula"></a>Revisar la fórmula copiada
1. En la lista, busque y seleccione el registro deseado.
2. Haga clic Eliminar.
3. Haga clic en Sí.

## <a name="approve-formula"></a>Aprobar fórmula
1. Haga clic en Aprobar fórmula.
2. En el campo Aprobado por, especifique o seleccione un valor.
3. Haga clic en Aceptar
4. Cierre la página.
