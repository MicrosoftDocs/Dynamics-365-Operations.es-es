# <a name="report-progress-on-a-mobile-job-device"></a><span data-ttu-id="8475b-103">Notificar progreso en un dispositivo móvil de trabajo</span><span class="sxs-lookup"><span data-stu-id="8475b-103">Report progress on a mobile job device</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8475b-105">Para poder ejecutar este procedimiento, debe tener el rol del administrador del sistema o del operador de maquinaria asociado a la cuenta de usuario.</span><span class="sxs-lookup"><span data-stu-id="8475b-105">To be able to run this procedure you must have the System administator or Machine Operator role associated with the user account.</span></span>

1. <span data-ttu-id="8475b-106">Vaya a Control de producción > Ejecución de fabricación > Dispositivo de tarjetas de trabajo.</span><span class="sxs-lookup"><span data-stu-id="8475b-106">Go to Production control > Manufacturing execution > Job card device.</span></span>

3. <span data-ttu-id="8475b-109">Haga clic en Iniciar sesión.</span><span class="sxs-lookup"><span data-stu-id="8475b-109">Click Log in.</span></span>
4. <span data-ttu-id="8475b-110">Haga clic en el botón Filtrar.</span><span class="sxs-lookup"><span data-stu-id="8475b-110">Click the Filter button.</span></span>
5. <span data-ttu-id="8475b-111">Active o desactive la casilla Aplicar filtro de configuración.</span><span class="sxs-lookup"><span data-stu-id="8475b-111">Check or uncheck the Apply configuration filter check box.</span></span> <span data-ttu-id="8475b-112">Si establece un filtro, puede usar la unidad de producción 110 en USMF.</span><span class="sxs-lookup"><span data-stu-id="8475b-112">If you set a filter you can use production unit 110 in USMF.</span></span>

9. <span data-ttu-id="8475b-116">Haga clic en el botón Iniciar trabajo.</span><span class="sxs-lookup"><span data-stu-id="8475b-116">Click the Start job button.</span></span>
10. <span data-ttu-id="8475b-117">Haga clic en Aceptar</span><span class="sxs-lookup"><span data-stu-id="8475b-117">Click OK.</span></span>
11. <span data-ttu-id="8475b-118">Haga clic en el botón Informar progreso.</span><span class="sxs-lookup"><span data-stu-id="8475b-118">Click the Report progress button.</span></span>
12. <span data-ttu-id="8475b-119">Haga clic en Aceptar</span><span class="sxs-lookup"><span data-stu-id="8475b-119">Click OK.</span></span>
13. <span data-ttu-id="8475b-120">Haga clic en el botón Trabajo siguiente.</span><span class="sxs-lookup"><span data-stu-id="8475b-120">Click the Next job button.</span></span>

24. <span data-ttu-id="8475b-132">Haga clic en Detener descanso.</span><span class="sxs-lookup"><span data-stu-id="8475b-132">Click Stop break.</span></span>

26. <span data-ttu-id="8475b-134">Haga clic en Cancelar.</span><span class="sxs-lookup"><span data-stu-id="8475b-134">Click Cancel.</span></span>
