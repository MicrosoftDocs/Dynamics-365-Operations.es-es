# <a name="introduction-to-dynamics-365-for-operations"></a>Introducción a Dynamics 365 para operaciones

[!include[banner](includes/banner.md)]

Microsoft Dynamics 365 para operaciones es la aplicación de negocio de Microsoft para empresas. Permite que varias personas rápidamente tomar decisiones más acertadas a través de una interfaz de usuario inteligente que proporciona acceso a datos y perspectivas en tiempo real. Permite a las empresas de transformación y innnovate al ofrecer la lógica empresarial probada que les permite rediseñar sus procesos de negocio con mayor rapidez. Proporciona a las empresas la flexibilidad necesaria para crecer a su propio ritmo y hacer negocios casi en cualquier lugar y momento, lo que les permite escalar sus operaciones a nivel mundial para satisfacer las necesidades del negocio. 

Este manual del usuario puede ayudarle a aprender cómo hacer 365 Dynamics para operaciones para su negocio.
